Como hay varias personas preguntando cómo activar los comandos del Object Manager y no lo entendieron, escribiré aquí.

Para activar las partes relacionadas con clases y objetos SHIFT+CTRL+F12
Para activar la parte Functions SHIFT+CTRL+F11

Si desea vincular una variable de objeto a una clase Ex:
Variable ODlgTest
Copie la variable al portapapeles de Windows con SHIFT+CTRL+C
Pregunta SHIFT+CTRL+F12
El OM buscará la variable para el proyecto activo y comprobará si está vinculada. De lo contrario, aparecerá un cuadro de diálogo para que seleccione el origen de la clase que desea vincular.
Classes (Origem Nucleos) (con las clases relacionadas con el núcleo que funcionarán en todos los proyectos)
O classes (Origem Projetos) (solo clases relacionadas con el proyecto activo)
En este caso, selecciono Core Classes y vincularé la clase Fivewin tDialog.
Si se vuelve a llamar a la variable oDlgTeste, se enumerarán todas las propiedades (Fechas) y los métodos que hacen referencia a la clase.
Si copia un texto con el nombre de la clase Ex:
tDialog y presione SHIFT+CTRL+F12, la clase tDialog se mostrará independientemente de la variable que no se elija.

Para funciones es SHIFT+CTRL+F11
Si copia el nombre de una función y esta función existe en los núcleos o proyectos, la función con la sintaxis se pegará automáticamente en el editor de texto.

Si copia la función y al principio tiene @ Ex:
@Hb_qSelf, el enlace Harbour Wiki se abrirá en el explorador de funciones Harbour y el contenido se completará automáticamente.
