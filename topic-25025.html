A click on field 22 ( Row 4 / Col 4 )

calculates the DBF-record-position ( of 30 Records )

oBrw:bLClicked := { | nRow, nCol | ( nRPos := oBrw:KeyNo(), ;
nCPos := oBrw:SelectedCol():nCreationOrder, ;
nRecPos := ( ( nRPos - 1 ) * 6 ) + nCPos ) }
