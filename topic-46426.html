This is a custom dropdown (looks like Bitrix24 CRM), not a native <select>. You need to simulate the click interaction

via JavaScript. Here's how:



Step 1: Click the dropdown to open it



// First, click the dropdown to open the options popup


Code: Select all Collapse
</s>oWebView:ExecuteScript( ;
     'document.querySelector(''[data-name="UF_CRM_1768552906866"]'').click()' )<e>


Step 2: Wait briefly, then click the desired option



The popup is dynamically created. After clicking, Bitrix renders a menu with main-ui-select-inner-item elements. You

need a small delay:



Code: Select all Collapse
</s>// Select "Option 3" (VALUE=1854) after a short delay
  oWebView:ExecuteScript( ;
     'setTimeout(function() {' + ;
     '  var items = document.querySelectorAll(".main-ui-select-inner-item");' + ;
     '  for (var i = 0; i < items.length; i++) {' + ;
     '    if (items[i].textContent.trim() === "Option 3") {' + ;
     '      items[i].click();' + ;
     '      break;' + ;
     '    }' + ;
     '  }' + ;
     '}, 300)' )

  Alternative: Select by VALUE instead of text

  If you know the value (e.g. 1854), you can set it directly via the data attributes:

  // Programmatically set the value by manipulating the data-value attribute
  // and dispatching the change event
  local cJS := ;
     'var el = document.querySelector(''[data-name="UF_CRM_1768552906866"]'');' + ;
     'var items = JSON.parse(el.getAttribute("data-items"));' + ;
     'var target = items.find(function(i) { return i.VALUE == 1854; });' + ;
     'if (target) {' + ;
     '  items.forEach(function(i) { i.IS_SELECTED = false; });' + ;
     '  target.IS_SELECTED = true;' + ;
     '  el.setAttribute("data-items", JSON.stringify(items));' + ;
     '  el.setAttribute("data-value", JSON.stringify(target));' + ;
     '  el.querySelector(".main-ui-select-name").textContent = target.NAME;' + ;
     '  el.click();' + ;
     '  setTimeout(function() {' + ;
     '    var popup = document.querySelectorAll(".main-ui-select-inner-item");' + ;
     '    for (var i = 0; i < popup.length; i++) {' + ;
     '      if (popup[i].textContent.trim() === target.NAME) {' + ;
     '        popup[i].click(); break;' + ;
     '      }' + ;
     '    }' + ;
     '  }, 300);' + ;
     '}'

  oWebView:ExecuteScript( cJS )
<e>

Complete helper function:



Code: Select all Collapse
</s>function SelectUIOption( oWebView, cFieldName, xValue )

<i>    </i> local cJS

<i>    </i> // xValue can be numeric (VALUE) or string (NAME)
<i>    </i> if ValType( xValue ) == "N"
<i>    </i>    cJS := 'var el = document.querySelector(''[data-name="' + cFieldName + '"]'');' + ;
<i>    </i>           'el.click();' + ;
<i>    </i>           'setTimeout(function() {' + ;
<i>    </i>           '  var items = JSON.parse(el.getAttribute("data-items"));' + ;
<i>    </i>           '  var t = items.find(function(i){ return i.VALUE == ' + LTrim(Str(xValue)) + '; });' + ;
<i>    </i>           '  if (t) {' + ;
<i>    </i>           '    var popup = document.querySelectorAll(".main-ui-select-inner-item");' + ;
<i>    </i>           '    for (var i = 0; i < popup.length; i++) {' + ;
<i>    </i>           '      if (popup[i].textContent.trim() === t.NAME) { popup[i].click(); break; }' + ;
<i>    </i>           '    }' + ;
<i>    </i>           '  }' + ;
<i>    </i>           '}, 300)'
<i>    </i> else
<i>    </i>    cJS := 'document.querySelector(''[data-name="' + cFieldName + '"]'').click();' + ;
<i>    </i>           'setTimeout(function() {' + ;
<i>    </i>           '  var popup = document.querySelectorAll(".main-ui-select-inner-item");' + ;
<i>    </i>           '  for (var i = 0; i < popup.length; i++) {' + ;
<i>    </i>           '    if (popup[i].textContent.trim() === "' + xValue + '") { popup[i].click(); break; }' + ;
<i>    </i>           '  }' + ;
<i>    </i>           '}, 300)'
<i>    </i> endif

<i>    </i> oWebView:ExecuteScript( cJS )

  return nil<e>


Usage:

// By value


Code: Select all Collapse
</s>SelectUIOption( oWebView, "UF_CRM_1768552906866", 1854 )<e>


// By name


Code: Select all Collapse
</s>SelectUIOption( oWebView, "UF_CRM_1768552906866", "Option 3" )<e>


The key insight is that these Bitrix custom dropdowns require two clicks: one to open the popup, then one on the

option. The setTimeout gives the popup time to render. If 300ms isn't enough, increase to 500.

Code: Select all Collapse
</s>  #include "hbcurl.ch"

  FUNCTION Main()

<i>    </i> LOCAL hCurl
<i>    </i> LOCAL cUrl      := "https://your-domain.bitrix24.com/rest/1/your-webhook-token/crm.deal.add"
<i>    </i> LOCAL cPayload  := '{"fields":{"TITLE":"New Deal","STAGE_ID":"NEW","CURRENCY_ID":"USD","OPPORTUNITY":5000}}'
<i>    </i> LOCAL cResponse := ""
<i>    </i> LOCAL nError

<i>    </i> hb_curl_global_init()

<i>    </i> hCurl := curl_easy_init()

<i>    </i> IF ! Empty( hCurl )

<i>    </i>    curl_easy_setopt( hCurl, HB_CURLOPT_URL, cUrl )
<i>    </i>    curl_easy_setopt( hCurl, HB_CURLOPT_POST, .T. )
<i>    </i>    curl_easy_setopt( hCurl, HB_CURLOPT_POSTFIELDS, cPayload )
<i>    </i>    curl_easy_setopt( hCurl, HB_CURLOPT_HTTPHEADER, { "Content-Type: application/json" } )
<i>    </i>    curl_easy_setopt( hCurl, HB_CURLOPT_DL_BUFF_SETUP )

<i>    </i>    nError := curl_easy_perform( hCurl )

<i>    </i>    IF nError == HB_CURLE_OK
<i>    </i>       cResponse := curl_easy_dl_buff_get( hCurl )
<i>    </i>       ? "Response:", cResponse
<i>    </i>    ELSE
<i>    </i>       ? "Error:", curl_easy_strerror( nError )
<i>    </i>    ENDIF

<i>    </i>    curl_easy_cleanup( hCurl )

<i>    </i> ENDIF

<i>    </i> hb_curl_global_cleanup()

  RETURN NIL<e>
