Si utilizas Funcky, este es el codigo:

Code: Select all Collapse
aPrinters := {}
oFunckyPrinters := TOleAuto():New("FUNCkyFindPrinter")
oFunckyPrinters:FindFirst()
DO WHILE oFunckyPrinters:Found()
      AADD(aPrinters,oFunckyPrinters:Name())
      oFunckyPrinters:FindNext()
ENDDO
MsgList(aPrinters)

MsgInfo(oFuncky:DefaultPrinter(),"Default Printer")

Funcky es un componente OLE que trae mas de 1000 metodos y funciones para usar con xHarbour.

Es la nueva version de la vieja libreria Funcky para Clipper de 1987, ahora 20 años despues, sigue actual pero ahora para Windows y para todos los lenguajes de programacion, xHarbour incluido.

Tiene componentes para todo:

  • Manipulación de Bits
  • Conversión de datos
  • Manejo de fechas
  • Gestión de discos y unidades
  • Información del entorno
  • Gestión de archivos
  • Rutinas de Internet (IIS uploading, SMTP, FTP, protocolo TCP-IP, estadísticas de red, Winsock, etc.)
  • Comunicación entre procesos
    Rutinas varias (chequeo de tarjetas de crédito, validación de códigos postales, números aleatorios, etc.)
  • Gestión de red (dominios, equipos, usuarios, información, etc.)
  • Registro de eventos de NT
  • Gestión de servicios de NT
  • ODBC y OLEDB
  • Gestión de impresoras
  • Manipulación del registro
  • Manipulación de cadenas
  • Objetos de sincronización
  • Información del sistema
