Cristobal,

Ese es el comportamiento por defecto de un Edit de Windows.

Para cambiarlo tendrias que usar bGotFocus y que desde ahí no se seleccione nada:

oEdit:bGotFocus = { || oEdit:SetSel( 0, 0 ) }
