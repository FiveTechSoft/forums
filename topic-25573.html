I started with the docs of my new Image-tool.
In case of any Image-problems / questions, it can help to solve the problem.
The download-link will follow.
Many calculations are needed for the different sections.
A interesting situation is included : You can check the difference of 2 images
displayed next each other ( medical ).

What You can do :
1. Select a image. The window will be adjusted to image-size
2. select 1 image for a dialog
3. select 2 images for a dialog. The images are adjusted to resource
4. select 2 images and display the images with 1 resource ( skip )

Each dialog includes realsize-preview, physical resizing and printing.

1. Define image for the main-window ( window adjusted )



2. Menu-selection of dialog-styles



3. Display a dialog with 1 image



4. Display a dialog with 2 Images adjusted to resource



5. Display a dialog with 2 images ( using 1 resource with skip )



A Image-click will close the preview-window !!!

6. Original size preview. The screen is splitted to show 2 oversized images



7. Original size preview. 1 Image full display. 2 Image oversized with scrollbars.



8. Original size preview. The complete width of Images 1 and 2 is smaller screensize.



9. Physical resizing

A new defined width, will keep the aspect ratio !!!
Your original will not be changed and a new Image will be created adding a => _1 <= to the image-name.



10. A combination of alphablended BMP and JPG



11. Transparent-test : image resized from resource



12. Transparent-test : window-preview of 2 oversized images next each other with scrollbars.
Image 1 the width and height is oversized, Image 2 only the width is oversized.



13. Test printing alphablended images.
