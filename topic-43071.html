deberias cambiarlas para que en vez de usar ... LIB "winfis32.dll" usen ... LIB hDll

declara static hDll al comienzo del PRG donde las tengas

y en tu código haz:

hDll := LoadLibrary( "winfis32.dll" )

y cuando termines de usarlas:

FreeLibrary( hDll )
