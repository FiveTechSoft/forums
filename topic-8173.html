Yo tengo en todos mis clientes Linux con Mandriva 2006 y funcionan perfectamente los programas de FiveWin.

Miconfiguración de Samba es:

[global]

1. Server Naming Options:

workgroup = NT-Domain-Name or Workgroup-Name

workgroup = GRUPO

netbios name is the name you will see in "Network Neighbourhood",

but defaults to your hostname

netbios name = SERVIDOR

server string is the equivalent of the NT Description field

server string = Servidor Samba %v

[GESTION]
comment = Public Stuff
path = /mnt/gestion
public = yes
writable = yes

Esto en el Linux comparte el directorio /mnt/gestion y ahí deberán estar las aplicaciones de la forma:

/mnt/gestion/Aplicaci

Considerando que dentro de Aplicaci esta la aplicación y sus Dbfs

   Aplicaci/programa.exe
   Aplicaci/clientes.dbf

O si tienes SET DEFAULT TO "DATOS"

   Aplicaci/programa.exe
   Aplicaci/DATOS/clientes.dbf

Ejecutar desde Linux con privilegios de usuario "root"

root@SERVIDOR~]#chmod -R 777 /mnt/gestion <Intro>

( root@SERVIDOR~]# ) es el prompt de Linux

Para establecer los permisos

y en windows hay que conectar a unidad de red con el comando:

C:\>net use s: \SERVIDOR\GESTION

Desde el ordenador windows deberás ver la unidad "S" de la forma:

S:\Aplicaci siendo Aplicaci una carpeta de la unidad "S"

Y dentro de Aplicai estará el ejecutable
