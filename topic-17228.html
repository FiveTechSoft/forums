The new Update 4.1 with many new Functions und included GDI-fixes.
In case You still find Problems, just let me know.

Download : http://www.pflegeplus.com/fw_downloads/Vtitles6.zip ( 7.1 MB )

You Install-Directory needs 5 Sub-directorys :
/System ( System-Buttons )
/Set1 ( Templates )
/Set2
/Set3
/Set4
A created EXE-File of Your Project, will be saved in the Install-Directory.
Freeimage.dll must be included.

4 different Template-Sets are included.
It is much better now, to arrange different VTitle-Styles
With a straight Painter-Button select, SET1 is loaded.
Allways the 1. Project of the Set is loaded ( Top-Record )



Selecting a Set, a Browser shows the Preview of all included Projects of the Set.
The Index-Top-Position will be selected.



From inside the Painter, You can select / change the activated Project.



For the Browser, You need small Screenshots of the VTitle.
There is a new Button for a Screenshot included.
Cut ad Resize the BMP and define the Screenshot inside the Project-Manager.



You can add the Screenshot to the Project-Browser.
Just define a name and choose the Button < Template >



The Text-Position-Slider-Range is changed, to make it possible, to connect
vertical Textlines.
