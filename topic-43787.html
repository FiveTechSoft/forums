Seré breve ya que este tema se ha tocado muchas veces en el foro.

Buscando la manera de como enviar mensajes desde mi sistema FW+HARBOUR vía whatsapp, logré hacerlo a medias, utilizando una hoja Excel siguiendo lo que se explica en el siguiente video de youtube (espero que no haya problema con poner enlaces hacia otros sitios dentro del foro). Cabe resaltar que la hoja Excel es de uso libre.

https://youtu.be/JrhE2Vx2b18?si=a-1F_inpM6ajZOfe

Primero hice unas adecuaciones de gusto personal a la hoja y la deja asi, ya que me sirve para mi caso:

http://www.datanet.space/fw/mensajesexcel.xlsm

El procedimiento que usé es: desde mi programa FW preparo un browse con los datos necesarios para el envio de los mensajes. Ese browse lo exporto a una hoja de excel temporal desde la cual, el usuario de mi programa debe hacer un "copy&paste" a la hoja sacada de youtube que enviara los mensajes. Hasta ahora ha funcionado con 2 de mis clientes que usan mi software de cuentas por cobrar. Están enviando los alertas de cobranza exitosamente.

Ahora bien, los envíos los hace una macro diseñada con visual basic. La pregunta es: ¿será posible que alguno de los master pueda adecuarlo para usarse como una función fw?. Hay otros temas secundarios como la posibilidad de baneo por parte de whatsapp si se abusa con el uso de la cuenta, pero ya eso seria otro tema. Como comenté, esta funcionando en dos de mis clientes desde hace un par de meses y no ha habido problemas.

Dejo la inquietud abierta a ver si nos puede ser útil a quienes necesitamos poder hacer los envíos directos desde nuestras aplicaciones.

Estimado José,

Puedes hacerlo directamente desde FWH sin necesidad de crear una hoja de cálculo ni tener que usar VBA, etc

Simplemente usa esto desde FWH en un bucle:

ShellExecute( 0, "open", StrTran( "whatsapp://send?phone=" + cTelefono + "&text=" + cMensaje ), " ", "%20" ) )

SysRefresh() // ó SysWait( 5 ) prueba con distintos valores hasta que te funcione bien

CreateObject( "WScript.Shell" ):SendKeys( "~" )

Este ejemplo funciona correctamente. Es necesario instalar la app de whatsapp y debe estar funcionando antes de ejecutar nuestra llamada:

whatsapp.prg
Code (fw): Select all Collapse
#include "FiveWin.ch"

function Main()

   ShellExecute( 0, "open", "whatsapp://send?phone=34722461100&text=FWH," + Time() )
   SysWait( 1 )
   CreateObject( "WScript.Shell" ):SendKeys( "~" )

return nil
To send msgs to a whatsapp group instead of a phone number use this:
whatsapp://send?text=encoded_message&phone=group_id
Find out the group ID of the group you want to send the message to. You can do this by opening the group info screen in WhatsApp and tapping on the invite link. The group ID is the part after “chat.whatsapp.com/”. For example, if the invite link is “https://chat.whatsapp.com/ABCDEF123456”, then the group ID is “ABCDEF123456”
