Francesco,

Questo è il codice della funzione ShowKeyboard():
Code: Select all Collapse
HB_FUNC( SHOWKEYBOARD )
{
   SipShowIM( SIPF_ON );
}

come si vede, si usa la funzione diretta di Windows. qui sono i possibili valori di utilizzare:
c:\vce\include\arm\sipapi.h:
Code: Select all Collapse
#define SIPF_OFF	0x00000000
#define SIPF_ON 	0x00000001
#define SIPF_DOCKED	0x00000002
#define SIPF_LOCKED	0x00000004

Ci sono altre funzioni che nel file che si può rivedere.

Quando acquistare FWPPC acquisti si ha diritto a due mesi di aggiornamenti gratuiti. Da quel momento è passato, ora è possibile acquistare un aggiornamento alla nuova versione di acquisto da FWPPC http://www.fivetechsoft.com sezione purchase "FWPPC upgrade"

Documentazione di FWPPC di FWH è lo stesso tranne che per alcune funzioni non sono disponibili in FWPPC
