There are at least 3 options to go

1) EasyReport (www.reportdesigner.info) Written in FiveWin, includes the visual designer along with a LIB that let you print your reports designed in ER.

2) The most common used (I think) Crystal Reports, which is the "standard" for visual report designers under Windows, the most of the programming languages use it. FiveWin provides two ways to use Crystal, one usign a TCRPE class and other via ActiveX.

3) You can also use List&Label the european equivalent of Crystal Reports, you can handle from FiveWin using ActiveX also (http://en.combit.net/reporting_tool/page1122.aspx)
