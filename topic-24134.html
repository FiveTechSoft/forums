Code (fw): Select all Collapse
How can mimic CA-Clipper's Keyboard command?

Use this function:

Function fwKeyboard(oWnd,nKey)

oWnd:SetFocus()
oWnd:PostMsg( WM_KEYDOWN, nKey, 0)

Return NIL
