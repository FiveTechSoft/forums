Luego, para evitar porblemas nos vamos al Method Destroy()

Code (fw): Select all Collapse
METHOD Destroy() CLASS TFolder

   local n

* Código Nuevo
   IF ::oFont2 != Nil
       ::oFont2:End()
   ENDIF
* Fin Código Nuevo

   if ::oFont != nil
      ::oFont:End()
   endif

   for n = 1 to Len( ::aDialogs )
      ::aDialogs[ n ]:bValid = { || .t. }
      ::aDialogs[ n ]:End()
   next

return Super:Destroy()


Luego en la Cabecera Folder.Ch Hacer la siguiente modificación

Code (fw): Select all Collapse
#xcommand REDEFINE FOLDER [<oFolder>] ;
             [ ID <nId> ] ;
             [ <of: OF, WINDOW, DIALOG> <oWnd> ] ;
             [ <prm: PROMPT, PROMPTS, ITEMS> <cPrompt,...> ] ;
             [ <dlg: DIALOG, DIALOGS, PAGE, PAGES> <cDlgName1> [,<cDlgNameN>] ] ;
             [ <color: COLOR, COLORS> <nClrFore> [,<nClrBack>] ] ;
             [ OPTION <nOption> ] ;
             [ ON CHANGE <uChange> ] ;
             [ <lAdjust: ADJUST> ] ;
         [ FONT <oFont> ];
       => ;
          [<oFolder> := ] TFolder():ReDefine( <nId>, [\{<cPrompt>\}],;
             \{ <cDlgName1> [,<cDlgNameN>] \}, <oWnd>,;
             <nOption>, <nClrFore>, <nClrBack>,;
             [{|nOption,nOldOption| <uChange>}], <.lAdjust.>,<oFont> )


Y eso es todo, una vez compiladas las modificaciones en la lib, podremos definir Fonts para la Folder y los diálogos contenidos en la folder con el método ReDefine. Espero sea de utilidad mi aporte.
