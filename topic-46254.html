Paisano porque no compilas con HBMK2.EXE ?
Es muy facil,
te envio un ejemplo:

GO.BAT

Code: Select all Collapse
set path=c:\compile\BCC770\bin
if exist ScPHS.exe del ScPHS.exe
c:\compile\harbour\bin\hbmk2 ScPHS.hbp -comp=bcc
if exist ScPHS.exe ScPHS.exe

PROY.HBP

-I.\Includes
