Hello,

A customer of mine uses my application for many years without any problem.

A few weeks ago, he installed a new network :
- a server with Windows 2008 Small Business Server 64-bit
- new workstation with Windows 7 Professional 32-bit

My application is build with FWH 10.04 and the latest xHarbour Builder.

Michel,

this could be a problem with SMB 2.0 protocoll, microsoft had enhanced this protocoll in server 2008. Try to disable SMB 2.0

Here are two links with explanations how to do it.
