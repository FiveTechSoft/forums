Byron,

We do recommend to use the free xHarbour version that you can download from here:

https://bitbucket.org/fivetech/harbour-xharbour-builds/downloads/xharbour_1.2.3_32bits_bcc7_20150603.zip

It is the one that we use to test xHarbour and FWH (current version).

xharbour commercial version is based on PellesC and we have to use VC98 (yes, from 98!) to build FWH for it and as
VC98 headers are outdated, we always have issues with it.

My advise: Use Harbour and FWH. In case that you want to use xHarbour, then use the free xHarbour that we use with FWH,
so you can use the modern Borland bcc7 with it.

Sorry for the slow reply. Some of us are using FWH with Harbour ( Antonio has a distribution ) and FWH plus Microsoft VC 2013.

xHarbour ( xBuilder ) with FWH 1510
