El primer browse que se implementó en FW es el TWBrowse. Es muy rápido, ya que se diseñó para ordenadores lentos.

A partir de ese, se deriva la clase TCBrowse, que ya usa objetos columna.

Finalmente el TXBrowse es el más completo y por lo tanto el más complicado de usar.

A parte de esos tres propios de FWH, varios usuarios han implementado otros browses, como el de Hernán, Manuel Mercado, etc.
