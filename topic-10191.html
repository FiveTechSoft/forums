I use the sort option when I have multiple search possibliities .. consider this code :

oRs:Sort := "fname"
oRs:MoveFirst()
oRs:Find("fname = '"+cFIND+"'" )

oLBX:ReFresh()
SysReFresh()

.. or

oRs:Sort := "lname"
oRs:MoveFirst()
oRs:Find("lname = '"+cFIND+"'" )

oLBX:ReFresh()
SysReFresh()

..
