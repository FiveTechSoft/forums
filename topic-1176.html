Sliders work only before speech begins.

If you want to change volume and pause while speaking, you must use ::TagSpeech(nVolume,nmsecsPause) method.

Accordingly to documentation not all computers support ::TagSpeech.

Note that volume, speed and pitch ranges are 0..100
