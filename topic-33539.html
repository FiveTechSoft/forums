Use the following values for nRow and nCol (or something similar):

Code (fw): Select all Collapse
nRow = oRpt:nRow - oRpt:nStdLineHeight + oRpt:nStdLineHeight / 2
nCol = oRpt:oDevice:nHorzRes() / 5
