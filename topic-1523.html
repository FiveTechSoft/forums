La nueva capacidad de los diálogos de usar bitmaps como fondo está preparada para probarla a fondo, antes de que distribuyamos el build de Febrero de FWH.

Si eres usuario de FTDN, ó usuario de FWH 2.7 Enero, y estás interesado en probarlo ya, por favor solicítalo. Dejaremos de aceptar solicitudes una vez haya un numero suficiente de probadores.

Para usar esta nueva característica es tan simple como:
Code: Select all Collapse
   DEFINE BRUSH oBrush RESOURCE "Sky"
   
   DEFINE DIALOG oDlg RESOURCE "Test" BRUSH oBrush TRANSPARENT

   ...

Así de sencillo
