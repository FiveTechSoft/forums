Antonio,

La teoría es que sí se puede usar XP como servidor de fax. Al realizar la conexión hay que espicificar el nombre del equipo que hace de servidor de fax, esto es:

Code: Select all Collapse
oFax:Connect ( "computername" )


Esto es lo que hay a este respecto:

Connecting to the Fax Server

Most programmatic fax management requires you to connect to the fax server. You can either specify the full path and name of a remote server, or use an empty string to connect to the local fax server.
