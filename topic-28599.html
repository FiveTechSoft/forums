Also, make sure the "usertype.dat" file is copied to "C:\Program Files (x86)\Microsoft Visual Studio 11.0\Common7\IDE" and not "C:\Program Files\Microsoft Visual Studio 11.0\Common7\IDE"


usertype.dat
Code (fw): Select all Collapse
DEFINE
WINDOW
DIALOG
FONT
ACTIVATE
FROM
TITLE
COLOR
SAY
ON
RIGHT
CLICK
VALID
RESOURCE
CENTERED


Ojo que conforme mejoremos usertype.dat, hay que salir de Visual Studio y volver a entrar para que se activen las nuevas definiciones!
