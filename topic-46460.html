Gracias César, no me ha vuelto a molestar hasta ahora.

Via .BAT: GUARD.BAT

Code: Select all Collapse
@echo off
:: Verifica se o script está rodando como Administrador
net session >nul 2>&1
if %errorLevel% == 0 (
    echo Rodando com privilegios de Administrador...
) else (
    echo Por favor, execute este script como Administrador.
    pause
    exit
)

echo Desativando Device Guard, Credential Guard e VBS...

:: 1. Desativar via Registro
reg add "HKLM\SYSTEM\CurrentControlSet\Control\DeviceGuard" /v "EnableVirtualizationBasedSecurity" /t REG_DWORD /d 0 /f
reg add "HKLM\SYSTEM\CurrentControlSet\Control\Lsa" /v "LsaCfgFlags" /t REG_DWORD /d 0 /f
reg add "HKLM\SYSTEM\CurrentControlSet\Control\Lsa" /v "CredentialGuard" /t REG_DWORD /d 0 /f

:: 2. Desativar Hypervisor e Plataforma de Maquina Virtual (opcional, mas recomendado)
bcdedit /set hypervisorlaunchtype off
dism /online /disable-feature /featurename:Microsoft-Hyper-V-All /norestart
dism /online /disable-feature /featurename:VirtualMachinePlatform /norestart
dism /online /disable-feature /featurename:HypervisorPlatform /norestart

echo.
echo Processo concluido.
echo IMPORTANTE: Reinicie o computador para aplicar as alteracoes.
pause

Super many thanks.

Ese script desactiva Device Guard, Credential Guard y VBS (Virtualization Based Security) en Windows. Es un script
legítimo que algunos desarrolladores necesitan porque:

  • Hyper-V y VBS bloquean VirtualBox/VMware — no pueden coexistir con el hypervisor de Windows

  • Credential Guard puede interferir con apps legacy que acceden a credenciales de red de forma directa

  • Penalización de rendimiento — VBS añade overhead que puede afectar a apps que hacen muchas llamadas al kernel

    El script está bien escrito. Lo único que sugeriría al usuario:

  • Que lo guarde como .bat y lo ejecute como Administrador (botón derecho → "Ejecutar como administrador")

  • Que reinicie después de ejecutarlo — es obligatorio para que surtan efecto los cambios

  • Si en el futuro necesita revertir, cambiar los 0 por 1 en los reg add y usar bcdedit /set hypervisorlaunchtype auto
