Función Skipper2:
Code: Select all Collapse
Function Skipper2(Vector,nCur,nSkip)
Local nOld:=nCur

Default nSkip:= 1
nCur :=nCur + nSkip
IF nCur > LEN(Vector)
   nCur:= LEN(Vector)
ENDIF
IF nCur < 1
   nCur:= 1
ENDIF

Return nCur - nOld
