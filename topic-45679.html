lets build it:

1. We create a libcurl.def file using c:\bcc77\bin\impdef.exe

c:\bcc77\bin\impdef libcurl.def libcurl.dll

2. We create the import lib using PellesC polib.exe

polib /def:libcurl.def /out:libcurl.lib
