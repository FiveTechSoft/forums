Las más comunes:

Control - C Copiar Texto
Control - X Cortar Texto
Control - V Pegar Texto
Control - P Imprimir
Control - G Guardar Archivo
Control - E Seleccionar Todo el Texto
Control - N Nuevo Documento
Control - A Abrir Documento
Control - Z Deshacer
Control - Y Rehacer
Control - B Buscar Palabra
Control - F4 Cerrar Documento
Control - + Aumentar Zoom
Control - * Zoom Normal
Control - - Disminuir Zoom
Control + I Ir a Linea
Control + F9 Compilar como Script
F9 Compilar como Exe en Archivo
Control - D Compilar con Debuger
