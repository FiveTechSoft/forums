Buttons sended

Maybe You still need some Borders, to create different Button-styles ?
( easy to create using PIXELFORMER )

Formats : PNG, BMP and ALPHA



Download :
http://www.pflegeplus.com/fw_downloads/borders.zip

Creating a new Button with Border :

1. Import the Button < Menu File > from the Collection, You want to add a Border
2. Resize the Button to 50 x 50 or smaller < Menu Edit >
3. use < Select all from the menu Edit >
4. use < Copy from the menu Edit >



5. Import the Boder-button You want to use
6. use < Paste from the menu Edit > ( the Button will be added centered to the Border-button )
7. use < Export Menu File> with new Filename



To create a round Button, You can make the Corners of the selected Button transparent
to keep the Circle of the new Button





You can create a complete different Button using 2 ore more resized Buttons
merged to a new one.

1. open Pixelformer twice
2. Inside the 1. instance select the 1. Button and resize
3. Inside the 2. instance open a Border-button
4. copy the modified Button from the 1. Window
5. select the 2. Window and paste.
6. move the merged Button to the Position inside the Border-button
7. select the 2. Button in Window 1, resize, move to Window 2 and paste.

You can copy just a part of a Button, creating a new one !!
