tienes que definir el objeto de los radiobuttons antes de mostrarlos si quieres que aparezcan refrescados, y para ello simplemente defínelos utilizando la clausula ON INIT en el ACTIVATE DIALOG:

Ejemplo:
Code (fw): Select all Collapse
Define dialog oDlg ...
 redefine radio 1...
 redefine radio 2...
 ...
Activate dialog oDlg center ON INIT refresh_Radio()
