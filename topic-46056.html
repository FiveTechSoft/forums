[header]Step[/header]

[row]
[cell]1. Clone Repo[/cell]
[cell]cd ~

git clone https://github.com/karpathy/nanochat

cd nanochat[/cell]
[cell]Clone the standard nanochat repository from Karpathy.[/cell]
[cell]Creates ~/nanochat directory with all source files.[/cell]​
[/row]
[row]
[cell]2. Create venv[/cell]
[cell]python3.10 -m venv .venv

source .venv/bin/activate[/cell]
[cell]Set up Python 3.10 virtual environment.[/cell]
[cell]Activates .venv; use uv for faster package management.[/cell]​
[/row]
[row]
[cell]3. Install Dependencies[/cell]
[cell]uv sync[/cell]
[cell]Install PyTorch, CUDA libs, maturin, and other deps (~3-5 min).[/cell]
[cell]Resolved 91 packages; CUDA: True, GPU: NVIDIA GeForce RTX 3060.[/cell]​
[/row]
[row]
[cell]4. Verify Torch/CUDA[/cell]
[cell]uv run python -c "import torch; print(f'CUDA: {torch.cuda.is_available()}, GPU: {torch.cuda.get_device_name(0) if torch.cuda.is_available() else None}')"[/cell]
[cell]Confirm CUDA and GPU detection.[/cell]
[cell]Output: CUDA: True, GPU: NVIDIA GeForce RTX 3060.[/cell]​
[/row]
[row]
[cell]5. Install nanochat Editable[/cell]
[cell]uv pip install -e .[/cell]
[cell]Editable install (symlinks to source).[/cell]
[cell]nanochat==0.1.0 installed; enables module imports.[/cell]​
[/row]
[row]
[cell]6. Verify nanochat Imports[/cell]
[cell]uv run python -c "from nanochat.gpt import GPT, GPTConfig; print('nanochat installed OK')"[/cell]
[cell]Test core model imports.[/cell]
[cell]Output: nanochat installed OK.[/cell]​
[/row]
[row]
[cell]7. Compile Rust Tokenizer[/cell]
[cell]uv run maturin develop --release --manifest-path rustbpe/Cargo.toml[/cell]
[cell]Build RustBPE tokenizer (~2-5 min).[/cell]
[cell]📦 Built wheel; RustBPETokenizer available.[/cell]​
[/row]
[row]
[cell]8. Verify Tokenizer[/cell]
[cell]uv run python -c "from nanochat.tokenizer import RustBPETokenizer; print('Tokenizer compiled OK')"[/cell]
[cell]Confirm tokenizer module.[/cell]
[cell]Output: Tokenizer compiled OK.[/cell]​
[/row]
[row]
[cell]9. Set PYTHONPATH[/cell]
[cell]export PYTHONPATH="$(pwd):$PYTHONPATH"[/cell]
[cell]Add repo root to Python path for multiprocessing.[/cell]
[cell]Permanent: Add to ~/.bashrc; required for torchrun.[/cell]​
[/row]
[row]
[cell]10. Download Dataset[/cell]
[cell]PYTHONPATH="$(pwd):$PYTHONPATH" uv run python -m nanochat.dataset -n 40[/cell]
[cell]Download 40 FineWeb-EDU shards (~4 GB, 5-15 min).[/cell]
[cell]Downloading 8 shards using 4 workers...; Target: ~/.cache/nanochat/base_data/.[/cell]​
[/row]
[row]
[cell]11. Verify Dataset[/cell]
[cell]ls ~/.cache/nanochat/base_data/ | wc -l

du -sh ~/.cache/nanochat/base_data/[/cell]
[cell]Check downloaded shards.[/cell]
[cell]40 files; ~4 GB total (shard_00000.parquet to shard_00039.parquet).[/cell]
[/row]
[row]
[cell]12. Train Tokenizer (Optional)[/cell]
[cell]uv run python -m scripts.tok_train --max_chars=2000000000[/cell]
[cell]Train custom BPE tokenizer (~10-20 min CPU).[/cell]
[cell]Generates tokenizer.json (~65K vocab); Edit vocab_size=32768 for smaller model.[/cell]​
[/row]
[row]
[cell]13. Eval Tokenizer (Optional)[/cell]
[cell]uv run python -m scripts.tok_eval[/cell]
[cell]Test tokenizer compression (~1 min).[/cell]
[cell]Compression ratio ~4.8 chars/token (vs. GPT-4).[/cell]
[/row]
[row]
[cell]14. Pretrain Model[/cell]
[cell]PYTHONPATH="$(pwd):$PYTHONPATH" uv run torchrun --standalone --nproc_per_node=1 scripts/base_train.py --depth=10 --device_batch_size=4 --max_seq_len=1024 --compile --num_iterations=800[/cell]
[cell]Base training: depth=10 (~100M params, ~1-2 hours on RTX 3060).[/cell]
[cell]Overriding: depth=10...; Loss ~11.09 → ~2.0; Checkpoints in checkpoints/d10/; Monitor with watch -n 1 nvidia-smi (VRAM ~7 GB).[/cell]
[/row]
[row]
[cell]15. Midtrain (Post-Pretrain)[/cell]
[cell]PYTHONPATH="$(pwd):$PYTHONPATH" uv run python scripts/mid_train.py --model_path checkpoints/d10[/cell]
[cell]Add reasoning datasets (SmolTalk/MMLU/GSM8K, ~10 min, 4 GB VRAM).[/cell]
[cell]Improves chat coherence; Output: checkpoints/d10/midtrain.pt.[/cell]​
[/row]
[row]
[cell]16. Supervised Fine-Tuning[/cell]
[cell]uv run python scripts/sft.py --batch_size=2[/cell]
[cell]Align to ChatGPT-style responses (~5 min, 2 GB VRAM).[/cell]
[cell]Enables conversational capabilities.[/cell]​
[/row]
[row]
[cell]17. Evaluation[/cell]
[cell]uv run python scripts/eval.py[/cell]
[cell]Run benchmarks (~5 min).[/cell]
[cell]MMLU ~25-35%, GSM8K ~5-10%, HumanEval ~8%, ARC-Easy ~30%.[/cell]​
[/row]
[row]
[cell]18. Launch Web UI[/cell]
[cell]uv run python -m nanochat.webui[/cell]
[cell]Start ChatGPT-like interface.[/cell]
[cell]Localhost:8000; Test with trained d10 model.[/cell]​
[/row]
[row]
[cell]19. Full Pipeline (Alternative)[/cell]
[cell]bash speedrun.sh[/cell]
[cell]Automated script (edit base_train.py depth=10 for RTX 3060).[/cell]
[cell]Handles tokenizer → pretrain → midtrain → SFT → eval → UI; Generates report.md.[/cell]​
[/row]
