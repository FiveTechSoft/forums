Jose:

Esa es "la pequeña diferencia", el uso LIBRE hoy, no quiere decir que el dia de mañana no se les suba la mosca a la cabeza y comiencen a cobrar.

En cuanto a MySQL la cosa esta clara, si tu utilizas la base de datos para actividades NO LUCRATIVAS por ejemplo llevar el censo de las ballenas de baja california, y almacenar los datos de calentamiento global, entonces su uso no tiene costo.

Peeero, si tu utilizas MySQL PARA CUALQUIER COSA que genere un beneficio economico a ti, a tu cliente o a un tercero, entonces debes de pagar la licencia de uso, si lo usas para llevar la contabilidad, la facturacion, etc. todo lo que tenga que ver con ganar o gastar dinero, tiene que pagar licencia, aun si lo utilizas internamente en tu empresa para tu administracion.
