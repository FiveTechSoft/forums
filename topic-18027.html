Acabo de probarlo y sip si se pudo, los TABS ya funcionan, nada más que luego de hacer lo que me dice el Maestro Antonio, se debe mandar a reconstruir las pestañas con oFolder:SetPrompts().
Code (fw): Select all Collapse
    @30,30 FOLDER oFold ;
    PROMPT "Datos Gen","Pestaña 2" pixel of ::oWinGen 
    oFold:aEnable = { .t. ,.t.}
    oFold:LoadPages(  { "DATOS" }, { || RedefineLosControles( oFold,aodat,adat,aoImg,aImg ) } )
