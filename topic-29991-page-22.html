- Added Templates of code user define
Templates are activated by holding down the CTRL + ALT keys and without releasing them, click on the desired template (template codes have two digits: for example 01, 09 or 99)
The user has 99 templates available to define: from 01 to 99
If you select one template blank, or you press code of template blank: dialog for definition is shown
Also you you can define, delete or modify the templates in the configuration screen
