Instead of cSql4 := oCon4:UpdateSQL(...) and then oCn4:Execute( cSql4 ), we can
Code (fw): Select all Collapse
oCon4:Update( "balances", "month3,month4", { "MAR", "APR" }, { "ID", number } )

Code (fw): Select all Collapse
oRs4:Update( "month3,month4", { "MAR", "APR" } )
