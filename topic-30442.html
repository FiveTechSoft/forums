Jairo,

Aqui tienes un ejemplo muy sencillo usando QTextEdit():

Code (fw): Select all Collapse
#include "FiveTouch.ch"

function Main()

   local oDlg, oText

   DEFINE DIALOG oDlg

   oText = QTextEdit( oDlg )

   oText:SetText( "Hello world" )

   ACTIVATE DIALOG oDlg CENTERED

return nil
