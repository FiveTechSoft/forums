TSButton y Ventana

Posted: Tue Aug 03, 2010 12:01 PM
Hola Amigos,

Tengo una aplicación con una ventana principal, en ella he puesto un botón de la clase TSButton
y me lo muestra. El problema es que al pasar el ratón sobre el botón, no se activa ni permite
hacer click. Es como si estuviera en un segundo plano.

Os paso una muestra de mi código:

Code (fw): Select all Collapse
DEFINE WINDOW oWnd MDI ;
     TITLE cTitulo ;
     ICON oIcon ;
     COLORS CLR_BLUE, CLR_WHITE MENU oMenu

   DEFINE BUTTONBAR oBar OF oWnd SIZE 30,50 2007

   DEFINE BUTTON oBtn4 OF oBar ;
   TOOLTIP OemToAnsi( "Administración de Propietarios" ) ;
   RESOURCE "admprop_btn" ;
   ACTION( MsgInfo( "hola" ) )  ;
   MESSAGE "hola"

// Este es el que falla.
@ 5,5 SBUTTON oBtnAvisos OF oWnd ;
   TOOLTIP "Muestra los avisos pendientes." ;
   MESSAGE "Muestra los avisos pendientes." ;
   COLORS { | oBtnAvisos | If( oBtnAvisos:lMouseOver, CLR_RED, CLR_BLACK ) }, CLR_LIGHTBLUE ;
   RESOURCE "MAS" ;
   OFFICE ;
   ACTION MsgInfo( "hola" )

ACTIVATE WINDOW oWnd MAXIMIZED


Es la primera vez que intento poner un botón en una ventana sin que dependa de la barra de botones ni integrado en un dialogo, supongo que algo se me escapa.

Si alguien tiene alguna idea, bienbenida sea.

Hola,

Tines que asignarlo a oWnd:oWndClient

Code (fw): Select all Collapse
@ 5,5 SBUTTON oBtnAvisos OF oWnd:oWndClient  ;
   TOOLTIP "Muestra los avisos pendientes." ;
   MESSAGE "Muestra los avisos pendientes." ;
   COLORS { | oBtnAvisos | If( oBtnAvisos:lMouseOver, CLR_RED, CLR_BLACK ) }, CLR_LIGHTBLUE ;
   RESOURCE "MAS" ;
   OFFICE ;
   ACTION MsgInfo( "hola" )
