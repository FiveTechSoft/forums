Luis,

Más simple aún que usar un webservice es llamar al siguiente código en PHP alojado en el servidor desde tu aplicación funcionando en FiveTouch:

inserta.php
Code (fw): Select all Collapse
<?php
$servername = "localhost";
$username = "username";
$password = "password";
$dbname = "myDB";

// Create connection
$conn = new mysqli($servername, $username, $password, $dbname);
// Check connection
if ($conn->connect_error) {
    die("Connection failed: " . $conn->connect_error);
} 

$sql = "INSERT INTO installs (username,computername,language,nation,exename,ipaddress)
VALUES (?,?,?,?,?,?)";

if ( $stmt = $conn->prepare( $sql ) ) 
{
  $stmt->bind_param("sssiss", $user, $computer, $lang, $nat, $exe, $ip );
  $user = mysqli_real_escape_string( $conn, $username );
  $computer = mysqli_real_escape_string( $conn, $computername );
  $lang = mysqli_real_escape_string( $conn, $language );
  $nat = $nation;
  $exe = mysqli_real_escape_string( $conn, $exename );
  $ip = mysqli_real_escape_string( $conn, $ipaddress );

  $stmt->execute();
  $stmt->close();
}

$conn->close();
?>

Albeiro.

Apple es muy estricto con la autorización para poder instalar una app en el teléfono ó tablet asi que tienes
que construir la app de FiveTouch tu mismo usando el proyecto que proporcionamos:

https://github.com/FiveTechSoft/fivetouch/blob/master/ios/fivetouch.pro

Abres ese fichero desde QT Creator, construyes FiveTouch y FiveTouch se instalará en tu teléfono ó tablet usando tu firma digital.
Esta firma digital la genera la aplicación Xcode de Apple al construir un ejemplo de prueba, creándose un "Provisioning Profile".

Este proceso anteriormente era bastante complicado, afortunadamente ahora es muy sencillo y se realiza de forma automática
desde la aplicación Xcode de Apple. Por supuesto tienes que usar un ordenador Apple para poder correr Xcode.
