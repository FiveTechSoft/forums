Horacio:

Este es un ejemplo de conexion

Code (fw): Select all Collapse
oCon   := FW_OpenAdoConnection( { "MYSQL", "localhost", "TuDataBase", "root", "TuContraSeña"}, (.T.) )
IF oCon == nil
      MsgInfo( "No se pudo lograr la conexión al servidor, REVISE LA CONEXION DE SU RED O LA CONEXION A INTERNET !", "Error" )
      FW_ShowAdoError(oCon)
      RETURN( .F. )
ELSE
ENDIF


También revisa tu antivirus y si tienes abierto el puerto 3306, es el puerto por default.
