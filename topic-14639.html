Yo lo que hago es añadirle
Code (fw): Select all Collapse
When Publicas:cAccesot=.T.


Donde Publicas:cAcccesot es una variable publica que se carga al iniciar la aplicación con valor .F., y cuando la aplicación te pregunta la clave de usuario, pues dependiendo del tipo de acceso de permite o no.
Todas las variables plublicas las guardo en datas dentro de una función.
