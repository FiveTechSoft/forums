James,

This seems a right fix:
Code (fw): Select all Collapse
   if ::hWnd != 0
      Super:Destroy()
   else
      if ::oBrush != nil
         ::oBrush:End()
      endif
   endif

Because if we End() a brush once more than needed we take the risk to destroy the brush that may be in use by other controls.
