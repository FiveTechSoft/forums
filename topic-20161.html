I figured it out

It was a poorly used ".and." that messed everything up.

I had done something like this:
Code (fw): Select all Collapse
IF nRow > 275 .and. nGraphSize = 0


Where it should have been:
Code (fw): Select all Collapse
IF nGRaphSize = 0
   IF nRow > 275
