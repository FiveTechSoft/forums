include in your rc file ( resource with name AppIcon )

AppIcon ICON DISCARDABLE ".\MyIcono.ico"

in your application

DEFINE ICON oIcon RESOURCE "AppIcon"

Set icon to windows object
Code (fw): Select all Collapse
      oWnd:SetIcon( oIcon )
