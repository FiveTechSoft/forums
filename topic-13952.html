la idea es crear un entrono de configuracion general que sirva para configurar individualmente los proyectos, donde se le puede indicar al ide, carpetas, librerias, compiladores, recursos, etc...

cada pestalla del modulo de configuracion habla por si misma solo al verlas a simple vista, les dejo una breve explicacion de cada una

Project Setup...
Permite indicar las diferentes carpeta donde se alojaran los distitos archivos que se utilizaran en el ide, aqui se pueden incluir las librerias de terceros, el compilador que se desa usar y si se va a utilizar el tema de windows.

Compiler...
localizacion de las carpetas de FWH
estan cargadas todas las opciones comunes de los compiladores de H y xH

Harbour y xHarbour...
aqui se cargan los diferentes directorios donde se ubican los archivos de los compiladores, compilador, librerias, CH y si desea utilizar alguna otra opcion que no se encuentre entre las opciones de la pestaña "compiler" lo puede hacre desde aca, de encontrar correctamente el directorio de las librerias, estas son cargadas automaticamente y se podran escojer manualmente cada una o seleccionar por default las que se usan comunmente

las letras rojas indican que el directorio o archivo es inexistente.

linker...
aqui se ubican los diferentes archivos que nos van a permiter hacer la compilacion y link final, las librerias que se usan para ese proceso y el tipo de recurso a utilizar, no se podran agregar librerias de directorios inexistentes...
esta es una imagen que muestra como se veria si encontrara correctamente los directorios y archivos



cada uno de estos valores son modificables por proyecto sin necesidad de alterar los generales
our best documentation is the source code
