Hemos comenzado una nueva implementación de FiveWeb para ser programada y usada con Harbour

Esta disponible libremente hasta que lleguemos a la versión 1.0. Podeis conseguirla desde aqui:
http://code.google.com/p/fiveweb/

Todo el código fuente incluido y ejemplos:
http://code.google.com/p/fiveweb/downloads/detail?name=fiveweb_0.1.zip&can=2&q=

Se compila con Harbour y se ejecuta desde el servidor como un CGI:
Code (fw): Select all Collapse
#include "FiveWeb.ch"

function Main()

   MsgInfo( "Hello world from FiveWeb" )

return nil
