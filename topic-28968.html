Si quieres que el browse sea editable, puedes hacer:

oBrw:nEditTypes := EDIT_GET

para que una determinada columna sea editable:

oCol:nEditType := EDIT_GET (u otros valores segun lo que vayas a editar)

Adrian,

Para hacer que una columna no sea editable, simplemente haz oCol:lEditable := .F.

Si quieres que el ser editable dependa de una condición, entonces usa este codeblock:

oCol:bEditWhen = {| oCol | ...condición que ha de devolver un valor lógico... }

Para saltar de una columna a otra determinada, haz como te ha indicado Jose Luis.

Para hacer una columna no editable:

oCol:nEditType := 0 ( este estado es por defecto)

Para ir a una columna específica:

 oBrw:GoToCol(  nCol )  // nCol es el ordend e creación de la columna

 ó

 oBrw:GoToCol( &quot;cCabecera de la columna&quot; )

 ó

 oBrw:GoToCol( oObjetoColumna )
