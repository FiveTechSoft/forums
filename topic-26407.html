Mr Silvio

I have an array of many colors

Function InitColorsArray()


Please refer to your first posting:
After initializing the two arrays
aClrNames := {}
aColorNum := {}
This single line code is all that is enough to give you the two arrays as you wanted.
Code (fw): Select all Collapse
   AEval( aColors, { |a| AAdd( aClrNames, a[ 1 ] ), AAdd( aColorNum, StrTran( a[ 2 ], ' ', ',' ) ) } )
   xbrowse( { aclrnames, aColorNum } ) // this line is for testing the result

But to have the colors in the format "252,200,200" is not useful to you. Instead you want the RGB computed number of the colors.
For this use the following code.
Code (fw): Select all Collapse
   AEval( aColors, { |a| AAdd( aClrNames, a[ 1 ] ), ;
      AAdd( aColorNum, nRGB( Val( Token( a[ 2 ], ' ', 1 ) ), ;
                             Val( Token( a[ 2 ], ' ', 2 ) ), ;
                             Val( Token( a[ 2 ], ' ', 3 ) )  ) ) } )
   xbrowse( { aclrnames, aColorNum } ) // only for testing the result


I do not know if it is possible to paint different lines of combobox in different colors. But you can show different color bitmaps for each line.
