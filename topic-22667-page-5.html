Hola a todos,
Al usar este código en FiveWeb:
Code (fw): Select all Collapse
<a href="#" onclick="OpenProject()"><li>Open</li></a>

usando href="#", en el anchor, se crea una referencia a la página de donde es llamada... ej: http://www.fivetechsoft.com/fiveweb/# , lo que provoca un incremento de una página en el historial de páginas...

Code (fw): Select all Collapse
<a href="javascript : OpenProject()"><li>Tools</li></a>

No provoca este incremento, y creo sería mejor para llamar a una función desde menú (al menos que alguien sepa de algún inconveniente de usar este método)
