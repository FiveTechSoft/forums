Luis,

Las diferencias de posicionamiento vienen por el calculo que hacemos para centrar los diálogos en el que usamos el ancho y la altura del <body>.

Ya hemos advertido diferencias entre Chrome e IE, y posiblemente tambien las haya con Opera.

Hay que poner algunas trazas en este código y probarlo desde Opera:
Code (fw): Select all Collapse
      this.dlg.style.top    = ( parseFloat( body.parentNode.clientHeight ) / 2 ) - ( parseFloat( this.dlg.style.height ) / 2 );
     this.dlg.style.left   = ( parseFloat( body.parentNode.clientWidth ) / 2 ) - ( parseFloat( this.dlg.style.width ) / 2 );
