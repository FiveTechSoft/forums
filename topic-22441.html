Daniel

You will have to re-compile your Barlib library .. here is the Lib_fwh.bat file I have modified to recreate the Barlib.lib file with Borland 5.5.

Please make whatever changes you need to the paths and environmental variables.

Hope this helps.

Rick

Code (fw): Select all Collapse
@ECHO OFF
CLS

REM set include=.\INCLUDE;%INCLUDE%
REM set hdir=c:\Xharbour
REM set bcdir=c:\bcc55\bin

C:\XHARBOUR\bin\harbour SOURCE\CODE128.PRG /n /i\fwh904\include;C:\XHARBOUR\include /oOBJ\CODE128.C
C:\XHARBOUR\bin\harbour SOURCE\CODE3DE9.PRG /n /i\fwh904\include;C:\XHARBOUR\include /oOBJ\CODE3DE9.C
C:\XHARBOUR\bin\harbour SOURCE\EAN13.PRG /n /i\fwh904\include;C:\XHARBOUR\include /oOBJ\EAN13.C
C:\XHARBOUR\bin\harbour SOURCE\GO_CODE.PRG /n /i\fwh904\include;C:\XHARBOUR\include /oOBJ\GO_CODE.C
C:\XHARBOUR\bin\harbour SOURCE\BARLIB.PRG /n /i\fwh904\include;C:\XHARBOUR\include /oOBJ\BARLIB.C
C:\XHARBOUR\bin\harbour SOURCE\CODABAR.PRG /n /i\fwh904\include;C:\XHARBOUR\include /oOBJ\CODABAR.C
C:\XHARBOUR\bin\harbour SOURCE\INT25.PRG /n /i\fwh904\include;C:\XHARBOUR\include /oOBJ\INT25.C
C:\XHARBOUR\bin\harbour SOURCE\BANNER.PRG /n /i\fwh904\include;C:\XHARBOUR\include /oOBJ\BANNER.C
