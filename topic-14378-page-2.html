Dear Anser,

We missed this post from you. We are going to publish a new 9.01 build with those missing files.

The error in TestHead.prg is caused by opening the same DBF again and again. A workaround is:
Code: Select all Collapse
   static cAlias := "AAA"

   USE CUSTOMER NEW SHARED Alias ( cAlias += "A" )

We have included this change in the modified 9.01 build that we are going to pubish in some minutes.

Thanks!
