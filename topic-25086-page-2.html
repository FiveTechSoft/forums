EMG,

In the past it was quite a bit of work. It is much easier now. However, in the past I would backup my data ( which is done each day automatically on 2 different drives ). I would then make a stack of DVD's with the programs to install. I reinstalled Windows with a full format first. ( This makes sure my tracks are all aligned under the head, so I get best reads ... and no spurious data ). Then I would install programs. I only installed the ones I was currently using. That always helped. I had the disks for the others if I needed them. Then I copied the data back.

Now it is much easier. I have downloadable installs, or web based installs, for most software. I can keep everything on DVD's and/or the backup drive. I can completely load a new computer now in about 3 hours with everything I need. I'm using Office 2013 ... it installs from the web. I have my UE Studio, FW, Harbour, and some supplemental programs in a directory on my 2 TB backup drives. I also have the full install for Visual Studio 2012 and Visual Expression Suite on that drive. Once I plug in the drive, its time to run a few installs from it onto the new machine, then copy my data folders from the drive ( all programming, distribution, etc are in these ), and I go to work. Documents, photos, etc. are now in the cloud and synchronized locally.

We try so many things in our work, and so many programs we try, and discard, do not clean out properly. This makes it so much easier.
