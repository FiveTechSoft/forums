I do not know how to use Google API to upload files to Google Drive, but this is what I do to copy files to, create files on and manage files on my Google Drive or DropBox.

Code (fw): Select all Collapse
   local cGoogleDriveFolder   := "c:\Users\" + NetName( 1 ) + "\Google Drive\"
   local cDropBoxFolder       := "c:\Users\" + NetName( 1 ) + "\DropBox\"

   CopyFile( "c:\fwh\bitmaps\attach.bmp", cGoogleDriveFolder + "attach.bmp" )
   xbrowser directory( cGoogleDriveFolder + "*.*" ) autosort
   xbrowser directory( cDropBoxFolder + "*.*" ) autosort
