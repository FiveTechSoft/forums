To define the properties of an object and avoid putting on each object name <oBj>: data or <obj>: method

WITH OBJECT <oBj>

: <data1>: = ...
           : <data2>: = ...
           : <data3>: = ...
           : <method> ()
END

equivalent to:

<oBj>: <data1>: = ...
      <oBj>: <data2>: = ...
      <oBj>: <data3>: = ...
      <oBj>: <method> ()

I hope I explained
