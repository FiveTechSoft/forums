Son varios temas:

  1. Para hacer scroll automático del contenido de una ventana podeis revisar fwh\samples\TestWScr.prg

  2. En un entorno MDI las barras de scroll de la ventana principal ya aparecen automaticamente y permiten desplazar la posición de las ventanas MDICHILDs.

  3. En un diálogo no tenemos implementado un scroll de forma automática. Eso es algo que hay que hacer via programación, ya que el API de Windows no lo proporciona de forma automática.
