Instruccione para instalar harbour y UT en Adroid, en este caso Zebra MC330L

1.- Instalar Termux desde google play ( posiblemete tenga que darle permisos de todo )
2.- En la terminal correr :

pkg install mc // <= Master commander
pkg install make
pkg install git
git clone https://github.com/harbour/core.git harbour
cd harbour
make
make install

nano // <= aqui es un editor y solo escriba una prodedure main ; ? "Hola" ; return , CTRL X ; Y ; y poner el nombre del programa test.prg

3.- Una vez de nuevo el la terminal y estando el el folder de harbour

hbmk2 test // <= Si todo esta bien, entonces ...
./test // si todo va bien, presentara hola

4.- En la terminal

cd
git clone https://github.com/carles9000/ut.git
cd ut
./go_linux.sh

cd sample
cd basic
cd app
nano app.prg // <= Cambir el puerto a 8081, grabar y salir
cd ..
nano app.hbp // remover las ultimas instrucciones referetente a windows, grabar y salir
./go_linux

5.- Si todo va bien, ya puede abrir una pagina y poner

localhost:8081 // Ojo, si acaso puso el mismo puerto

6.- Venga, Espero que les sirva. Saludos
