Armando,

Una Api en resumen es la capacidad que tiene un programa de conectarse con otro mediante una biblioteca de funcionalidades. Lo primero que has de ir acotando es esta api en que sistema y entorno se usara, si para atacar tu sistema en windows, en internet,... en que protocolos se comunicará o interfasaran las peticiones, si tendras capas de seguridad,...

Puedes tener un programa de contabilidad en windows que tenga una api que permita que te conectes y pasarle/recibir datos. Esta api (subprograma) tendrá unas reglas de conexión, petición y un formato/s de datos pre-establecidos.
