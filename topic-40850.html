Sin FOR NEXT
Code (fw): Select all Collapse
adel(a, 5)
adel(a, 6)
adel(a, 7)
adel(a, 8)
aSize(a, Len(a)- 4)


o bien mi preferida sin FOR NEXT:
Code (fw): Select all Collapse
aDeleteVarios(a, {|x, n| n >= 5 .and. n <= 8})


Code (fw): Select all Collapse
FUNCTION aDeleteVarios(a, bCondicionBorrado)
Local nI
Local nBorrados
Local nFinal
Local nItem  // Nº item real
*
nFinal:= Len(a)
nBorrados:= 0
nI:= 1
nItem:= 1
DO WHILE nI <= nFinal
   IF Eval(bCondicionBorrado, a[nI], nItem)
      aDel(a, nI)
      nBorrados++
      nFinal--
   ELSE
      nI++
   ENDIF
   nItem++
ENDDO
aSize(a, Len(a)- nBorrados)
RETURN a

Code (fw): Select all Collapse
for n := 8 to 5 step -1
   ADel( aArray, n, .t. )
next

// OR

for n := 5 to 8
   ADel( aArray, 5, .t. )  // use 5 not 'n'
next
