El problema esta aqui:

D:\FW.lib

Clases.lib
Externas\FWcal\btnget.LIB


Tienes que definir la variable entre comillas:
Code (fw): Select all Collapse
CLASESEXTERNAS1 = "D:\FW - Clases externas"
