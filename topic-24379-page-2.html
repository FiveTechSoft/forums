Para configurar xDevStudio para harbour hice lo siguiente: En el menú Ferramentas -> Configurar compiladores.
Allí se encuentran los Scripts de compilación, yo elegí “Harbour 32 bit BCC/Mingw”, Pulsar “Adicionar”



Aparecerá un dialogo donde configuraremos los path de los compiladores a usar ( este es el que uso actualmente )



En "versión del compilador" es una descripción de las características del script, el resto supongo que es obvio. Se me ocurre que la inclusión de FWH aquí es innecesario, ya que el mismo lo declararás en el proyecto que use este script. De todas maneras lo dejé porque no molesta.
Después de haber configurado el script tendrás que generar el proyecto que lo use. Si configuraste para xHarbour no tendrás problema para configurarlo en harbour. Espero haber sido claro. Sigo el hilo por si tienes dudas
