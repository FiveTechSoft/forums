Willi,

En el archivo google.html hay que incluir la línea:
Code: Select all Collapse
map.setMapType(G_SATELLITE_MAP);

Code: Select all Collapse
    function initialize() {                                              
      if (GBrowserIsCompatible()) {                                      
        map = new GMap2(document.getElementById("map_canvas"));          
        map.setMapType(G_SATELLITE_MAP);
        map.addControl(new GLargeMapControl());                          
        map.addControl(new GScaleControl());                          
        map.addControl(new GMapTypeControl());     
        // bind a search control to the map, suppress result list
        map.addControl(new google.maps.LocalSearch(), new GControlPosition(G_ANCHOR_BOTTOM_RIGHT, new GSize(10,20)));
        geocoder = new GClientGeocoder();     
      }                                                                  
    }


Andrés,

Esta es una alternativa que se me ocurre, aunque me parece un poco chapuzera:

fijate que en el documento google.html, en él aparece la línea:
Code: Select all Collapse
Coords = marker.getPoint()

Esta línea graba en la variable "Coords" los datos de las coordenadas.
Ese valor se podría grabar en un documento de texto para poder leerlo desde tu aplicación.

Ya te digo que me parece un poco chapuzero y seguro que existen muchas y mejores formas de hacerlo, pero esta es la primera que se me ocurre.
