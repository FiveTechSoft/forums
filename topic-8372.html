Antonio funciono muy bien solo que agregando un uno (1)en la funcion play() por favor enseñame porque se comporta de esa manera, es decir porque hay que poner un uno, si pongo cero no funciona pero si pongo dos o tres etc si funciona adjunto el codigo de la probeta que emplee.

#include "FiveWin.ch"

function Main()

local oAvi, oWnd,oTm

DEFINE WINDOW oWnd FROM 3, 3 TO 20, 40 ;
TITLE "Probando avi con temporizador"
DEFINE TIMER oTm of oWnd action oAvi:Play(1) interval 1500

@ 3, 2 VIDEO oAvi FILE "C:\varios\video\Download.avi" SIZE 260, 55 PIXEL NOBORDER

ACTIVATE WINDOW oWnd ON INIT(oTm:activate())

return NIL
