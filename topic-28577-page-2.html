Has de usar el acceso directo "Simbolo de sistema de las herramientas nativas de VS2012 X86"
o el que corresponda a la version de Visual Studio que utilices VS2010, VS2012, VS2013

Pero debe estar en:

C:\Program Files (x86)\Microsoft Visual Studio 11.0\VC>dir *.bat


El fichero es vcvarsall.bat

Recuerda que el directorio depende de la version que tengas instalada.
El ejemplo que pongo es para Visual Studio 2012
Si fuese Visual Studio 2013 sería:

C:\Program Files (x86)\Microsoft Visual Studio 12.0\VC>

pero el fichero sigue siendo el mismo: vcvarsall.bat
