Antonio, yo lo uso asi, sin problemas, todo contenido, y solo lo necesario para fwh. (no mas )

VC 2019 32/64 bits (incuido todo)
http://161.0.60.84/owncloud/index.php/s/oHug5m0AVdjxoQT

con hacer un bat que apunte a los directorios correctos
por ejemplo mi caso que lo tengo en d:\compiler\vc2019
myvc32.bat

@echo off
set path=%path%;d:\compiler\vc2019\bin32
set lib=d:\compiler\vc2019\lib\win32
set include=d:\compiler\vc2019\include


myvc64.bat

@echo off
set path=%path%;d:\compiler\vc2019\bin64
set lib=d:\compiler\vc2019\lib\win64
set include=d:\compiler\vc2019\include


buildh32.bat (para fwh 32 bits)
http://161.0.60.84/owncloud/index.php/s/lN5GgPAsK5fy3ij

buildh64.bat (para fwh 64 bits)

Buenos días !

Un enlace de StackOverflow para obtener "Herramientas de compilación sin conexión para Visual Studio 2019 o 2022".

https://stackoverflow.com/questions/59868753/offline-build-tools-for-visual-studio-2019

Y en el siguiente enlace los componentes que podemos descargar https://docs.microsoft.com/en-us/visualstudio/install/workload-component-id-vs-build-tools?view=vs-2019

Podemos hacer la descarga de esta manera:

Code (fw): Select all Collapse
vs_buildtools.exe --add Microsoft.VisualStudio.Workload.MSBuildTools --layout c:\offlineBuildTool --lang en-us
