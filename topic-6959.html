Accesar a una DBF de Visual Fox Pro

Posted: Thu May 10, 2007 05:58 AM
Raymundo,

Puedes usar el ADORDD, sólo tienes que incluirle la cadena de conexión para FoxPro (ficheros con extensión .DBF)

Añade esta línea en ADORDD.prg:
Code: Select all Collapse
case Lower( Right( aOpenInfo[ UR_OI_NAME ], 4 ) ) == ".dbf"
       s_aConnections[ nWA ]:Open( "Provider=Microsoft.Jet.OLEDB.4.0;Data Source=" + aOpenInfo[ UR_OI_NAME ] + ";Extended Properties=dBASE IV;User ID=Admin;Password=;" )
regards, saludos
