Muchas Gracias a todos por las respuestas.....

Finalmente y gracias a Carlos.... la cadena de conexion es la siguiente:

Code (fw): Select all Collapse
TRY
   oCon := CreateObject( "ADODB.Connection" )
   oCon:ConnectionString := "DSN=Finca_Raiz_lan;Uid=postgres;Pwd=123456;"
   oCon:Open()
CATCH oError
   MsgAlert("Error en la conexión: " + oError:description )
END


Definitivamente mas simple no se puede....
