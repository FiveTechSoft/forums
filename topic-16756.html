A new &lt; VTitle-Painter &gt; Class VTitle FWH 9.08
Posted: Sat Sep 19, 2009 06:23 PM
Hello,

I started with a VTitle-Painter using the new VTitle-Class and the Result of first Tests are fantastic.

I can define any Style with any included Images on selected Backgrounds and Positions.
The needed Source is created as well, to make it easy, to include the designed VTitle in Your Application.
With just a Mouseclick, You can create new Combinations.
The Size of the VTitle can easy defined with a horizontal and vertical Slider.
All Settings can be saved to a DBF and will be restored with a new Start.
Because of many possible Settings, the Visual-Design will be a great Help for Users.

With all the possible Combinations, it is a hard work, to test it inside Your Application
for a good Result. Only for the VTitle-Body 64 Combinations are possible.
The Settings of the VTitle-Body :
Transparent, Base, Border, Shadow, Shadow-Position, Shadow-Size, Shadow-Itensity
all this in Combination with different Text and different Buttons.
The Buttons as well have different Settings.
The Work is nearly done. I think, the first Release for Testing, will be finished End of next Week.

A horizontal VTitle with Image



A Gradient Background and VTitle with defined Gradient



Many needed Parts can be used from other Tools.
That makes it easy, to create something new in a very short Time.



All possible VTitle-Settings :
Gradient
Shadow- Size, Position, Transparent-Level
Text-Font, Color, Brush, Base, Border,
Corner-Selection



Easy resizing with Sliders and define Position on Screen.



Define up to 3 Texlline on any VTitle-Position.
