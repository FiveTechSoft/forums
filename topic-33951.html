I don't know if this will work, but try this:

Code (fw): Select all Collapse
Function isWindowsLocked()
    Local lLocked:=.f.
    if getForegroundWindow() == 0  // Screen saver or login screen is foreground
      lLocked:=.t.
    endif
Return lLocked
