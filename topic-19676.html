Antonio has guided me to answer this post.

Vouch32 as library was last offered for (x)Harbour in 2007 and then
I never compiled with newer versions. Instead I provided Vouch32 Active-X Server
downloadable from http://www.vouch32.com which provides the same functionality
and can be used with any language. It is also free.

If you need it in pure lib version then you need to migrate to Harbour.
I did quiet a large changes and now I cannot compile it with xHarbour.
So when you are ready with Harbour, drop me a post, I will publish the library.
enjoy hbIDEing...
