See if this helps:

Printer Troubleshooter

Go to Settings

Select Update

Select Security Troubleshooter

Select Other Troubleshooters

Locate your printer in the Find and fix other problems section

Click on your printer and select Run the troubleshooter

Follow the on-screen instructions

Update your printer drivers

Check for a newer version of your printer driver

If not, go to the manufacturer's website and download and install the latest driver

Restart your computer and printer

Restart your computer

Turn off the printer, unplug all connected cables, and wait 20 seconds

Reconnect the cables and turn on the printer

Other procedures Restart the Print Spooler service, Reinstall the printer driver, Disable security software, Troubleshoot in a clean boot state, Roll back Windows Update.
