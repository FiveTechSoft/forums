Se me olvidaba, ya está disponible HDO para SQLite.
En breve comienzo HDO para MySQL/MariaDB.

Recuerden un mismo PRG que funciona con los principales gestores de bases de datos cambiado una sola línea:

Code (fw): Select all Collapse
oDb := THDO():new( "sqlite" )

Code (fw): Select all Collapse
oDb := THDO():new( "mysql" )

Code (fw): Select all Collapse
oDb := THDO():new( "odbc" )


El resto de código no cambia
