Os dejo aqui un enlace para poder bajar Runner XBase (RX), para quien quiera usarlo para hacer pruebas de código Harbour rápidamente. El sistema ha sido diseñado con el objetivo de probar, construir o prototipar un aplicación web.



Empieza de una manera cómoda a probar tus primeras páginas, formularios,... a encajar todas las piezas y darle una lógica a todo el proceso.



El sistema es freeware y pretende echar una ayuda a todos los que por una razón u otra siempre le ha costado dar este salto.



Cualquier duda o aclaración os la intento aclarar.

RX no es un servidor web como los demás. No podrás ejecutar directamente las paginas a no ser que crees una ruta para ellas. Mírate la documentación para entender como funciona, esta en español e ingles. Actualmente esta desactivado el acceso libre a las páginas en html por tema de seguridad y de concepto. Como puedes observar nadie puede ejecutar directamente el código html\xtreme a no ser que se ejecute como tu las definido en el módulo.



Es un concepto nuevo, vamos a ver como se adapta...
