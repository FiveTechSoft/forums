I use

Code (fw): Select all Collapse
hBmp:=ReadBitmap(0, CurDrive()+':\'+Curdir()+'\QRCODE.BMP') 
    nHig=nBmpHeight(hBmp) 
    nWid=nBmpWidth(hBmp) 
    oWord:AddImagen( 10, 20, 10+nHig, 20+nWid, CurDrive()+':\'+Curdir()+'\QRCODE.BMP',{1,1,1,1,1,1})
