Sorry - last question was unnecessary.
There exists package NewsSync, but I don't have any experience with it:
-------
NewsSync synchronizes messages between Usenet and phpBB.
All Usenet messages from a single newsgroup (e.g. alt.food.wine
or, alt.php, etc.) are synchronized with a single phpBB forum. One
way (from Usenet to phpBB) and two-way synchronizations are
supported. In two-way synchronization, phpBB users can
participate in a discussion across the Internet that extends outside
of phpBB.
NewsSync is supported on both Windows and Unix. However,
Windows has a few limitations: administrator access is required,
and it is strongly recommended that unless you are in full control of
the Windows box you should switch to a Unix system.
NewsSync also supports a few open source content management
system integrations that support phpBB. These include postnuke
and phpnuke. Others may work but have not been tested.
------
