You can add fourth element to the array elements:

aAdd( aRezult[i], str(aRezult[i,3],4)+str(aRezult[i,1],2) )

and the sort it by the fourth element

aOut:= ASORT(aRezult,,, { |x, y| x[4] < y[4] })
