Vcpkg es una herramienta de gestión de paquetes de C++ de código abierto desarrollada por Microsoft. Puedes obtener vcpkg de varias formas:



1. **GitHub**: Puedes clonar el repositorio de vcpkg desde GitHub utilizando el comando `git clone https://github.com/microsoft/vcpkg.git`. Luego, debes compilarlo manualmente siguiendo las instrucciones en el README.md.

2. **Releases de GitHub**: Puedes descargar la versión binaria más reciente de vcpkg desde la página de releases de GitHub. Busca la versión adecuada para tu sistema operativo (Windows, Linux o macOS) y descarga el archivo zip o tar.gz correspondiente.

3. **Instalación vía script**: Puedes ejecutar el script de instalación oficial de vcpkg para Windows o Linux/macOS. Para Windows, ejecuta `powershell -Command "iwr -useb https://raw.githubusercontent.com/microsoft/vcpkg/master/bootstrap.ps1"` en PowerShell. Para Linux/macOS, ejecuta `bash <(curl -s https://raw.githubusercontent.com/microsoft/vcpkg/master/bootstrap.sh)` en la terminal.

4. **Instalación vía paquete**: En algunos sistemas operativos, como Ubuntu o macOS (con Homebrew), puedes instalar vcpkg utilizando el administrador de paquetes del sistema. Por ejemplo, en Ubuntu, puedes ejecutar `sudo apt-get install vcpkg`. En macOS con Homebrew, puedes ejecutar `brew install vcpkg`.

5. **Visual Studio**: Si ya tienes instalado Visual Studio, puedes instalar vcpkg desde la tienda de extensiones de Visual Studio. Busca "vcpkg" en la tienda y sigue las instrucciones de instalación.



Una vez que hayas obtenido vcpkg, debes agregar la carpeta de vcpkg a tu variable de entorno `PATH` para poder utilizarla desde la línea de comandos.
