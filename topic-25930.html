You can add up to 3 textlines.
Selection <Deformed> will use the 3 lines to create a Logo



The needed Infos :

1. Text 1: My special Logo
1. Text 2: ---
1. Text 3: ---
1a. Format : JPG
1b. Format : PNG

2. Font : Arial

3. Background
-------------
a) Color : -
b) Gradient : -
c) Material : S 1

4. Text-brush
-------------
a) Color : -
b) Gradient : - -
c) Material : V 2

5. Deformed : B 3
6. Outline : B 1
7. 3D-border : B 12
8. Shadow : B 2
9. Frame : E 1 ( with selected Material )
