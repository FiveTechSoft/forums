Enrique,

FiveWeb usa HTML + CSS + JavaScript en la parte cliente, y PHP en la parte servidor.

Cuando una aplicación arranca, usa HTML, desde ahí se ejecutan funciones en javascript. Si revisas el código en JavaScript verás que hay mucha similitud en como se programa con FiveWin.

FiveWeb NO es un traductor de PRG a Web, ni usa PRGs. No sirve para portar una aplicación FiveWin a la Web. Es una forma nueva, muy potente y facil de crear aplicaciones para la Web.

El borrado de registros ya está funcionando

http://www.fivetechsoft.com/fivegen (envíame un email para obtener tu login y contraseña si quieres ser beta tester)

Este código es muy interesante ya que muestra como usamos "codeblocks" en JavaScript y también como simulamos ejecución "modal":

Code (fw): Select all Collapse
function Delete()
{
   var nSelected = BrwSelCount( oFrame );
     
   if( nSelected == 0 )
   {
      MsgInfo( "Please select one record to edit" );
      return;
   }
   else if( nSelected > 1 )
   {
      MsgInfo( "Please select only one record to edit" );
      return;
   }        

   OnYes = function() 
   {
      oDlg.End();
      oFrame.src = "http://www.fivetechsoft.com/fivegen/delete.php?table=" + Alias() + 
                   "&recno=" + BrwSelRecord( oFrame );
   };
      
   MsgYesNo( "Do you want to delete it ?", OnYes );             
}


OnYes es una variable pública que contiene un "codeblock". Será evaluado cuando "Yes" se seleccione desde MsgYesNo().

Aprovecha esta oportunidad de aprender estas técnicas avanzadas de programación para la web
