Function Localiza(cKey,oTbr,nKey)
Local nSeekrec:=0,xTbr
xTbr:=oTbr
nSeekec:=recno()
if GetASyncKey(VK_BACK) --> Esta es la funcion
cKey:=substr(cKey,1,Len(cKey)-1)
end if
seek upper(cKey) softseek
if !found()
go nSeekRec
end if
xTbr:Refresh()
return

El método ON CHANGE de tu get quedaría asi:

ON CHANGE (Self:Assign(),Localiza(""+substr(oGet1:cText,1,Self:nPos-1) + upper(chr(nKey)),oTbr,nKey))
