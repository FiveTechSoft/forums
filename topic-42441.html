hi,

i have this to change "Name of TAB"
Code (fw): Select all Collapse
   aPrompts := ACLONE( oTab:aPrompts )
   aPrompts[ nDim ] := aToken[ LEN( aToken ) - 1 ]  // Name of Folder
   oTab:SetPrompts( aPrompts )
