Jose Luis, los botones redondos son sólo gráficos (en este caso, con el mismo fondo que el fondo del dialogo, para no complicarme). Me implementé una clase que simplemente sustituye gráficos, uno para botón normal otro para cuando pasa el ratón por encima y otro para cuando lo pulsas, con algunas cosas más, claro, pero básicamente es eso. Lo mismo ocurre con las solapas de que se adivinan detrás del popup menu, son sólo gráficos con trasparencias y blendings, nada más. Un poco de imaginación, mucha paciencias y ya está.
