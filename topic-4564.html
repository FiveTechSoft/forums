Javi:

Debes indicar que el font definido lo use el objeto oReport

DEFINE FONT oFont NAME "Arial" SIZE 0, -26
DEFINE FONT oFnt NAME "Arial" SIZE 0, -10

REPORT oReporte .....;
FONT oFont,oFnt

Y despues

oReport:oHeader:aFont[1] := {||1}

Observa que el 1 esta definiendo el número de font que le has asignado al objeto oReport en función al orden en que los asignaste

oFont será el 1, oFnt será el 2 etc etc

Ojalá esto te ayude.
