Otto,

Try something like this:

Code (fw): Select all Collapse
#include "Fileio.ch"

function isFile( cFile )
   local lSuccess:=.t., nHandle:=0
   nHandle := FOPEN( cFile, FO_READWRITE + FO_SHARED)
   IF FERROR() != 0
      lSuccess:= .f.
      //? "Cannot open file, DOS error ", FERROR()
      BREAK
   ELSE
      FCLOSE()
   ENDIF
return lSuccess
