Code (fw): Select all Collapse
</s>if oBrw:nDataType == DATATYPE_RDD

else
///
endif<e>

Another way:
Code (fw): Select all Collapse
</s>if !Empty( oBrw:oRs )
   // ADO
elseif !Empty( oBrw:cAlias )
