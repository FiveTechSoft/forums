Amigos:

He logrado modificar el archivo XML y puedo salvar con diferente extensión, el problema ahora
es que para abrir un diseño solo se pueden abrir las que tienen extensión .fr3, no permite
abrir otra extensión.

Modifique en
Code (fw): Select all Collapse
  <StrRes Name="dsRepFilter" Text="Informe (*.fr3)|*.fr3|Informe 4 (*.fr4)|*.fr4"/>
  <StrRes Name="dsComprRepFilter" Text="Informe Comprimido (*.fr3)|*.fr3|Informe comprimido 4 (*.fr4)|*.fr4"/>
