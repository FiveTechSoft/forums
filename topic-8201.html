> 2. Please give me the detail and sameple of avantage feature for TPREVIEW class in FWH 7.09

You can have as many previews simultaneously as you may need. i.e.:
Code: Select all Collapse
PRINT oPrn1 PREVIEW
   PAGE
      oPrn1:Say( 2, 2, "Test" )
   ENDPAGE
ENDPRINT

MsgInfo( "one" )

PRINT oPrn2 PREVIEW
   PAGE
      oPrn2:Say( 2, 2, "Test" )
   ENDPAGE
ENDPRINT

MsgInfo( "two" )

PRINT oPrn3 PREVIEW
   PAGE
      oPrn3:Say( 2, 2, "Test" )
   ENDPAGE
ENDPRINT

MsgInfo( "three" )
