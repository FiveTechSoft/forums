Ari,

Tienes que usar el emulador de Pocket PC , no el de Windows CE.

FWPPC funciona en Windows CE tambien, lo que ocurre es que el emulador de Windows CE no es de arquitectura ARM, sino Intel.

En cambio, el emulador de Pocket PC si usa arquitectura ARM.

You have to use the Pocket PC emulator, not the Windows CE one.

FWPPC also works with Windows CE, but the Windows CE emulator does not use ARM architecture, but Intel one.

Instead of that, the Pocket PC emulator uses the ARM architecture
