Hakan,

Defining it from source you can write
Code: Select all Collapse
@x,y DTPICKER oDate VAR dDate ....
oDate:nStyle := nOr (DTS_SHOWNONE,oDate:nStyle)


The checkbox is unchecked, if dDate is empty.
