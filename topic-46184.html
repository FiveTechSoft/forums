HIX &amp; AntiGravity magical experience ✨✨
Posted: Mon Dec 08, 2025 03:11 AM

You are going to live a magical experience. Yes, as it sounds :)

Follow these steps to experience magic ✨✨

  1. Install HIX from here: https://github.com/carles9000/hix Place it where you prefeer, in example: c:\hix

  2. Install Google AntiGravity from here: https://antigravity.google/

  3. Copy the provided HIX + AntiGravity examples to the same HIX folder: https://forums.fivetechsupport.com/viewtopic.php?t=46183

  4. Run hix.exe and write start and press enter

  5. Run antigravity.exe

  6. From AntiGravity select "File", "Open folder" and choose "c:\hix"

  7. From HIX click on "Open browser" icon (top bar, right side, chrome icon)

  8. Navigate to localhost/calculator.prg

  9. Take some time to experience it

  10. Go to AntiGravity and press Ctrl + L, to start talking to Gemini

Vibe coding with AntiGravity 🧙🏼‍♂️🪄

  1. Tell the agent your intention:

I want to build a "Hello world" webapp using this API: https://github.com/carles9000/hix/wiki the server is already running at localhost

  1. The agent will create a Task:

    • Research Hix API and requirements
    • Create Implementation Plan
    • Initialize project structure
    • Implement Hello World using Hix API
    • Verify application

  • The agent will start working to complete the task and it will present you an implementation plan.

    • Simply answer:

    continue

    or

    go

    1. Once the agent completes the webapp it will present you a walkthrough.md report. Click open and review it

    2. Tell the agent to execute it:

    show me the browser output

    Once AntiGravity browse is shown for the first time you will be requested to accept the AntiGravity Chrome extension

    From that moment on, start talking with the agent to tell it what you want and how ⚡⚡⚡
