the syntax is wrong

menuitem want a say as

MENUITEM "Specificare la stampante"
file "MENU.bmp" ;
ACTION PrinterSetup() ;
MESSAGE "bla bla "

if you use a variable sample oItem

menuitem oItem Prompt " blabla" ;
file "MENU.bmp" ;
ACTION PrinterSetup() ;
MESSAGE "bla bla "
