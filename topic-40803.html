TSwitch class

Command:

xcommand @ <nRow>, <nCol> SWITCH [ <oSw> VAR ] <lVar> ;

         [ PROMPT &lt;cCaption&gt; ] ;
         [ &lt;of: OF, WINDOW, DIALOG&gt; &lt;oWnd&gt; ] ;
         [ SIZE &lt;nWidth&gt;, &lt;nHeight&gt; ] ;
         [ &lt;help:HELPID, HELP ID&gt; &lt;nHelpId&gt; ] ;
         [ FONT &lt;oFont&gt; ] ;
         [ &lt;change: ON CLICK, ON CHANGE&gt; &lt;uClick&gt; ] ;
         [ VALID   &lt;ValidFunc&gt; ] ;
         [ &lt;color: COLOR, COLORS&gt; &lt;nClrFore&gt; [,&lt;nClrBack&gt;] ] ;
         [ &lt;design: DESIGN&gt; ] ;
         [ &lt;pixel: PIXEL&gt; ] ;
         [ MESSAGE &lt;cMsg&gt; ] ;
         [ &lt;update: UPDATE&gt; ] ;
         [ WHEN &lt;WhenFunc&gt; ] ;
         [ BORDERSIZE &lt;nBorderSize&gt; ] [ BORDERCOLOR &lt;nClrBorder&gt; ] ;
         [ THUMBSIZE  &lt;nThumbSize&gt;  ] [ THUMBCOLOR  &lt;nClrThumb&gt; ] ;
         [ &lt;lRad: RADIOSTYLE&gt; ] [ &lt;lOval: OVALSTYLE&gt; ];

You can see this command in fivewin.ch
