Hola Ubiratan:
SGS wrote:Preciso disso porque gostaria de chamar com F5 uma função que capture a imagem do sistema, porem, não consegui acionar nenhuma tecla se o menu estiver ativo.
Cuando un menú está abierto, la API de Windows atrapa el teclado y solo permite el uso de las teclas que el menú debe procesar. Pero para lo que tú quieres hacer existe una tecla específica (Prnt Scrn) la cual sí funciona con el menú abierto y coloca en el portapapeles la imagen completa de la pantalla.
