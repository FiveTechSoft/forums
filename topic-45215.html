Dear Massimo,



Please do this:

git clone https://github.com/harbour/core harbour

git clone https://github.com/fivetechsoft/fivemac



then do:

cd harbour

make

cd ..

cd fivemac

make



cd samples

./build.sh fivedbu



that should build fivedbu with no issues and run properly on your macs
