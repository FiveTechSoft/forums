http://www.ammyy.com/en/

Things attracted me

1) It is a free remote control software
2) A replacement for expensive TeamViewer ( Non commercial use is free in TeamViewer)
3) Similiar to TeamViewer, ie No need for Real IP and IP forwarding. No installtion required. Just run the exe
4) Very small in size. Only 144KB
5) I Found Ammyy a bit slow compared to TeamViewer. May be due to the high Internet traffic on my network while I tested

It has 2 modes Public Router mode (Like TeamViewer) and a Private router mode. I tested both and working fine

As per the site

Ammyy Admin is a highly reliable and very friendly tool for remote computer access. You can provide remote assistance, remote administration or remote teaching for your clients, employees and friends regardless of their location. Ammyy Admin connects remote computers within seconds without any installation or configuration. You can view and/or control (using your keyboard and mouse) any applications running on the remote computers.

Dear Mr.Detlef Hoefner,

Could you please tell me which of these programs are needed ?

If you are goint to use a Public router (Supported by AMMYY )then you need only AMMYY_Admin.exe

AMMYY_Router.exe is to be used only if you are going to have your own Private router setup.

For the time being it is better to use the Public router (AMMYY_Admin.exe). And once you get familiar and need more performance you may use your Private Router (AMMYY_Router.exe )

How to set up them?


Assuming that you are trying to access your freinds desktop thru Ammyy

In your PC do the following

1) Run AMMYY_Admin.exe
2) Please take note of the ID shown in the Text Box. This is required
2) Goto the Menu AMMYY -> Settings
3) On the Ammyy settings dialog, On the Routers Group choose Public and press OK button. That's all at your end

At your freinds PC

1) Ask your friend to do the above given steps in his PC
2) Ask him to let you know the ID shown in his Ammyy Admin screen. (This can be automated later)
3) Ask him to click on the Start button available in the Tab Client

Now at your PC

1) Choose the Operator Tab
2) Type the Client ID and click the Connect button. That's all

Once you click Connect Button, you friends PC will ask whether to allow you, he has to approve it. (This part can be automated, so that in future it can be connected without any user intervention. The Ammyy can be run as a service also)

This can be used to support you clients without any software licencing cost

I have found that using a Private Router gives much more performance

I believe that by default the setting is to use Public Router, which means you don't have to do anything.
1) Ask your friend to Run AMMYY_Admin.Exe
2) Ask your friend to let you know his ID
3) Ask your friend to Click on the Start Button in the Client Tab. That's all
4) In your PC you run AMMYY_Admin.Exe
5) Type your Friend's ID and Click on the Connect Button on the Operator Tab

Hope it is clear to you.

Dear Mr.Harvey,

What's involved in setting up a private router? Could you elaborate?


By default Ammyy will be using thier router setup (Public Router, Similiar to Teamviewer) which is easy to setup. We can have our own Router setup, which is more efficient (according to me).

Private router is nothing different. You need to have a Static IP at the location where u r planning to run the Private router. In one of your servers/PC the Ammyy_router.exe will be running and all your clients will use Private router with your router's Static IP. You should also designate a port to be used by the router. On your router (which does the NAT) you should forward all incoming (Ammyy router port) traffic to the PC running the Ammyy_Router.Exe

I feel it is more secure and there is considerable performance difference.
