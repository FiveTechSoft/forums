Try adding this to ON INIT clause of the ACTIVATE DIALOG command:

Code (fw): Select all Collapse
oDlg:PostMsg( WM_KEYDOWN, VK_TAB )


Works fine here. I can see the square cursor when TABbing to a combobox.
