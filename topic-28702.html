Entendi ahora
Code (fw): Select all Collapse
SELECT tabla1.*,tabla2.desactivado  FROM tabla1 LEFT JOIN tabla2 
ON tabla1.id = tabla2.id
WHERE tabla2.desactivado = 0 OR tabla2.desactivado IS NULL
