Mr Rick

Please make this modification in XBrowse.prg.

Find these lines at the end of METHOD SetColFromADO
Code (fw): Select all Collapse
   if lAutoOrder
      oCol:cSortOrder   := oCol:cHeader
   endif

Modify this as :
Code (fw): Select all Collapse
   if lAutoOrder
      oCol:cSortOrder   := '[' + oField:Name + ']'
   endif
