Manuel,

Yo utilizo ahora N++ para FWH. Me instale un extensor para la sintaxis coloreada pero ni lo he parametrizado, pero funciona. Si te interesa ya te pasare la extension. La info para modificar o tunearte la puedes encontrar aqui - >http://udl20.weebly.com/index.html.

Notepad no es gestor de proyectos, pero puedes compilar y enlazar perfectamente desde él. Tiene una consola q puedes ejecutar algunos paràmetros propios y ejecutar luego procesos bat, es muy facil y limpio. De hecho te muestro como lo hago para crear un exe desde notepad

Code (fw): Select all Collapse
// Notepad++ Crear ejecutable para for Fivewin Harbour, made by Carles Aubia
cls

// Salvamos fichero prg
NPP_SAVE 

// cmd /c cd "$(CURRENT_DIRECTORY)"
cd $(CURRENT_DIRECTORY)

//Compilamos 
c:\Hal\zn++1602.bat $(NAME_PART)


Este script lo tienes grabado en el notepad y puedes tener tantos como quieras.

Bueno...es una pincelada. Si quieres algo mas lo pides.

[EDITADO]Lo uso porque me toca tambien codificar en php, javascript, html,... es el editor que usamos y se adapta perfectamente para mis necesidades en FWH
