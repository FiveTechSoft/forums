Hola que tal:

Lo que yo hago para lanzar una aplicación es crear un archivo .LNK o .BAT:

LNK
Code (fw): Select all Collapse
if file("cyewin.lnk")
  erase cyewin.lnk
endif
oShell  := TOleAuto():New( "WScript.Shell" )
o := oShell:CreateShortCut( 'cyewin.lnk')
o:WindowStyle      := 1
o:TargetPath       := CurDrive() + ':\' + CurDir() + '\cyewin\cyewin.exe'
o:Arguments        := ValEjer
o:WorkingDirectory := CurDrive() + ':\' + CurDir() + '\cyewin'
o:Description      := 'CyEWin, Contabilidad y Estimaciones para Windows'
o:IconLocation     := CurDrive() + ':\' + CurDir() + '\cyewin\cyewin.ico'
o:Save()
ShellExecute(0,"Open","cyewin.lnk",NIL,NIL,1)
erase cyewin.lnk


BAT
Code (fw): Select all Collapse
if file("impr.bat")
  erase impr.bat
endif
set console off
m := fcreate ("impr.bat",0)
fwrite (m,"cd impranua")
fwrite (m,chr(13))
fwrite (m,chr(10))
fwrite (m,"java -jar PlataformaModulosInformativas.jar /E:" + nombfich + " /V:S /R:" + errofich)
fwrite (m,chr(13))
fwrite (m,chr(10))
fwrite (m,"cd ..")
fwrite (m,chr(13))
fwrite (m,chr(10))
fclose (m)
waitrun("impr.bat",0)
erase impr.bat


Al final uso waitrun o Shellexecute dependiendo de si quiero que espere o no a que termine la ejecución. Y borro el archivo que he creado.

Un saludo y espero que te sirva.
