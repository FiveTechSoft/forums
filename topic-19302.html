Hola Noe,

Yo lo uso así (con btnget)

Code (fw): Select all Collapse
redefine btnget oFechaIn VAR dFechaIn ID 4002 OF oDlg RESOURCE "Calen" SPINNER;
          on up (++dFechaIn,oFechaIn:Refresh())  ;
          on down (--dFechaIn,oFechaIn:Refresh())  ;
          action (aFechaIn := GetCoors(oFechaIn:hWnd),;
          dFechaIn:=FwCalendar(dFechaIn,aFechaIn[1],aFechaIn[4],oDlg,;
          nRGB(200,200,200),nRGB(100,100,100),;
          nRGB(168,45,69),{nRGB(232,242,255),CLR_BLUE} ),;
          oFechaIn:Refresh() )
