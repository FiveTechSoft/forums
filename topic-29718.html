Encontré esto, Talvez alguien que lo traduzca a FWH.

Con la Api HideCaret puedes ocultar el cursor:

Private Declare Function HideCaret Lib "user32" (ByVal hwnd As Long) As Long

El siguiente ejemplo oculta el cursor de Text1 al hacer clic sobre el

Private Sub Text1_Click()
HideCaret Text1.hwnd
End Sub

Adaptalo a tus necesidades

Y si quieres volver a mostrar el cursor lo puedes hacer con la API ShowCaret

Private Declare Function ShowCaret Lib "user32" (ByVal hwnd As Long) As Long

ShowCaret Text1.hwnd
