Antonio, evidentemente, usar una infraestructura en red local no supone ningún problema desde que usamos Clipper, y ahora Harbour, teniendo en cuenta las consideraciones normales al respecto: apertura shared de las bases de datos, y los necesarios bloqueos a la hora de actualizarlas.
Otro tema es la infraestructura a nivel de hardware: contar con un cableado adecuado, switches / hubs de buena calidad, y un servidor de archivos apropiado: disco ssd, etc., y no hablo solo de que sea un ordenador "dedicado", sino que lo conveniente además es contar con un software adecuado: windows 7, bueno vale, pero lo lógico sería probar a tener un SO como un Server de Microsoft, no?.
Aunque por su sencillez de implementación, la opción del RDP también es para tenerla muy en cuenta, otro tema es si hay que contar con licencias de software para poder usar el RDP, etc.
