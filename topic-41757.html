Leandro:

O condiciona el salto de página solo cuando sea la impresión de una factura

Code (fw): Select all Collapse
IF lFactura
    IF oVrd:nNextRow > oVrd:nPageBreak
    ...
    ...
    ENDIF
ENDIF
