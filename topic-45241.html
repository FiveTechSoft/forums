Sage 200: Es uno de los ERP más utilizados en España. Ofrece módulos para inventarios, ventas y más. Además, cuenta con una API que permite la integración con e-commerce y otros sistemas.



Ekon: Este ERP es muy popular entre las PYMES en España. Ofrece una solución completa para la gestión de inventarios, ventas y otros procesos empresariales. También dispone de una API para integraciones externas.



Odoo: Aunque es una solución global, Odoo tiene una fuerte presencia en España y ofrece soporte en español. Es muy flexible y cuenta con una API robusta para integraciones con e-commerce y otros sistemas.



Velneo: Es una plataforma de desarrollo de aplicaciones empresariales que permite crear soluciones a medida, incluyendo ERP. Ofrece una API para integraciones y es muy utilizada en España
