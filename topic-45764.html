Estimados



Aquí pueden descargar

https://code.visualstudio.com/download



Elegir el .zip y descomprimir donde deseen



Para que se vayan ambientando a éste maravilloso editor y asi estar preparados para el Webinar que dictará Antonio



Lo pueden usar con BCC o MSVC + (x)Harbour



Deben instalar la siguientes extensiones

- Harbour and xHarbour de Antonio Perricone.

- Project manager de Alesandro Fragnani

- Task de Actboy168 para que aparezca un botón en la barra de tarea para compilar directamente.



Crear su Task asi

{

"version": "2.0.0",

"tasks": [

{

"label": "Compilar",

"type": "shell",

"command": "D:\\Sistemas\\Contab\\GoHbmk.bat",

"group": {

"kind": "build",

"isDefault": true

}

}

]

}


El GoHbmk.bat para BCC

if exist contab.exe del contab.exe

Set path=D:\Sistemas\Compiler\BCC77\BIN

D:\Sistemas\Compiler\HarbourBC\bin\hbmk2 prg\proyecto.hbp -comp=bcc64 -run


El proyecto.hbp

-gui

-w0

-inc

-ID:\Sistemas\Compiler\FWH\include;D:\Sistemas\Compiler\BCC77\include



Contab.PRG

Contab11.PRG

Contab.RES

Contab.RC

D:\SISTEMAS\SOURCES\PROCESOS.PRG



-workdir=..\OBJ

-optim-

-ldflag+=-aa

-LD:\Sistemas\Compiler\FWH\LIB;D:\Sistemas\Compiler\BCC77\lib

-lFive64

-lFiveC64



xhb.hbc

hbziparc.hbc


Con ésto es suficiente para compilar sus sistemas con BCC



Para MSVC deben instalar Visual Studio 2022 que también se lo puede hacer portable



Espero les ayude y se decidan a cambiar. Vamos que se puede.
