Forse mi spiego meglio utilizzando un esempio di Fivewin.
Vorrei colorare di verde la colonna rappresentante il campo "User Id".
Se non è possibile usare ListBox, potrei avere un esempio di come fare altrimenti?
Grazie

DEFINE DIALOG oDlg FROM 3, 3 TO 26, 79 TITLE cTitle

@ 0, 1 SAY cListName OF oDlg

@ 1, 1 LISTBOX oLbx FIELDS USER->Name, USER->User, USER->Title ;
HEADERS "User Name", "User Id", "User Title" ;
SIZE 284, 137 OF oDlg

@ 13, 8 BUTTON btnModify PROMPT "&Modify" OF oDlg SIZE 40, 12
@ 13, 15 BUTTON btnDelete PROMPT "&Delete" OF oDlg SIZE 40, 12
@ 13, 22 BUTTON btnSearch PROMPT "&Search" OF oDlg SIZE 40, 12
@ 13, 29 BUTTON btnEnd PROMPT "&Exit" OF oDlg SIZE 40, 12

ACTIVATE DIALOG oDlg

RETURN( NIL )

Ok, quello è un TWBrowse. Lo riconosci dalla clausola FIELDS. Per colorare la seconda colonna puoi usare qualcosa del genere:

Code (fw): Select all Collapse
oLbx:nClrPane = { | nCol | If( nCol = 2, CLR_HGREEN, ) }
