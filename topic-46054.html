HIX - Harbour web server

Posted: Fri Oct 24, 2025 12:31 PM


Ep !

Os presento HIX un web server preparado para ejecutar desde el minuto 1 todas las páginas web que desees. De la misma manera que hay un ejecutable Apache que ejerce de server, HIX también es un ejecutable.

Lleva toda la librería de UT y está pensado para los que quieren solo programar web con los elementos tradicionales. La base para el backend es harbour, por lo que mucha cosa ya lo tendréis solucionado. Todo ha surgido de un par de desarrolladores que no querian compilar UT, solo querian tener el servidor como herramienta de apoyo para sus aplicaciones.

Ejecutable Harbour
Backend Harbour

En cuestión de minutos los que tengan un poco de nociones ya podéis de manera muy fácil poner cualquier formulario para que vuestros clientes puedan acceder desde cualquier teléfono, tablet, ordenador…

El sistema funciona desde cónsola, la configuración es muy sencilla y podréis crear aplicaciones web robustas y seguras muy fácilmente.Quien desee podrá crear rutas, ssl, firewall, gestión de accesos, status de error, configurar lo esencial de harbour,...

La idea: Un click y tenéis un servidor en marcha.

El pre-release ya está listo y lo podéis bajar desde github. https://github.com/carles9000/hix/releases/tag/v1.00

Toda la info y ayuda la podreis encontrar en la wiki https://github.com/carles9000/hix/wiki

Aquí en el blog tenéis una explicación de todo para las dudas que han ido saliendo.
https://httpd2.blogspot.com/2025/10/hix.html

Harbour es mas que autosuficiente para este escenario web. No necesitamos ni python, ni php, ni otros lenguajes, ya tenemos el nuestro.Creo que a más de uno le puede ayudar.
