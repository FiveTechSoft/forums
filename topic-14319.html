Dynamic News / Noticias dinámicas
Posted: Sun Jan 18, 2009 11:21 PM

Recently I had the chance to test a RSS reader software with an interesting feature that seemed very interesting to me: the "autoplay" feature, this means that the application by itself goes from news to news.

I thought that it was very interesting idea for offices, public places, etc. so I thought that a web version could be very usefull, without having to setup the software. etc so this weekend I played a little with javascript and php and here you have the result

http://www.fivetechsoft.com/news

Right now it is just a working prototype but the idea is that every user could set up his RSS of choice, look, etc. Better watch news instead of looking a "nice" screensaver but without any practical utility

Recientemente tuve la ocasión de usar un software para lectura de noticias RSS pero con una característica que me llamo mucho la atención: la capacidad de hacer "autoplay", es decir, que el programa solo va cambiando de noticia en noticia.

La cuestión es que me pareció muy interesante para oficinas, lugares públicos, etc. y pensé que seria práctico que estuviese accesible desde la web sin instalación, etc. y este fin de semana he hecho algunas pruebas con javascript y php y aqui teneis el resultado

www.fivetechsoft.com/news

De momento es un prototipo funcional pero la idea es que cada usuario pueda configurar sus fuentes de noticias (RSS), apariencia, etc. Mejor ver noticias de actualidad que mirar un salvapantallas "bonito" pero que no "informa"
