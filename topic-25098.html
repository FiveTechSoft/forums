Con MySql, lo hago así:
Code (fw): Select all Collapse
 oTbTemp:End()    //destruyo el objeto anterior

 oBrw:SetMySql(oTbTemp,.f.)    //lo cargo al browse

 oTbTemp:Refresh()
 oBrw:Refresh()    //refresco el browse
