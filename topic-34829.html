As I advised you in private mail, it's one of the many Harbour bugs. You have to replace this

Code (fw): Select all Collapse
_oWord:Set("Visible",.f.)


with this

Code (fw): Select all Collapse
_oWord:Visible = .f.


and this

Code (fw): Select all Collapse
_oFind:Set("MatchWildcards",.t.)


with this

Code (fw): Select all Collapse
_oFind:MatchWildcards = .t.
