¿Hola buenos días como vas?

Para el envió de los mensajes de texto, nosotros lo hacemos mediante un proveedor de este tipo de servicios, ellos nos entregan una url y nosotros simplemente la consumimos.

Ejemplo:

http://forums.fivetechsupport.com/viewtopic.php?f=6&t=37322&p=222984&hilit=envio+sms&sid=6fd436d711e561a3245ccebc5d594f8a&sid=6fd436d711e561a3245ccebc5d594f8a#p222950

El tema del whatsapp lo único que puedes hacer es utilizar la API de ellos, hacer el llamado a la url, por medio de navegador a whatsapp web ó de la aplicación (si la tienes instalada), de ahí para allá ellos se encargan del resto.

Code (fw): Select all Collapse
cUrl := "https://wa.me/573tunumero/?text=Hola,%20necesito%20ayuda"

@ 187,90 URLLINK oUrlLink PIXEL OF oDlg TOOLTIP "Hablar por Whatsapp" URL "Por Aquí" 
oUrlLink:bAction := {|| ShellExecute( oDlg:hWnd, 'Open', cUrl ) }


Para el tema del correo electrónico, en este foro hay mucha información, basta con que busques "enviar mail"
