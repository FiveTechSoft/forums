Tim,

The following sample works fine here using latest xHarbour from SVN and latest FWH:

Code (fw): Select all Collapse
#include "Fivewin.ch"


FUNCTION MAIN()

    LOCAL oHttp

    LOCAL cRet

    oHttp = TIpClientHttp():New( "http://www.emagsoftware.it/download.htm" )

    oHttp:Open()

    cRet = oHttp:ReadAll()

    oHttp:Close()

    ? LEN( cRet )

    MEMOWRIT( "MYRESULT.TXT", cRet )

    RETURN NIL
