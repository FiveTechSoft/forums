si has descomprimido e instalado FW, y las carpetas donde has ubicado
FW... en... c:\fwh
BCC7... en... c:\BCC7
harbour... en... c:\harbour

así es como lo tengo yo.

vas a la carpeta fwh\Samples e intenta compilar un prg con buildh.bat nombreprg(sin la extension .prg)
Yo tuve que copiar el fichero builh.bat a buildh (1).bat por guardarlo y "retocar" el fichero buildh.bat para que funcionase.

líneas 12 y 13 por...
rem if "%FWDIR%" == "" set FWDIR=...
rem if "%HBDIR%" == "" set HBDIR=c:\harbour
set FWDIR=...
set HBDIR=c:\harbour\BCC

abre una consola con CMD y trabaja desde MSDOS.

Una vez ya compiles el primer prg, puedes probar con el resto de prgs... y atreverte a utilizar el Editor para PRGs y proyectos FW, FiveEdit.
