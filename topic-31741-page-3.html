Marc,

If you use uppercase here:

PRGS = \
TOOLBAR2.PRG

Then you have to use upper case here too:

TOOLBAR2.exe : $(PRGS:.PRG=.OBJ) TOOLBAR2.res

and here:

$(PRGS:.PRG=.OBJ) : $(PRGS:.PRG=.C)
$(PRGS:.PRG=.C) : $(PRGS)
