Al mover el diálogo por encima del browse, haces que este se repinte y al repintarse cambia el registro seleccionado en la base de datos.

Puedes hacer oBrowse:cAlias = "" para que no se repinte el browse, ó bien anotar el registro en donde estás y antes de guardar los datos, volver a posicionarte en donde estabas.

FWH es FiveWin para Harbour, y es un producto distinto al FW 16 bits que tienes. Es un producto comercial que puede comprar desde www.fivetechsoft.com sección pedidos.

Aunque es un producto distinto, es compatible con FW 16 bits. Está desarrollado en 32 bits y usa el API de Windows de 32 bits.
