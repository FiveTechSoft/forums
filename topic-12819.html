Algún ejemplo en el uso de HB_ZIPFILE

Posted: Mon Sep 22, 2008 01:56 PM

Hola,, aqui va:

Para Zipear:
aFilZip // nombre del archivo zip que contendra los files a zipear,
aFiles // array que contiene el nombre de los files a zipear
cPass // contraseña para que el zip este seguro,,,

HB_ZipFile( cFilZip, aFiles, 9,,.T., cPass, .F., .F. )

si lo quieres con un control meter
HB_ZipFile( cFilZip, aFiles, 9, {|| oMeter2:Set(nRow++)} ,.T.,
cPass, .F., .F. )

Para desZipear

cFile // ruta y nombre del archi zip a descomprimir
cPass // la contraseña de archivo zip
cFTem // carpeta en la cual de descomprimiran los files
aFile // array que contiene los nombres de los files a descomprimir

aFile := hb_GetFilesInZip( cFile )

!HB_UNZIPFILE( cFile, {|| oMeter1:Set(nRow++), SYSREFRESH()}, .t., cPass, cFTem, aFile)

Salu2
