No, no, no.
El secreto sólo está en montar la URL a enviar, luego la envías a internet explorer por ejemplo así:

Code (fw): Select all Collapse
// **********************************************************************
FUNCTION ConsultaWeb( cUrl )

/* Lanza una WWW desde Internet Explorer para obtener su resultado; no abre ninguna ventana.
   -------------------------------------------------------------------------------------- */

Local oExplorer := Nil
Local cResult   := ""

oExplorer := TOleAuto():New("InternetExplorer.Application")
oExplorer:Navigate( cUrl )
oExplorer:Visible:=.F. // IE invisible

// Las siguientes 4 lineas son para pausar la carga de una pagina, omitelas si no te son necesarias
//WaitSeconds(5) // deshabilito, no hace falta
WHILE oExplorer:Busy
    HB_IDLESLEEP(1)
ENDDO

// Guardamos en texto del resultado de la web
cResult := oExplorer:Document:Body:InnerText

// Estos métodos dan error.
//oExplorer:End()
//oExplorer:Destroy()

oExplorer := Nil

//IF "OK"$cResult
//ALERT("El envio del SMS fue un exito")
//ENDIF

Return cResult
// **********************************************************************
(Por cierto, esta función está sacada del foro)

A partir de ahí te puedes complicar la gestión hasta donde quieras. (que si controlo si ha ido bien, que si quiero que se envíe más tarde, que lo guardo en BB.DD, etc)
