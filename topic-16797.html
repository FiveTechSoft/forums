Pablo,

Respecto al software:

  • browses tipo iphone, usables con el dedo, con kinetic, soporte de caches para acelerar su funcionamiento, etc.
  • Transparencias, alpha channel, nuevos controles "virtuales"
  • ActiveX funcionando con IE (solo en Windows CE de momento)

No se ha usado ningún control propio de Windows. Todo el interface, completamente, esta hecho con el nuevo FWPPC, muy al estilo iphone

Ahora estamos trabajando para organizar todo el software hecho, y poder ofreceros una nueva versión de FWPPC cuanto antes.

En realidad hemos construido con FWPPC una nueva capa por encima del sistema operativo, con la que el usuario final no detecta que está usando Windows CE.

Un trabajazo, eso ademas de la portabilidad del VOIP, PTT, Video en streaming, etc

La razón de que se haya usado Windows CE es muy simple: la licencia cuesta 3 dólares, frente a 30 dólares que cuesta la de Windows Mobile
