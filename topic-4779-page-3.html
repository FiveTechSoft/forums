Osvaldo,

> Will there two way to desings form's ?
> I mean, via VFP and Interface Builder ?

Yes, we want that FoxPro users may be able to reuse their existing Windows forms with FiveMac.

Our plan is to provide a form designer built-in with FiveMac, that will allow to modify existing forms and create new ones.

Interface Builder creates NIB files (XML format) that may be also used from FiveMac.

>
If VFP form, could be loaded from prg ... All the object propierties will accesed from prg level ?
>

Yes, a true object is created using and providing the same properties.
