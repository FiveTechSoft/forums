I tried first with tablets running Windows RT, and in these was only possible via RDP, now I bought tablets running Windows 8.1 32 bits, Like Dell venue pro 8, Toshiba encore, Lenovo Miix2 and Lenovo ideapad 8 and how the system is Windows 8, our systems functioning normally. without changes in touch screen.
These new tablets with Windows 8.1 are with reduced price, around $ 200US, since Microsoft released Windows license free for systems with less than 9-inch screen, as well as released Windows phone 8 license for free, so must be released more products with lower prices in the coming months, and our applications work normally in desktop mode without changes.
Windows phone 8.1 Developer preview has a vpn and enterprise mode, but not tested yet.
but to run in store, like a "modern style" application, needs a lot of changes in (x)harbour and FWH, as has already been discussed in other threads.

Windows server 2012 R2 has a remote appl mode, with this you can run your aplication in tablets with android, ipad, and (of course) Windows, without changes.
