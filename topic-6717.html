Hola,
para dejar columnas fijas a la izquierda.
Code: Select all Collapse
oBrw:nFreeze             := 2 //Nº de columnas

Para no permitir swap entre columnas
Code: Select all Collapse
oBrw:lAllowColSwapping   :=.F.


Para no permitir que se redimensionen las columnas. (Para cada columna que no quieres opcion de redimension.)
Code: Select all Collapse
oCol:lAllowSizing:=.F.
