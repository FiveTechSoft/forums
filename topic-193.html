Hola:

Yo importe todos mis recursos de WS hacia Pelles y No tuve ningún problema (exceptuando cosillas de recursos propios)

Lo que si hice fue grabar el recurso en formato RC y eso lo abrí con Pelles.

Comprueba en los dialogos que:
no tengas ninguna llamada a Borland.
si usas btnbmp, como estan definidos los botones. En pelles para usar btnbmp tendrá que estar así (bueno cambiando tamaños)
Code: Select all Collapse
CONTROL "", 202, "TBtnBmp", 0x00000000, 21, 0, 15, 15

si usas tsbrowse, controla como esta definido,
Code: Select all Collapse
CONTROL "", 162, "TsBrowse", WS_TABSTOP|0x00800000, 0, 18, 524, 160

comprueba las líneas estáticas que usas para separar controles,
Code: Select all Collapse
CONTROL "", 801, "Static", SS_ETCHEDFRAME, 2, 16, 528, 1
