Con TSButtons podes usar el metod ChangeBitMap
ChangeBitMap( anBmps, acBmps, lResource )
Por ejemplo
oBot:ChangeBitMap( 1, "NUEVO",.T. )
Donde cambias la imagen principal del boton con el BITMAP de recurso de nombre "NUEVO"
O si tenes definido mas de una imagen para el boton (Cuando se presiona, cuando esta disable, etc) le pasas 2 arrays, uno con los botones de que posicion cambias, y otro con los nuevos recursos
