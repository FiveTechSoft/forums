Pete,

Please modify the Method ColAtPos() in Class TXBrowse this way:
Code (fw): Select all Collapse
   METHOD ColAtPos( nPos ) INLINE ::aCols[ ::aDisplay[ MinMax( If( nPos == nil .or. nPos == 0, 1, nPos ), 1, Len( ::aDisplay ) ) ] ]
