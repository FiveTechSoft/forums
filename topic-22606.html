Hola a todos:

Algunos compañeros me han mandado e-mails sobre como funciona el tema de las webcam USB, para los que no tenéis muy claro como funciona y no usáis las últimas versiones de FWH, aquí os dejo un programa completo con el código fuente incluido.

Este programa yo lo uso para poder ver los nidos de los loros en la época de cría con unas 10 webcam distintas. Para no modificar el programa inicial que hice, simplemente ejecuto 10 veces el programa y en cada uno veo una webcam distinta.

Creada con la versión FWH812, aunque la he probado hasta la versión FWH1109 sin problemas.

1.- Podemos crear imágenes capturadas de la webcam en disco.
2.- Podemos grabar videos sin problemas con "NO RESPONDE".
3.- Modificar formato de video.
4.- Copia imagen webcam al Portapapeles.
5.- Podemos crear Streem para ver las capturas en Internet. ( solo es una pequeña base, en la carpeta HTML/Ejemplo teneis un fichero HTML para ver los stream de la cam, lo podeis mejorar esto solo es una base ).



Descargar:
http://www.grupes.es/basura/webcam.rar

Para aquellos que estabais interesados ver la Webcam vía explorardor ( HTML), no necesariamente tenéis que usar por ejemplo este programa, simplemente ejecutar webcam.html de la carpeta "html\Ejemplo WebCam OnLine Web-modo local", y vuestra webcam se vera vía Web.

También tenéis otra opción que es el programa Yawcam, ejecutáis el programa, le indicas que quieres usar Streem, luego ejecutas WebCam.HTML de la carpeta HTML\Ejemplo Webcam OnLine Web con YawCam y veréis vuestra webcam en internet.

Hola Luis;

Correcto, probando ahora con un portatil da ese problema. Yo hasta ahora lo tengo montado con webcam USB funcionado perfecto, pero con webcam integradas efectivamente tenia problemas al inicializar el controlador.

Ya esta solucionado al menos con las pruebas que acabo de hacer con 2 tipos de webcam, uno integrada y la otra USB, ahora ya inicializa correctamente.

Simplemente en el boton "ACTIVAR"añadir oWebCam:Initialize(), no se sabe porque hay que inicializar dos veces el controlador, el codigo fuente del boton "ACTIVAR" tiene que ser:

Code (fw): Select all Collapse
     DEFINE BUTTON RESOURCE "MenuP_Media","MenuP_Media","MenuP_Media";
            NOBORDER OF oBar;
            ACTION ( oWebCam:Initialize(),;
                     oWebCam:Initialize(),;
                     If( oWebCam:isConnected,;
                       ( oWnd:oMsgBar:cMsgDef := "WebCam: Conectada",;
                         oWnd:oMsgBar:Refresh()),));
            TOOLTIP "Conectar la WebCam";
            FONT oFont;
            PROMPT "Conectar";
            WHEN ! oWebCam:isConnected .AND. !oWebCam:lTimer
