Rick:

Did you try:

Code (fw): Select all Collapse
oLbxB:aCols[5]:bOnPreEdit := { | uVar | Set( _SET_INSERT, ! Set( _SET_INSERT ) )}
and
oLbxB:aCols[5]:bOnPostEdit := { | uVar | Set( _SET_INSERT, ! Set( _SET_INSERT ) )}
