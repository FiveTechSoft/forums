Features : ( here the readme.txt)

you can create tabs

for each tabs you can : edit the caption , add a group

for each group you can : edit the caption on the footer , defines the group by lines for insert groupbuttons or defines the group by column to insert normal buttons
add an action on group, delete a group ( this feature not run well), change the size of the group, add a separator

for each button you can add an image , change the type, change the layout, change the border,
change size and position ,delete the button, add a menu, edit or delete the caption

if you defines the group on lines you can add button header's group, button middle, button end of group
