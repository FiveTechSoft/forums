All applications that you build with Fivewin + Harbor 32 bits will work on 32-bit and 64-bit OS. Applications built with Fivewin + Harbor 64-bit will only work on 64-bit OS. The products for 32 and 64 bits are different and require separate licenses.
If you don't need a 64-bit environment, I recommend you after so long without updating the 32-bit version for Windows.

FiveDroid and FiveLinux are at your disposal here:

https://github.com/FiveTechSoft/fivelinux

https://github.com/FiveTechSoft/FiveDroid
