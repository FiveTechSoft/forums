Marco,

You can use this function:

Code (fw): Select all Collapse
#pragma BEGINDUMP

#include <hbapi.h>

HB_FUNC( CONOUT )
{
   printf( "%s\n", ( char * ) hb_parc( 1 ) );
}   

#pragma ENDDUMP


and this command to use the ?:

#xcommand ? <x> => ConOut( <x> )
