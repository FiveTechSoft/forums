if you want to create a C Struct
Code (fw): Select all Collapse
typedef struct _TRIVERTEX { ;
  long x;
  long y;
  unsigned short red;
  unsigned short green;
  unsigned short blue;
  unsigned short alpha;
} TRIVERTEX;

If you want to construct a Harbour Class
Code (fw): Select all Collapse
class Trivertex
   data x 
   data y
   data Red
   data Greem
   data Blue
   data Alpha

   method new()
endclass
