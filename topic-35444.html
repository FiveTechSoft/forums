Silvio,

If you wanted you could try ODBC using php and sure you could connect to the different extensions, of which there is also dbase -> http://php.net/manual/es/refs.database.php

TWeb hasn't relation to the type of database engine you want to use. This depends on how you have configured your server and the different libraries that you can use with php to connect with them.

Maybe I started a controversy but I did not waste time with dbf on a website. I think it's stupid, but it's just my humble opinion. I think that in a web environment you can easily get more performance using the power of DB like oracle, ms server and of course mysql.

TWeb solo es una libreria que te ayude a dar el salto a la web y ayudarte a introducir poco a poco con los diferentes conceptos, lenguajes, situaciones, ... de una manera fácil. He puesto a disposición de todo el mundo una libreria con todos sus fuentes y de manera gratuita un trabajo, creeme... de muchos años.
