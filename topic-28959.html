Muy optimizado makefile para Microsoft
Posted: Wed Jul 30, 2014 02:29 PM
Hace ya algún tiempo aprendí como usar "listas" de nombres de ficheros y a hacer reemplazos en esas listas (lo he usado en makefiles de Borland). Asi que intenté el mismo concepto en un makefile de Microsoft y finalmente he sido capaz de implementar el mismo concepto. Lo bonito de este makefile (MyApp.mak) es que solo precisa una lista de nombres de ficheros, y eso es todo (previamente teniamos que escribir los mismos nombres de ficheros en distintas partes del makefile)

Este código $(PRGS:.prg=.obj) significa: En la lista de PRGS reemplaza .prg con .obj. Como podeis ver se usa esta técnica en varias partes del makefile.

Disfrutadlo!

go.bat
Code (fw): Select all Collapse
@set oldpath=%path%
@set oldinclude=%include%
@set oldlib=%lib%
@set oldlibpath=%libpath%
if exist "%ProgramFiles%\Microsoft Visual Studio 12.0\VC\vcvarsall.bat" call "%ProgramFiles%\Microsoft Visual Studio 12.0\VC\vcvarsall.bat"
if exist "%ProgramFiles(x86)%\Microsoft Visual Studio 12.0\VC\vcvarsall.bat" call "%ProgramFiles(x86)%\Microsoft Visual Studio 12.0\VC\vcvarsall.bat"
c:\"Program Files (x86)\Microsoft Visual Studio 12.0"\VC\bin\nmake -fMyApp.mak
@set path=%oldpath%
@set include=%oldinclude%
@set lib=%oldlib%
@set libpath=%oldlibpath%
@set oldpath=""
@set oldinclude=""
@set oldlib=
@set oldlibpath=
if errorlevel==0 MyApp.exe


MyApp.mak
Code (fw): Select all Collapse
#Microsoft VS2013 make sample, (c) FiveTech Software 2014

HBDIR=c:\harbour
FWDIR=c:\fwh
VCDIR="c:\Program Files (x86)\Microsoft Visual Studio 12.0\VC"
SDKDIR="c:\Program Files (x86)\Windows Kits\8.1"

PRGS =       \
one.prg  \
two.prg     \
three.prg \
four.prg 
#place here as many PRGs as needed

.SUFFIXES: .prg .c .obj .rc .res

MyApp.exe : $(PRGS:.prg=.obj) MyApp.res
   echo $(PRGS:.prg=.obj) > msvc.tmp

$(PRGS:.prg=.obj) : $(PRGS:.prg=.c)
$(PRGS:.prg=.c) : $(PRGS)

MyApp.res : MyApp.rc
   rc.exe -r -d__FLAT__ MyApp.rc 
   
.prg.c:
   $(HBDIR)\bin\harbour $< /n /i$(FWDIR)\include;$(HBDIR)\include

.c.obj:
   cl.exe -c -TC -W3 -I$(HBDIR)\include -I$(SDKDIR)\include -I$(VCDIR)\include $<
