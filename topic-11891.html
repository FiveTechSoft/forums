Es posible cambiar el color del Prompt GroupBox, dejando el fondo transparente?


Code: Select all Collapse
  DEFINE DIALOG oDlg RESOURCE 1013 BRUSH oBrush
         oDlg:lTransparent:=.T.
    
        REDEFINE GROUP oGrp PROMPT "SENHA" ID 302  OF oDlg ;
             TRANSPARENT COLOR CLR_BLACK

Hola Ale:

Prueba con TSLines (es parte de SButtonx.lib)
Code: Select all Collapse
   REDEFINE GRAY BOX ID 302 OF oDlg LABEL "Label" ALIGN DT_RIGHT ;
            COLORS CLR_WHITE, CLR_GRAY, CLR_RED
CLR_RED es el color del prompt

En recursos:
Code: Select all Collapse
 CONTROL "", 302, "STATIC", SS_WHITERECT | WS_CHILD | WS_VISIBLE, 151, 142, 46, 46
