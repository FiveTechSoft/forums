para 32 bits
"%VCINSTALLDIR%"bin\link @msvc.tmp /nologo /subsystem:windows,5.01 /force:multiple /NODEFAULTLIB:msvcrt /machine:X86 /nxcompat:NO /Ignore:4006

para 64 bits

importante para compatibilidad
subsystem:windows,5.01 y subsystem:windows,5.02

Antes de construir harbour, es necesario comentar la definicion de una constante en el archivo makefile, en la carpeta de la lib png .\SRC\3RD\PNG , esto para que no haga falta la definicion de esa funcion de png.
Code (fw): Select all Collapse
...
   ifneq ($(HB_HAS_PNG_LOCAL),)

      ifneq ($(HB_HAS_ZLIB),)

         HB_CFLAGS += $(foreach d,$(HB_HAS_ZLIB),-I$(d))
         HB_CFLAGS_DYN := -DPNG_BUILD_DLL

         #comment here HB_CFLAGS += -DPNG_NO_STDIO
...
