busmatic wrote:
Todas mis librerias que he creado debo de compilarlas para que sean compatibles con Harbour? de ser asi Como.


Se compilan igual que si estuvieras en Clipper, compilas con Harbour, pero necesitaras una "segunda pasada" con el Borland C++, para generar los OBJ, el mismo borland incluye una herramienta llamada TLIB que sirve para armar las librerias a partir de los OBJ creados por el C++

busmatic wrote:
Porque las ambiguedades al definir variables ?


Porque no haz definido las variables, es decir, te las haz sacado de la manga sin haberlas definido como PRIVATE LOCAL STATIC ó PUBLIC, es decir haz hecho cosas como :

Code: Select all Collapse
X:= 5


sin hacer antes un

Code: Select all Collapse
LOCAL x  //(o PUBLIC o PRIVATE o lo que se necesita)
x:= 5
