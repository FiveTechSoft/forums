Hola Mario. Mira si te funciona esto:

oLbx:nClrText := RGB(0,0,0) // text
oLbx:nClrPane := RGB(235,225,45) // background 

 //Apuntador
 oLbx:nClrBackFocus := {|| GetSysColor( COLOR_ACTIVECAPTION ) }  //fondo
 oLbx:nClrForeFocus := {|| CLR_WHITE }   //texto

o asi:
/*
Cabeceras:
oBjeto:nClrBackHead := RGB(...) // fondo
oBjeto:nClrForeHead := RGB(...) // texto
Apuntador:
oBjeto:nClrBackFocus := RGB(...) // fondo
oBjeto:nClrForeFocus := RGB(...) // texto
Renglones:
oBjeto:nClrText := RGB(...) // text
oBjeto:nClrPane := RGB(...) // background

Además, dentro de los renglones, el color puede ser asignado por medio de un bloque de código y desplegar los renglones de distintos colores. Ejemplo:
oBjeto:nClrText:={|| IF dbf->campo > 0, RGB(),RGB()}
*/
