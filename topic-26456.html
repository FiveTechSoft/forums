Hola josferdi
Has probado a poner en el recurso que tenga barra de scroll vertical?
Si tu ves tu recurso con un editor de texto, (formato .rc del recurso) veras que en el diálogo del folder tienes algo así
Code (fw): Select all Collapse
SUB1 DIALOG DISCARDABLE 18, 18, 218, 80
STYLE WS_CHILD|DS_3DLOOK      
..........

Pues añade la barra de scroll
Code (fw): Select all Collapse
SUB1 DIALOG DISCARDABLE 18, 18, 218, 80
STYLE WS_CHILD|DS_3DLOOK |WS_VSCROLL            
..........
