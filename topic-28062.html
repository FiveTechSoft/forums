Enrrique,

Primero debes tener un certificado digital, luego buscas la libreria xmlsec y la utilidad xmlsec y ya tienes como hacer la firma si tu archivo esta ok,

Con esta linea de comando puedes firmar un xml : xmlsec1 –sign –id-attr:ID <Id a firmar> –output ArchivoFirmado.xml –pkcs12 <firma digital> –pwd <password de firma digital> ArchivoAFirmar.xml

Tambien puedes firmar usando ole con las lib del office, pero no lo he probado ( En el sdk esta todo es para la version 5 del xml de microsoft ).

Tambien puedes consultar este enlace : http://www.edreams.cl/factura-electroni ... pymes-sii/
