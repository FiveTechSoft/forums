Hola Marca:

Yo en programas en FW 16bits lo tengo codificado en la funcion EnMemoria, no la he probado en FWH, pero no tiene porque ir mal. Si no funcionara bien, mira si hay que pasar algun parametro a la funcion GetTask().

NOTA: La funcion cFileNoPath(), es una funcion que quita el path del fichero, tu puedes probarlo sin estan funcion pasando directamente el nombre del fichero.

Function EnMemoria( nFile )

LOCAL aTasks := GetTasks(),;
aLen := 0 ,;
aResult := .F.

FOR aLen := 1 TO Len( aTasks )

   If cFileNoPath( aTasks[aLen] ) == Upper( nFile )

      aResult := .T.
      Exit

   End

NEXT

Return( aResult )
