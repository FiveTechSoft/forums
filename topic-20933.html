En el tema "Guardar la ejecuacion de comandos DOS" doy una solución usando la SHELL con:
Code (fw): Select all Collapse
CreateObject("WScript.Shell")

http://forums.fivetechsupport.com/viewtopic.php?p=108482#p108482

Sería algo asi:
Code (fw): Select all Collapse
oRemoteShell:=CreateObject("WScript.Shell")
oRemoteShell:Run("cmd /C TASKLIST > TASKLIST.TXT",0,.F.)
SysWait(1) // espera 1 segundo
cTexto:=MemoRead("TASKLIST.TXT")
.../...
oRemoteShell:=nil
