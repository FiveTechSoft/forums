Puedes almancenar tus recursos (bitmap, iconos y diálogos) en un archivo .dll desde el programa workshop. Con Fivewin se incluye una DLL vacía llamada screens.dll que puedes copiar con otro nombre (MIDLL.DLL por ejemplo) para añadirle recursos y usarla desde tu aplicación.
La forma de usarla es muy sencilla:

Al iniciar tu aplicación:
SET RESOURCES TO "MIDLL.DLL" // Activa el recurso
..........
Para usar un DIALOGO
DEFINE DIALOG oDlg RESOURCE "MIDIALOGO" TITLE cTitulo FONT oFont1 ICON "ICONO1"
..........
Para usar un BITMAP
REDEFINE BTNBMP oBitmap RESOURCE "BITMAP1","BITMAP2" ID 102 OF oDlg;
ACTION oDlg:End() NOBORDER
.............
Para usar un CURSOR
DEFINE CURSOR oCursorMano RESOURCE "Hand"
......................
......................

Al salir de la aplicación
SET RESOURCES TO //Desactiva el recurso

Y eso es todo.
