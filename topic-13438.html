Si, posiblemente. Para poder usarlos, haz esto:
Code: Select all Collapse
   local o := CreateObject( "Shell.Explorer" )
   local hOldObj := o:hObj
   local uRet 

   o:hObj = aParams[2]
   // a partir de aqui, lo puedes usar como un objeto OleAuto 
   ... 
   o:hObj = hOldObj 
   o = nil
