Un enorme mercado está esperando – Harbour + FiveWin pueden conquistarlo
Posted: Sun Sep 07, 2025 09:47 AM

Hola amigos,



Nos encontramos ante una oportunidad única. El servidor estándar para aplicaciones profesionales en DBF es costoso, difícil de licenciar y, sin embargo, incontables sistemas en todo el mundo todavía dependen de DBF. Estas empresas necesitan urgentemente un sucesor, y Harbour junto con FiveWin están en una posición perfecta para ofrecerlo.



¿Qué falta? Una sólida interfaz de administración, una capa flexible de microservicios y una integración moderna de servicios web. Harbour ofrece exactamente eso: open source, liviano y con la capacidad de integrar IA y nuevas tecnologías sin romper los sistemas heredados.



Y ahora hay un verdadero cambio de juego: con ChatGPT-5, Harbour y FiveWin reciben un apoyo extraordinario. A través de #pragma BEGINDUMP, las funciones en C se pueden incrustar fácilmente, y con la ayuda de la IA, cualquier programador experimentado en Clipper puede ampliar aplicaciones de una manera que ningún otro sucesor de Clipper permite.



SQL no es una alternativa real para muchas empresas: la conversión de bases de datos es costosa, arriesgada y disruptiva. Con Harbour podemos preservar las estructuras DBF existentes, modernizar paso a paso y llevar las aplicaciones de forma segura y eficiente a la era web.



Esta es una oportunidad abierta. Un mercado enorme está esperando, y dado que nuestra comunidad abraza el desarrollo asistido por IA (a diferencia de muchos otros foros de derivados de Clipper), tenemos la oportunidad de tomar la delantera y establecer el estándar para la próxima década.



Un cordial saludo,

Otto

Hello friends,



Many companies still run on old DBF applications.

The problem: the old standard server for DBF is expensive and hard to license.

https://forums.fivetechsupport.com/viewtopic.php?p=276265&hilit=ads+license#p276265





Yet, these companies want to keep using their systems – and at the same time move step by step into the modern web world.



The idea: FiveWin with the new FW/Harbour Database Server – the new standard that we, as a community, want to develop together.



free & open source

fast and stable

expandable with microservices



ready for web and AI



With ChatGPT-5 it gets even more exciting: using #pragma BEGINDUMP, we can easily integrate our own C functions. This means every experienced Clipper programmer can extend applications in ways no other successor of Clipper allows – without having to rebuild everything from scratch.



A huge opportunity – a new standard is about to be created, and we can set it together!
