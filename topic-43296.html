Silvio,

Change
Code (fw): Select all Collapse
local nWidth := oPanel:nRight-50
with
Code (fw): Select all Collapse
local nWidth := iif(Len(oPanel:aControls)==0, oPanel:nRight-50, oPanel:aControls[1]:nRight)
