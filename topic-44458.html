I solved all the problems with uuid and hbpcre the problem was just the bad configuration of bcc7's cfg files



Let me start by saying that I have a folder in c:\ called "Work" and in this I have all the other folders as you can see in this image



https://i.postimg.cc/R0pWVqKy/work.png" alt="" loading="lazy">





you must set all *. cfg files



Ilink32.cfg


Code (fw): Select all Collapse
</s>-L"c:\Work\bcc7\lib";"c:\Work\bcc7\lib\psdk"<e>




bcc32.cfg


Code (fw): Select all Collapse
</s>-I"c:\Work\bcc7\include";"c:\Work\bcc7\include\windows\crtl";"c:\Work\bcc7\include\dinkumware";"c:\Work\bcc7\include\windows\crtl";"c:\Work\bcc7\include\windows\sdk";"c:\Work\bcc7\include\windows\sdk\atl"
-L"c:\Work\bcc7\lib";"c:\Work\bcc7\lib\psdk"<e>
