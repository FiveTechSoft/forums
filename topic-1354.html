James

The latest xHarbour has changed the way you set up ADS ..

REQUEST ADS

rddRegister ( "ADS", 1 ) // no underscore _ infront of ADS
rddsetdefault( "ADS" )
SET SERVER REMOTE
SET FILETYPE to CDX

Notice the underscore is gone .. just use as above.
