      function Pagina2( oWnd )
      
         local n
         
         for n = 2 to 41
            oWnd:aControls[ n ]:SetText( AllTrim( Str( n + 39 ) ) )
         next
         
      return nil

From Pagina2() you may use your DBF to find the right text, and also change DATA bAction of buttons.
