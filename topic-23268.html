Code (fw): Select all Collapse
   DEFINE RIBBONBAR soRbn PROMPTS "Opciones","Configuraciones" OF soWnd ;
          HEIGHT 155 TOPMARGIN 30
   // Pestaña Opciones
   ...
   // Pestaña Configuraciones
   ADD GROUP aO[_GrpOpc2] RIBBON soRbn TO OPTION 2 PROMPT "Configuraciones" WIDTH GetclientRect( soWnd:hWnd )[4]-15

   @ 22, 10 ADD BUTTON aO[_BRecep] PROMPT "Agregar Usuario" BITMAP "bmp_DefUser" MOSTLEFT ;
            GROUP aO[_GrpOpc2] SIZE 90,60 ROUND ;
            ACTION fDefUsuarios()

   ADD SEPARATOR TO GROUP aO[_GrpOpc2] COLUMN 1    // no anda, probé poniendo 2 tambien

Hola

COLUMN usa la posicion en pixel dentro del grupo para ubucar el separador

ejemplo si el boton ubicado en 10, 10 mide de ancho 60 px su borde derecho estara en el pixel 70 (del grupo) podras asignarle el valor de la columna a 75 (COLUMN 75)
