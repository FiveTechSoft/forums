16 bits applications under "true 32" Operating Systems (Win NT / 2000 / 2003/ XP / Vista) need a 16 bits emulator, this is the job of the WOW (as Rick Stated: WOW = Windows On Windows) so your 16 bits applications are running under an emulator, not under the control of the operating system.

WOWEXEC and NTDVM (NT DOS Virtual Machine) are needed to run 16 bits applications and in some cases these two programs causes problems.

The real and only solution to get the rid of these problems is to move to FiveWin Harbour 32 bits.
