Es muy simple de usar:

1) Lo instalas en tu equipo servidor y le asignas un puerto (usualmente el 80)

2) Te vas a una terminal dentro de tu red, abres el navegador de internet y en la seccion de direcciones le pones la ip del equipo donde instalaste el WinASP. Te debe constestar con una pagina Web de bienvenida, eso quiere decir que estas conectado al servidor

3) A partir de este punto todo es ASP puro y duro. Para las paginas web se utiliza la misma metodologia que ASP, para empezar tienes que programar tu mismo tus paginas, pero estas tienen que tener en vez de la extension .htm o html la extensión .asp.

4) Dentro la pagina ASP tu vas a poder escribir tu codigo (x)Harbour dentro de la etiqueta <% ... %>, WinASP interpretara eso que tu escribiste y lo enviara como salida al navegador cliente.

5) para llamar a una pagina ASP simplemente desde el navegador haces:

http://xxx.xxx.xxx.xxx/pagina.asp

Y listo.

Te recomiendo que para mas informacion mires algun manual de ASP para que entiendas la metodologia de trabajo de Active Server Pages, la programación no tiene complicación, es codigo (x)Harbour.
