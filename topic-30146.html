Rick Lipkin
Code (fw): Select all Collapse
PrnSetCopies(nCopies)

IF cVIEW = "V"
   PRINTER oPRINT FROM USER                ;
   PREVIEW MODAL                           ;
   NAME "Print Routine for INVOICE "+cRepairNumber

   IF EMPTY( oPRINT:hDC )

ELSE

      MsgStop ( "Printer not Ready !" )

   ENDIF

ENDIF
