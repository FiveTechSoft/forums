Saludos

Si es correcto, BUTTONBMP hereda de BUTTON esta a su vez crea el boton que nos brinda el API de window

La idea es BUTTONBMP es poder agregarle la imagen al boton que creamos con el API de window (con/sin temas activos en la aplicacion) que este no permite agregarle imagenes por default, simplemente se captura el mensaje WM_PAINT para poder hacer eso.
