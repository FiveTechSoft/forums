Re: Como obtener Texto de varias lineas en controles Button???

Posted: Sat Sep 05, 2020 04:12 AM
Use the clause MULTILINE while creating the button.

Sample code:
Code (fw): Select all Collapse
   @ 10,10 BUTTON "This is a long long text in prompt" SIZE 40,60 PIXEL OF oDlg MULTILINE




If you are using RC file for dialogs and buttons, add BS_MULTILINE style to the button in the rc file.
