Jose Luis
No sé si es mas de los mismo, pero en una celda de edición lo hago así: (y funciona correctamente)
Code (fw): Select all Collapse
   WITH OBJECT hS1:oCol3:= ::oBrwIns:AddCol()
      :cHeader      := "Cantidad"
      :oHeaderFont  := ::hFont["SysFontB"]
      :bClrHeader   := {|| { CLR_MUSGODARK, GetSysColor( COLOR_BTNFACE ) } }
      :nHeadStralign:= AL_RIGHT
      :nWidth       := 70
      :bClrStd      := {|| { CLR_BLACK, CLR_MUSGO } }
      :nEditType    := EDIT_GET

      :bToolTip     := {|| ::ShowTooltip() }
      ...
   END
...

// ------------------------------------------------------------------------  \\
//        ToolTip en celda
PROCEDURE ShowToolTip()   CLASS TMantenimiento

   hS1:oCol3:cTooltip:= if( !::oQryIns:FieldGet( F_Stk2Cant ) == 0, "tu mensaje...", "" )

// Fin
