Hola Horacio:

Existe una forma de hacer referencia a las columnas en xBrowse y es la siguiente:

si tienes una columna cuyo cHeader es CODIGO, puede hacer los siguiente:

Code (fw): Select all Collapse
oBrw:Codigo:cOrdBag := "ID"

es decir, puedes obtener el objeto TXBrwColumn asociado a un HEADER.

Horacio, prueba asi:

FOR N:=1 TO LEN(oBrw:aCols)
oBrw:aCols[n]:bLClickHeader := {|r,c,f,o| msginfo(o:cHeader) }
NEXT
