No se el conocimiento web que tienes, pero quien entienda de web facilmente puede valorar lo que tiene con un par de pruebas de concepto.
Si tienes un alto conocimiento y necesitas alta performance de tu aplicacion 2.1 es la bestia, con ratios parecidos a php

Mod V1 que creó Antonio fue el primer mod, el que abrió el caminó a los demás, pero como todos los programas, las primeras versiones son las que mas resienten.

Si quieres ser un poco mas libre y no ligarte a lo que ofrece HIX al ser un producto empaquetado, UT te permite enlazar todo lo que necesitas en harbour.

Sobre CGI hay mucha lectura y quien entienda sabrá sus pros y contras. El argumento que mas venden usuarios de Harbour por todos los lados es que el código está en el exe. Con UT al enlazarte tu propio server puedes poner el código que estimas dentro del propio exe al igual que una aplicación windows. En HIX puedes cargar modulos hrb en el arranque.

Sobre aplicaciones con miles de usuarios... cualquiera te lo podría soportar. Si cada segundo 1 usuario te pide 1 petición son 3600 usuarios en 1 hora, que ya son miles, pero quias la pregunta seria sobre la concurrencia que necesitarías para tu sistema, en este punto dependen de muchos factores, como el tipo de maquina, memoria, tipo de backend que procesaras,...

Sobre el rendimiento, he hecho varias métricas durante este tiempo con todos los sistemas que hemos hecho, quizás este te sirva para valorar también

https://carles9000.github.io/index_doc_es.html?search=m%C3%A9trica
https://httpd2.blogspot.com/2022/12/rendimiento.html

Si no tienes conocimientos, HIX es la mejor opción, no necesitas nada, todo esta integrado, gestión bases de datos mysql, sqlite, postgre,.. creación de rutas, cargador de módulos compilados hrb, acceso carpetas, firewall integrado,... Con poco pones en marcha una entrada de pedidos desde un smartphone,... Te diria que en un dias la podrias tener ! pero como siempre todo depende del nivel web que uno tenga y como poderla programar, que esto da para otro debate... :D

Diria para acabar que quien quiera montar su web con Harbour ya no tiene excusas porque hemos creado en los últimos años herramientas para poderlo hacer perfectamente para todo tipo de perfiles.
