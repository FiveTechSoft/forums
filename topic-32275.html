buenas

la verdad tienes que experimentar y probar las opciones que trae, con pocas lineas de código puedes lograr mucho con esta clase, revisa tgraph.prg te darás cuenta de todo lo que puede hacer, te muestro algunas cosas que he logrado:







Code (fw): Select all Collapse
          REDEFINE GRAPH oGraph ID 21 OF xDlg TYPE 3

                  oGraph:cTitY              := "PERIODOS"
                  oGraph:cTitX              := "VALORES"
                  oGraph:cPicture           := "@E 999,999"
                  oGraph:lxGrid             := .T.
                  oGraph:lyVal              := .T.
                  oGraph:lDegrade           := .T.
                  oGraph:lLegends           := .T.
                  oGraph:lBordLeg           := .F.
                  oGraph:lxVal              := .T.
                  oGraph:lBarFull           := .F.
                  oGraph:nPoint             :=  4
                  oGraph:lDefault           := .F.
                  oGraph:nPenWidth          := 1.7
                  oGraph:lDegrade           := .F.


hasta se puede imprimir el gráfico



Code (fw): Select all Collapse
aPos:=oPrn:Cmtr2Pix(1.0,12.5)
aSiz:=oPrn:Cmtr2Pix(9.0, 5.0)

REPORT oRpt TITLE  " "," "," "," "," ","REGISTRO DE MICROBIOLOG......
    HEADER oGraph:Print(oPrn,aPos[1],aPos[2],aSiz[1],aSiz[2]), 
    FOOTER ALLTRIM(oApp:cSigEmp)+" "+oApp:cRut+" Página Nro : "
    FONT  Fon1, Fon2, Fon3


te animo a que pruebes y busques
