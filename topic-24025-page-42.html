------------------------------------- EDITO ---------------------------------

Para asignar el primer PRG, has de utilizar la herramienta de ordenación de PRGs
1.- Selecciona el Item "PRG" en el Tree de Proyecto
2.- Pulsa sobre el botón de ordenación y mueve el PRG principal a la primera posición
Después graba el proyecto

Lo haces así?

To assign the first PRG, you must use the PRG sort tool
1.- Select the "PRG" Item in the Project Tree
2.- Press the sort button and move the main PRG to the first position
Then save the project
