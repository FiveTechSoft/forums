Simple

Dejo algo de informacion español

Una biblioteca de código abierto de JavaScript para mapas interactivos aptos para dispositivos móviles

Leaflet, desarrollado por Vladimir Agafonkin y un equipo de colaboradores dedicados, es una biblioteca liviana pero robusta para mapas interactivos y está diseñada teniendo en cuenta la simplicidad, el rendimiento y la facilidad de uso. Está dirigido específicamente a ser móvil. Tiene todas las características que la mayoría de los desarrolladores necesitan para mapas en línea. El folleto se puede ampliar con una gran cantidad de complementos.

Leaflet comenzó su vida en 2010 como Web Maps API, una biblioteca de JavaScript para el proveedor de mapas CloudMade. Fue construido desde cero, pero usó partes del antiguo código API.

El folleto funciona bien en todas las principales plataformas móviles y de escritorio, y aprovecha las ventajas de JavaScript, HTML5 y CSS3 en los navegadores modernos, a la vez que se puede acceder a ellos en navegadores más antiguos. Permite a los desarrolladores sin un fondo GIS visualizar fácilmente los mapas web en mosaico alojados en un servidor público.

El folleto es compatible con capas de mosaico, marcadores, ventanas emergentes, polígonos y más, así como con las interacciones del usuario, como el zoom de la rueda de desplazamiento del mouse en el escritorio y el zoom multitáctil para iOS, Android 4.0 y posterior, y Windows Phone 8. cargue datos de características de archivos GeoJSON, aplíquelos y cree capas interactivas. Esto incluye marcadores con ventanas emergentes cuando se hace clic. Admite capas de servicio de mapas web, capas GeoJSON, capas vectoriales y capas de mosaico.

El código del folleto puede crear polígonos e interactividad (incluidas las pantallas táctiles móviles y los clics del mouse) en un mapa. No tiene sus propios datos de mapeo y se necesita una capa de mapa base. Si bien un tutorial de proyecto sugiere CloudMade, ya que inicialmente creó un Folleto de código abierto, también funciona con otras plataformas de mapeo. Estos incluyen MapBox y OpenStreetMap.

El folleto es comparable a la API patentada de código cerrado de Google Maps y la API de Bing Maps.
