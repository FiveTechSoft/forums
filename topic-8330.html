Saludos!

Para quien no conozca este programa, es un potentisimo editor gratuito (es un proyecto Open Source), que no tiene nada que envidiar a otros editores. Hasta ahora el único inconveniente que tenía, es que no poseía 'Sintaxis coloreada' para lenguajes xBase. Poco a poco, he construido una sintaxis para este editor que englobe todas las palabras reservadas, funciones, comandos, operadores, comentarios, etc.

Os adjunto la pagina Web de este proyecto, y el archivo de sintaxis creado por mi, por si os apetece probarlo y/o colaborar con el fichero del lenguaje. Os lo recomiendo.

http://www.mytempdir.com/2029204
http://notepad-plus.sourceforge.net/es/site.htm.

Para añadir el lenguaje xBase al editor, solo hay que copiar el archivo UserDefineLang.xml a la ruta:

%APPDATA%\Notepad++\

La intención de este post, no es hacer 'spam' ni mucho menos, (pues no participo en este proyecto OpenSource) sino que despues de haber descubierto un programa tan bueno, me veo en la 'obligacion' de compartirlo con la comunidad. Espero que disfruteis de el, pues os puede ayudar mucho en el desarrollo de vuestras aplicaciones.
