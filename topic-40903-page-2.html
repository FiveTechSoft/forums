What is harbourino?
https://winhotel.space/harbourinopublic ... rbour.html


Is it similar to TWeb and Beacon?
HARBORINO is a program for patching files with a built-in preprocessor.
You build your framework yourself.
One advantage is that the HTML and JS line numbers are retained. If you examine the website with F12, you will see the real line numbers.
