FWH 9.12 provides support for Ronded Buttons using Class TBtnBmp:
Code (fw): Select all Collapse
    @ 5, 10 BTNBMP oBtn OF oWnd  ;
       SIZE 60, 70 ;
       PROMPT cVer FILE "..\BITMAPS\sms.bmp" 2007 ;                
       FONT oFont BOTTOM ;
       ACTION MsgInfo("Hello") 

    oBtn:lEllipse = .T.
