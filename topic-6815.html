FiveTech Software y ViaOpen http://www.viaopen.com se complacen en anunciar la disponibilidad de una versión inicial de ADORDD para Harbour y xHarbour. Este RDD va a ser incluido en el CVS de Harbour por lo que damos la bienvenida a todos los usuarios de ADO para comenzar a probarlo y cooperar en su desarrollo.

Actualmente este código ya funciona:

REQUEST ADORDD

FUNCTION Main()

USE test.mdb VIA "ADORDD"

BROWSE()

USE

RETURN nil

Ya ha sido probado con Access y MySQL, y debe funcionar con cualquier motor de bases de datos que soporte ADO.
