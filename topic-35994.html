Hola Silvio,

FWeb es una libreria php que te ayuda a crear aplicaciones para internet. Está orientado en los programas de gestión e inspirada en la manera de programar aplicaciones en entorno windows. Es una concepción diferente a la hora de diseñar pantallas (usamos un editor de recursos), estructurar el código y montar nuestras aplicaciones.


Para hacer aplicaciones de gestión en la web necesitas tocar muchos lenguajes (html, php, jasvascript, css,jquery,…) y FWeb te ayuda a minimizar el impacto sin tener que usarlos todos y entender como funcionan. Tengo dos usuarios que en 3 meses sin haber tocado nunca nada de la web, ya tienen sus primeras aplicaciones.


The first you need configure you host to use ioncube. (See manual about instalation)

When you can do it, you need to put in a folder the framework

You can try lot of samples that i provide with fweb. (You need go step by step, see manual, I explain a lot of examples)

Fweb don't use dbf. You need change the chip. Mysql, oracle, ms server,... For me mysql is the best solution for internet aplications.

FWeb is php, no a exe like fiveweb
