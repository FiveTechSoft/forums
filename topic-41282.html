Dear Marc,

With mod_harbour CGI version we do it this way:
Code (fw): Select all Collapse
<Directory C:/xampp/htdocs>
  RewriteEngine on
  RewriteRule ^([a-zA-Z]*)$ /cgi-bin/modharbour.exe?$1 [NC,QSA]
</Directory>
