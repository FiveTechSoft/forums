Refresh xBrowse

Para un agregado alcanza con esto:
Code: Select all Collapse
IF (base)->(DBAPPEND())
   REPLACE ......
ENDIF
oBrw:SetFocus()
oBrw:UpStable()
oBrw:Refresh()

Y para una eliminacion con esto
Code: Select all Collapse
IF (base)->(RLOCK())
   (base)->(DBDELETE())
   (base)->(DBUNLOCK())
   (base)->(DBSKIP(1))
   oBrw:SetFocus()
   IF (base)->(EOF())
      oBrw:GoUp()
   ENDIF
ENDIF
oBrw:UpStable()
oBrw:Refresh()

Espero te sirva
