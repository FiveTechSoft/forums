I think that you don't need to convert the php code to FiveWin, instead you make a call to the php from fivewin and capture its output. The output seems to be JSON. You can further process the JSON output from Fivewin. For this to work, you need to know the URL that is to be called.

For eg. something like

oHttp:=CreateObject("Microsoft.XMLHTTP")
cURL:="http://YourURLtoAccessThisParticularReport.php"

Its just an idea.

EDIT:- It looks like that the APIkey and your username is fixed in the php code. Ask them whether you can pass this as parameters to the php. OR whether each client gets a different php code so that you can fix the APIKey and username in your php code itself.
