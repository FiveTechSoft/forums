James,

This seems to work.

Code (fw): Select all Collapse
FUNCTION HB_SERIALIZE( aArray)
RETURN CMIMEENC( ASAVE( aArray ) )

FUNCTION HB_DESERIALIZE( xArray )
RETURN AREAD( CMIMEDEC(  xArray ) )
