I found this to work:

odlg:bResized := { ||ResizeImage( odlg, oImage ) }

...

Static function ResizeImage( odlg, oImage )
oImage:Move( 0,0, odlg:nWidth(), odlg:nHeight() )
oImage:refresh()
return nil
