me parece que no es necesario cambiarlo a string, con que uses el picture @Z, lo valores que son cero los muestra en blanco.
mira las col debe y haber.

Code (fw): Select all Collapse
   ADD TO oBrwLM DATA MOVP->NUM     TITLE "Nº"       SIZE 050 CENTER PICTURE "@L 999"
   ADD TO oBrwLM DATA MOVP->FECHA   TITLE "Fecha"    SIZE 096 CENTER
   ADD TO oBrwLM DATA MOVP->DEBE    TITLE "Debe C$"  SIZE 096 PICTURE "@Z 999,999.99" COLOR CLR_RED
   ADD TO oBrwLM DATA MOVP->HABER   TITLE "Haber C$" SIZE 096 PICTURE "@Z 999,999.99" COLOR CLR_BLUE
   ADD TO oBrwLM DATA MOVP->SALDO   TITLE "Saldo C$" SIZE 096 PICTURE "999,999.99"
