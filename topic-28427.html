es necesario que en ciclos que tiene cierta duración poner cada x tiempo o x registros procesados una llamada a sysRefresh()
esto por que durante el ciclo while no se procesan los eventos de windows, provocando la congelación del programa, por ello cada tiempo es necesario llamar al procesado de eventos de windows.
Code (fw): Select all Collapse
do while !eof()
  ...
  if mod(recno(),50)=0
    sysrefresh()
  endif
  skip
enddo
