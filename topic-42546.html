Yo trabajo con base de datos

mis recomendaciones.



1. crea la base de datos para las imagenes desde código
Code (fw): Select all Collapse
</s>METHOD CreateTable(oCon) CLASS CreaTablas
   LOCAL cQry
   cQry := "CREATE TABLE IF NOT EXISTS images_data ( "
   cQry += "id_rec      INT UNSIGNED NOT NULL AUTO_INCREMENT,"
   cQry += "code_paciente    varchar(10) default ' ', "
   cQry += "tipo_img    varchar(10) default ' ', "
   cQry += "img_file    MEDIUMTEXT NULL , "
   cQry += "PRIMARY KEY (id_rec), INDEX (code_img)  ) "
   cQry += "ENGINE=InnoDB DEFAULT CHARSET=latin1"
   oCon:Execute( cQry )
RETURN (Nil)
 <e>


nota: Nunca usar imagenes en una base de datos con muchos campos entre mas pequeña es mejor



2. crea otra base de los datos con los datos del paciente (code_paciente)
