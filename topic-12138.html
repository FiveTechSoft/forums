yo trabajo con eagle1 de manu. Lo que hago es trabajar con array y cuando tengo que insertar / modificar lo hago con una sentencia SQL.

Para Hacer un insert:
Code: Select all Collapse
 cQuery := "insert into mytabla( documento, fecha, cliente ) values( '" + cNumDocumento + "', now(), '" + ctecodigo + "') "

ExecSql( cQuery )


Para un update:
Code: Select all Collapse
cQuery := "update mytabla2 set descripcion='" + cDescripcion + "' where recno='" + cRecno +"'"
ExecSql( cQuery )


Para un Select:
Code: Select all Collapse
cQuery := "select documento, fecha, cliente from mytabla where recno ='" + cRecno + "'"

aArray := ExecSql( cQuery )
