GetLastError() lo puedes implementar asi:

include <windows.h>

include <hbapi.h>

HB_FUNC( GETLASTERROR )
{
hb_retnl( GetLastError() );
}

Para el error de LockResource usa este flag al enlazar:

/FORCE:MULTIPLE
