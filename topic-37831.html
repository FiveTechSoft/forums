Hola Adolfo:
Puedes probar con algun font de codigo de barras que imprima 3 of 9, codigos de barras que representan letras, numeros y simbolos, como por ejemplo este:
Code (fw): Select all Collapse
AddFontResource( "Code39r.ttf" )
DEFINE FONT oFont4 NAME "Free 3 of 9"    SIZE 60,60
PRINT oPrn NAME "Etiquetas de Barras" PREVIEW 
          PAGE
          oPrn:Say(1,1 ,CodigoBarra1(STRTRAN(STR(nCodigo,6)," ","0")+cTalle+cColor),oFont4)
          ENDPAGE
ENDPRINT

STATIC FUNCTION CodigoBarra1( x )
RETURN "*"+x+"*" // El * es el codigo que abre y cierra el codigo de barra

Tendras que ver si tu lector lee este tipo de codigos
