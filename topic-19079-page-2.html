Silvio,

I'm not using XBrowse's own functions to calculate them, it's just a pretty simple for each loop and then assign totals to bfooter values.

Code (fw): Select all Collapse
aTots := {0,0}
for each aEle in oBrwDats:aArrayData
    aTots[1] += aEle[5]
    aTots[2] += aEle[7]
next

oBrwDats:aCols[4]:bFooter:= {|| alltrim( transform( aTots[1],     "99,999" ) )  }
oBrwDats:aCols[5]:bFooter:= {|| alltrim( transform( aTots[2], "999,999.99" ) )  }


As simple as this....

I know there are functions in Xbrowse doing this kind of things, but I really prefer making them by myself.
