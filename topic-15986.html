Hola amigo prueba asi:

Code (fw): Select all Collapse
DEFINE FONT oFont2 NAME "Courier New" SIZE 0,-16 BOLD


Code (fw): Select all Collapse
   REDEFINE COMBOBOX oRub VAR vRub ITEMS VecRubro ID 102 OF oDlg1
      oRub:SetFont(oFont2)
