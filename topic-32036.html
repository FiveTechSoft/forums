Detalle de los errores con Buttonbmp

1)
REDEFINE BUTTONBMP BITMAP 'b_accept' PROMPT 'Grabar ' TEXTRIGHT ID 4031 OF oDlg ACTION sound() Anda bien si defino el prompt
REDEFINE BUTTONBMP BITMAP 'b_xcancel' ID 4032 OF oDlg ACTION oDlg:end() TEXTRIGHT anda mal si no defino el prompt. Toma el prompt que trae desde el resource e ignora el TEXTRIGHT Sobreescribe el bmp



Que el TEXTRIGHT funcione respetando los prompt de los botones es importante porque evita tener que escribir todo de nuevo.
en la FWH 10.1 funciona bien.
