Noe

Tenes que renombrar el EXE que se está ejecutando, luego copias.

Code (fw): Select all Collapse
RENAME SISTEMA.EXE SISTEMA1.EXE
COPY FILE  path\SISTEMA.EXE TO SISTEMA.EXE
WINEXE(SISTEMA.EXE)  //PARA QUE EJECUTE EL NUEVO
QUIT
