El truco son 3 controles: VTaskBar, UN SPLITTER y un browse.

El secreto esta en la definincion del Splitter, su PREVIOUS CONTROL es el VTASKBAR y el HIND CONTROL es el BROWSE.

Francis,

Aqui tienes un ejemplo con un listbox, el splitter y el browse:
Code: Select all Collapse
   DEFINE WINDOW oWnd OF oWnd TITLE "Test Splitter"
   
   @ 0, 0 LISTBOX oLbx VAR cValue ITEMS aItems ;
      OF oWnd SIZE 150, 400

   @ 0, 19.6 LISTBOX oBrw FIELDS SIZE 300, 400 OF oWnd

   @ 0, 151 SPLITTER oSplit ;
      VERTICAL _3DLOOK ;
      PREVIOUS CONTROLS oLbx ; 
      HINDS CONTROLS oBrw ;
      SIZE 4, oWnd:nHeight - 70 PIXEL ;
      OF oWnd
   
   ACTIVATE WINDOW oWnd ;
      ON RESIZE oSplit:Adjust()
