si tu esquema es el siguuiente:

por ejemplo el ip fijo del proveedor es (ip: 165.100.15.10)

Conexion Internet de Proveedor ->
Router ->
'------>Pc Server (ip: 192.168.1.100)
'------> Estacion de trabajo 1 (ip: 192.168.1.101)
'------> Estacion de trabajo 2 (ip: 192.168.1.102)
'------> Estacion de trabajo 3 (ip: 192.168.1.103)
'------> Estacion de trabajo n (ip: 192.168.1.10n)

y tu estas queriendo accesar a tu server usando el 165.100.15.10 no te funcionara ya que tu server esta conectado al router en lugar de una conexion directa a internet.

Para poder conectarte de esta forma debes habilitar un desvio de puerto en el router, en donde le indicara que
aquellos intentos de conexion a la ip 165.100.15.10 usando el puerto 3601 por ejemplo, lo desvie a la pc en la red local
con el ip 192.168.1.100 (el server) para que de esta forma te puedas conectar desde fuera a la pc server.

cualquier duda, pon aca tu esquema para que podamos darte una manita
