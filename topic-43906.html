at the end of this function.



Example code:
Code (fw): Select all Collapse
</s>function MyBar( oPrev, oWnd )

   local oBar

   DEFINE BUTTONBAR oBar OF oWnd SIZE 64,64 2007
   DEFINE BUTTON OF oBar PROMPT "BTN" CENTER
