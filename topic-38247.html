rterraz:

Como indica Cristonbal el ROBOCOPY es muy bueno
Code (fw): Select all Collapse
ROBOCOPY D:\Sistemas G:\Sistemas /MIR /R:3 /W:3  //Hace copia de todos los nuevos y los que han sido modificados


También podes usar XCOPY
Code (fw): Select all Collapse
Xcopy D:\SISTEMAS\*.* G:\SISTEMAS\ /i/s/y/h/r/d:12/20/2019  //Copia todos los archivos con fecha del 20 hacia adelante.
