Visual FiveWin - primeros resultados!

Posted: Sat Oct 18, 2008 03:53 PM
Estamos orgullosos de anunciar que gracias a la cooperación de Patrick Fan, hemos podido adaptar FWH 8.10 para construir Visual FiveWin. Esperamos publicar una primera versión beta esta semana próxima.

Las grandes ventajas de este IDE son:

1. Es open source (código abierto) y gratis. Todo el mundo es bienvenido a usarlo, modificarlo y mejorarlo. Todo su código fuente va a estar disponible.

2. Va a estar disponible para la mayoría de las versiones de FiveWin (incluyendo versiones previas a la 8.10). Proporcionaremos los parches necesarios para que funcione con tu versión de FWH!

3. Soporta ficheros RC y puede ser usado con Harbour ó xHarbour!

4. No estas obligado a usarlo. Puedes seguir usando FiveWin de la forma
tradicional, y alternativamente puedes usar este IDE tambien!

Aqui teneis una primera imagen. La versión final usa xbrowse y se ve mucho mejor!
regards, saludos
