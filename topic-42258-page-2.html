vilian wrote:Is there a sample how to do the same with Borland C ?


Having a PRG opened from vcode, press Ctrl+Shift+B. You will be prompted for a tasks.json. Use this one:



tasks.json
Code (fw): Select all Collapse
</s>{
    "version": "2.0.0",
    "tasks": [
        {
            "label": "build and run",
            "type": "shell",
            "command": "c:/fwh/samples/buildh.bat",
            "args": [ "${fileBasenameNoExtension}" ],
            "options": {
                "cwd": "c:/fwh/samples"
            },
            "problemMatcher": [],
            "group": {
                "kind": "build",
                "isDefault": true
            }
        }
    ]
}<e>
