Code (fw): Select all Collapse
ANNOUNCE RDDSYS

REQUEST DBFCDX, DBFFPT

FUNCTION Main()

   RDDSETDEFAULT( "DBFCDX" )

   ...

RETURN NIL


Libs:

\xHarbour\lib\dbfntx.lib
\xHarbour\lib\dbfcdx.lib
\xHarbour\lib\dbffpt.lib
