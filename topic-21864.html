Hola Amigos:
Sería de utilidad que los TRBtns (los utilizo fuera de las ribbon), se comportaran como
los button en el sentido de que obtengan el foco y asi con dar Enter realicen la accion indicada en el Action.
Son botones que pueden tener diferentes formas (Round), pueden tener Bitmaps, pueden tener mas de una linea de Prompt y se ven mejor.

Gracias y saludos cordiales

Ruben Fernandez

Para finalizar: Para que en el RBBtn funcione CANCELAR un Valid, se debe agregar en la clase Control.prg en la linea
1003 la palabra TRBTN asi como estan las otras, TBUTTON, TBTNBMP. Para aclarar la linea 1003 queda asi:
Upper( oWnd:Classname() ) $ "TBUTTON;TBTNBMP;TBUTTONBMP;TRBTN" .and. ;

Y para que el boton cancele un Valid se debe poner oBtn:=.T.
