Pasa los datos a un ARRAY, ordenalos y llena los subtotales en forma manual, agrega las lineas que quieras entre grupo y grupo.
Con ayuda de oBrow:bClrStd pon los colores de acuerdo a un criterio que tu definas.

oBrow:bClrStd := { | | {SelColor1(), SelColor2() }

Function SelColor1()
Local retorno

Do case
case Opcion=1
retorno:= {nRgb(200,200,200),nRgb(125,150,142)}
Case Opcion=2
retorno:= {nRgb(100,200,150),nRgb(125,150,142)}
.......
Endcase

Return Retorno
