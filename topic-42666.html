Estimado Carlos,



De momento un tanto parado por las navidades pero esperamos retomarlo este nuevo año y continuar el trabajo :-)



> Cuando obtengamos funcionalidad web "desde" FWH, deberemos disponer de un servidor propio?



Sí, salvo que vaya a ser utilizado en una red local solamente



> Nos serviría un servicio de Hosting que tengamos contratado?



Solo en el caso de que sea un servidor dedicado



> El servidor deberá tener alguna característica "especial"?



Contestado en la pregunta anterior :-)



> Será compatible el uso de una aplicación de escritorio con el uso de otros usuarios vía web?



En principio si, ya que finalmente es el EXE quien realiza el acceso a las bases de datos, y para el EXE la web es un cliente más



De todas formas, especificarte que de momento es solo un prototipo y aun queda mucho para llegar a ese nivel. De momento

es solo una prueba de concepto muy lejos aún de ser funcional.
