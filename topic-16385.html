Hola,

Yo creo que se ordena bien

Code (fw): Select all Collapse
#include 'Fivewin.ch'

REQUEST HB_Lang_ES
REQUEST HB_CODEPAGE_ESWIN

FUNCTION Main()

   LOCAL a := {  OemToAnsi( 'Sánchez' ), 'SOTO', 'Santos', 'Sevilla', 'Salazar' }
   LOCAL c := ''
   LOCAL i

   HB_LangSelect("ES")
   HB_SetCodePage("ESWIN")

   a := ASort( a )

   FOR i := 1 To len(a)
       c += a[i] + CRLF
   NEXT

   msginfo(c)

RETU NIL


El resultado que aparce es:
SOTO
Salazar
Santos
Sánchez
Sevilla
