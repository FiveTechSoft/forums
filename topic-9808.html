Hello!

I am glad to inform you that FastReport for [x]Harbour (FRH) beta, trial, available for download now!

FRH is a report generator for giving [x]Harbour applications the ability to generate reports quickly and efficiently. FRH provides all the tools that developers need to develop reports.

FastReport for [x]Harbour is based on excellent FastReport 4 and fully adapted for [x]Harbour at HB API level. So FRH works directly with [x]Harbour-data and [x]Harbour-language!

Fully compatible with ANY Harbour or xHarbour -based systems in Win32.

More info and download area:

http://www.paritetsoft.ru/frh.htm

As you can see, the demo-application for FiveWin is not present still. But to make demo (to adapt) is not difficultly. For example, in demo for minigui it is necessary to replace definitions of windows, buttons, checkbox, etc with FiveWin-syntax simply .
