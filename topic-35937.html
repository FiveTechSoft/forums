Help me...

How to clear memory completely (deleting variables, fonts, and bitmaps) created or loaded by a other PRG ????

Example:

[file main.prg]

include fivewin.prg

PUBLIC oBtn, oMain

define window oMain FROM 0, 0 TO 500, 500 TITLE "Teste"
@10,20 BUTTON "Start" ACTION CallPrg2()
ACTIVATE DIALOG oMain

[file call.prg]

FUNCTION Main()

LOCAL oBmp, oFont

DEFINE FONT oFont NAME "Verdana" SIZE 0,-14
REDEFINE DIALOG oDlg RESOURCE "DlgTeste"
REDEFINE BITMAP oBmp ID 101 OF oDlg
REDEFINE SAY oSay OF oDlg FONT oFont
ACTIVATE DIALOG oDlg VALID QuitDlg()

STATIC FUNCTION QuitDlg()

// here, i want release all! Free resources, bitmap, font and variables creates by call.prg and
// return to call.prg without clear variables of function Main()
// How to do?

RETURN( .T. )

You don't need to release anything. LOCAL variables are automatically released at the end of the function and controls variables (oBmp, oSay, etc.) are released by their container (the dialog).
