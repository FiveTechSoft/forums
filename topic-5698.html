Amigo solo hay que colocar la clausula ON CHANGE y asignarle la accion a ejecutar.

Code: Select all Collapse
   REDEFINE FOLDER oFld1 ID 11 OF oDlg ;
   PROMPT "&Clientes","Datos &Pedidos","Premios &Ventas","Premios &Referidos","&Generar Archivo";
   DIALOGS "client","Plano1","Plano2","Plano6","Plano4" ON CHANGE (raton:=.T.,paso:=oFld1:nOption,paso())


Para saber cual solapa estas presionando

vsolapa:=oFld1:nOption
