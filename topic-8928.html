Io di solito uso questo:

Code: Select all Collapse
oIe = CREATEOBJECT( "InternetExplorer.Application" )

oIe:Navigate( "http://www.test.it/script.php" )

WHILE oIe:Busy
    SYSREFRESH()
ENDDO

oIe:Quit()
