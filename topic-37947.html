Esto puede darte una idea

Code (fw): Select all Collapse
ListVentWidth :=  oDialog:nWidth;

xBrowse oBrw OF oDialog ;
FIELDSIZES ( ListVentWidth * .10 ) , ( ListVentWidth * .50 ) , ( ListVentWidth * .15 ) , ( ListVentWidth * .25 );
