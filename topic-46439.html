Claude hooks: En settings.json local o global.

cuando Claude Code emite una notificación, ejecuta PowerShell para reproducir el sonido de sistema notify.wav

Code: Select all Collapse
  {
    "hooks": {
      "Notification": [
        {
          "hooks": [
            {
              "type": "command",
              "command": "powershell.exe -c \"(New-Object Media.SoundPlayer
  'C:\\Windows\\Media\\notify.wav').PlaySync()\""
            }
          ]
        }
      ]
    }
  }
