Now I resolved !!!!




On METHOD GetCoors() CLASS TVistaMenu

I insert this lines on case ntype:=1

IF ::aItems[n]:aItems[n2]:lMove
nBottom :=nBottom+2
ENDIF

before of ::aItems[n]:aItems[n2]:rcItem := { nTop, nLeft, nBottom, nRight }



THIS RUN ONLY FOR :

oWnd:nColumns := 2
oWnd:nType := 1
oItem2:= oWnd:AddItem( "Movimenti", "BIGIMAGE" )
