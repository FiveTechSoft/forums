Important fix

Posted: Sat Sep 15, 2007 08:05 AM

In Class TWindow Method EndPaint(), add this final zero:

METHOD EndPaint() INLINE ::nPaintCount--,;
EndPaint( ::hWnd, ::cPS ), ::cPS := nil, ::hDC := nil, 0

It solves painting errors!
