color propiedad en archivo INI
Code (fw): Select all Collapse
      INI oIni FILE "color.ini"
        SET SECTION "Cores" ENTRY "COR1" TO "111111111" OF oIni
        SET SECTION "Cores" ENTRY "COR2" TO "222222222" OF oIni
      ENDINI
cCor1 := RGB( Val(Left(COR1,3)),Val(Subs(COR1,4,3)),Val(Right(COR1,3)))   ou nRGB( ..., ... ,...) no me acuerdo
cCor2 := RGB( Val(Left(COR2,3)),Val(Subs(COR2,4,3)),Val(Right(COR2,3)))

   @ 1, 1 SAY "TESTE COLOR"  COLOR cCor1, cCor2 OF oDlg 

el mismo concepto a DBF

Jairo
nColor:=ChooseColor()
nRedText:=nRGBRed(nColor)
nGreenText:=nRGBGreen(nColor)
nBlueText:=nRGBBlue(nColor)

remplazas en tu dbf
afile->Rojo := nRedText
afile->Verde := nGreenText
afile->Azul := nBlueText

y cuando quieras verlo
REDEFINE SAY PROMPT "color de mi texto" ID 103 OF oDlg ;
COLOR nRGB(nRedText,nGreenText,nBlueText),nRGB(nRedText,nGreenText,nBlueText)

Jairo,

Para un texto con mutiples colores, etc, lo idóneo es que uses un control RichEdit, luego salvas el texto como RTF y asi lo almacenas en la DBF

Al volver a cargarlo en el RichEdit mostrará los colores, distintos tipos de fonts, etc. de forma automática
