Silvio:

Just an idea.

Using the VALID method you can do this:

Code (fw): Select all Collapse
STATIC cUserVar
....
....
....

        REDEFINE GET aGets[ 2] VAR cUserVar ID 102 OF oDlg UPDATE;
            PICTURE "@K";
            VALID ChgStr(oDlg)
....
....

STATIC FUNCTION ChgStr()
   IF ! EMPTY(ALLTRIM(cUserVar))
          cUserVar := ALLTRIM(cUserVar) + "%"
   ENDIF
   oDlg:Update()
RETURN(.T.)
