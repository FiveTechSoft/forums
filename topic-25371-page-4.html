Lo unico que he encontrado en falta desde mi adaptación a ST2 es la impresión... no es que utilizara demasiado papel para imprimir código, pero algunos fragmentos sí que eran de utilidad.
Con los editores clásicos (multiedit,notepad++...) esto no era problema pero, ¿ Donde está el comando imprimir en ST2 ?

Como casi todo en este editor es configurable, pues buscando y para quien no lo sepa, hay que instalar el package ExportHTML
Preferences->Package Control->Install Package

Una vez instalado, con click derecho aparecerá un menu con la opcion Export->Html.
A los que nos gustan las teclas rapidas para todo, Ctrl+P no ofrece lo esperado, por tanto lo configuraremos.
En Packages\User editar Default (Windows).sublime-keymap y añadir :
Code (fw): Select all Collapse
[
{
    "keys": ["ctrl+p"],
    "command": "export_html",
    "args": {
        "numbers": true,
        "wrap": 900,
        "browser_print": true,
        "multi_select": true,
        "color_scheme": "Packages/ExportHtml/ColorSchemes/Print-Color.tmTheme",
        "style_gutter": false
    }
}
]


Con este package tenemos tambien nuestro código en HTML y con sintaxis coloreada

+ info
https://github.com/facelessuser/ExportHtml

Que lo disfruten !
