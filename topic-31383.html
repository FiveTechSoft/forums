Como resaltar un hashtag o etiqueta en un get say o memo

Posted: Sun Sep 27, 2015 11:29 AM

Tengo un sistema de registro de novedades y me gustaría incorporarle algo similar a tweeter o instangram, la posibilidad de incorporar # o @ como etiquetas que me ejecuten acciones. Ejemplo: #Mandamientos_judiciales ó @Andres_González.

El primero # guardaría una base de datos que guardaría una relación del hashtag y el registro y el segundo @ localizaría al usuario y le mandaría el correo de esa novedad. Esto no presenta creo que muchas dificultades .

La cuestión que pregunto es si hay forma de resaltar esas etiquetas o hashtag en el get, de forma parecida a como lo hacen los programas mencionados. Tendría que poderse hacer desde el get o el say y que tras picar en el hashtag poder ver los resultados. O sea si pulso # ver los registros que aparece el hashtag o si pulso @ ver los registros de las novedades que se le han mandado. Sobre todo quiero saber cómo puedo resaltarlos visualmente, como por ejemplo cambiándolo de color a azul o subrayándolo.

Andrés:

Tal vez la clase SENSITIVE SAY te puede servir, te pongo un ejemplo:

Code (fw): Select all Collapse
#Include "SSay.Ch"

...
...
...
            REDEFINE SENSITIVE SAY oSai PROMPT "descargar ACROBAT READER desde aquí";
                FONT oFont2 ID 800 OF oFld:aDialogs[2] COLOR RGB(255,000,000) COLOROVER RGB(000,000,000);
                ON MOUSEOVER SndPlaySound("Select.Wav");
                ACTION ShellExecute(oWnd:hWnd,"Open","http://www.oldapps.com/es/adobe_reader.php?old_adobe=14?download",NIL,NIL,1) CENTERED //TRANSPARENT
