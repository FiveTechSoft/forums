Tengo esta función que me crea listad de distribución de correo:

Code (fw): Select all Collapse
function CrearLista(cNombre,aContactos)
   local Indice,oOutlook,oMail,oLista
   oOutlook:=CreateObject("Outlook.Application")
   oMail:=oOutlook:CreateItem(0)
   for Indice=1 to len(aContactos)
      oMail:Recipients:Add(aContactos[Indice])
   next Indice
   oLista:=oOutlook:CreateItem(7)
   oLista:DLName:=cNombre
   oLista:Display(.F.)
   oLista:AddMembers(oMail:Recipients)
   oLista:Save()
   oLista:Close(0)
   oLista:=nil
   oMail:=nil
   oOutlook:Quit()
return nil


A pesar de tener puesto oLista:Display(.F.) me sigue apareciendo la pantalla de "Listas de distribucion".
