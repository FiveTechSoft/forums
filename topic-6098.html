Holas,

la solucion para el problema del cellview, va reemplazando el metodo
startline de la clase TMReport


METHOD StartLine( nHeight, lSeparator ) CLASS TMReport

If ::lSalida
::StartLine( nHeight )
Else
Super:StartLine( nHeight, lSeparator )
EndIf

Return NIL
