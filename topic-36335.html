kpidata,

kpidata wrote:...no tengo en mi esa experiencia y conocimiento, mejor decir que aprendí de autodidacta..a la fuerza, y de seguro mi código es muy sucio.. con quizás otras funciones, clases o posibilidades..


Que seas autodidacta no tiene que ser ningún impedimento, pero si como entiendo necesitas programar en un nuevo entorno, ya sea con lenguajes nativos para android/ios o para web, necesitaras un cambio de chip y empezar un nuevo concepto de programación, y lo que quieres hacer si bien no es complicado, necesitas "empaparte" de nuevos conceptos que pasan desde saber montar tu propio servidor de pruebas a iniciarte en lenguajes nuevos y en este caso es en su mayor parte php.

La idea de TWeb es usar la misma filosofia que usamos con FWH y la manera de pensar como podemos diseñar nuestra web se intenta asemejar mucho a nivel conceptual, pero independientemente de esto el cambio te llevará unos dias para empaparte de estos nuevos conceptos. Si tu has realizado ya programas con FWH recuerda el tiempo que paso para crear tus primeros Hello World !...

kpidata wrote:...con esta opción, se podria??


Seguro que si pero se abria de mirar otros temas, por ejemplo que tipo de servidor usais, si teneis un server php, autenticaciones, updates,...

Un nuevo camino se esta abriendo para muchos de nosotros y cada vez necesitamos mas que nuestras aplicaciones entren en un nuevo ecosistema de entornos para que puedan cohabitar todas juntas. Base de datos hosteada para todo o parte del negocio, entornos cliente de escritorio win, mobilidad con web,...

Te recomiendo que mires el help que se encuentra en http://itarraco.com/tweb.help/ apartado Introducción y allá se explica desde como instalar tu servidor local hasta un tutorial de como funciona el sistema. Echale un ojo y mira si ves capaz de seguir adelante.

Esta es mi propuesta para poder dar el salto a la web y sino ya sabes que el mercado esta lleno de soluciones que tambien te pueden ayudar con este tema. Mientras no tengamos otra solución que nos permita un acceso inmediato creo que TWeb puede venir muy bien para apoyar a todos nuestro proyectos.
