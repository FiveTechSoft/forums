Hola

La idea es primero encriptar la tabla desde el ARC o utilizar la funcion
ADSENCRYPTTABLE, luego para utilizar la tabla hacerlo primero, debes
habilitar la encriptacion con ADSENABLEENCRYPTION, esto debe funcionarte
para tablas sueltas, ya que si utilizas diccionario ADD, no es necesario ya que todo es automatico.

Haz el siguiente ejemplo te creas una tabla DBF, luego haces un ejemplo
sencillo como

use tabla new
browse()

con datos se debe ver todo claro, luefo con el ARC encriptas la tabla y
con el mismo ejemplo solo deberas ver basura

luego cambias el codigo y añades algo como

use tabla new
tabla -> ( ADSENABLEENCRYPTION( 'clave' ) )
browse()

y la cosa deberia verse bien

Si no utilizas el ARC para encriptar puedes hacerlo asi

   use tabla
    adsenableencryption(&quot;clave&quot;)
    ADSDECRYPTTABLE()

espero sea de ayuda

saludos

Marcelo
