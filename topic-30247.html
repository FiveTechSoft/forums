Hola Elvira,
acá http://itsme.home.xs4all.nl/projects/windows/registry.html hay algunos trucos de registro, y uno de ellos se aplica a la seguridad de los CHM, en particular:


fix .chm file problem from network drives
windows .chm files can not be loaded from network drives by default, this can be very annoying.

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\HTMLHelp\1.x\ItssRestrictions]
"MaxAllowedZone"=dword:00000004



Tal vez lo puedas probar. Seguramente se pueda configurar o preparar un .reg que se ejecute en la instalación, o algo así, que sea sencillo de aplicar por el administrador del sistema.
