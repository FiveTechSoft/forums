Pego aqui los parametros que utilizo con 7-zip para obtener máxima compresión (más lenta por supuesto)

Que sirva como ejemplo:
Code (fw): Select all Collapse
   Local cXMLDir,cFileXML
   cXMLDir := "."
   cFileXML := cXMLDir + "\Factura_1"
   T7za():New():RunCMD("a","-y -t7z -m0=lzma:d=1024m:fb=273 -mx9 -md128m -mfb128 -ms=4g -ssw -bt -bb3",'"'+cFileXML+'"', '"'+cFileXML+".xml"+'"')
- Genera el archivo Factura_1.7z en la carpeta del ejecutable
- Importante notar en '"'+cFileXML+'"' las comillas antes y despues del nombre, para cuando se tiene espacios
