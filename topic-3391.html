Vuelvo a colgar un Link, a ver si funciona esta vez.
http://hyperupload.com/download/02c35ea ... Y.zip.html

En cuanto a ejemplos, es muy facil, como cualquier otro control:

Desde Resources:

REDEFINE GSAY oSgt VAR cVar ID ... OF oDlg VCENTER 3DBOX COLOR ...
o
REDEFINE GSAY oSgt VAR cVar ID ... OF oDlg CENTER VCENTER COLORSOMBRA 0 COLOR ...

O A mano:

@ 20,130 GSAY oSgt PROMPT "EJMPLO" OF oDlg SIZE 100,12 ......
o
@ 20,130 GSAY oSgt PROMPT "EJMPLO" OF oDlg SIZE 100,12 ...... TRANSPARENTE COLORSOMBRA CLR_OBLUE

ADVERTENCIA: Siempre hay que poner un VAR o un PROMPT para que funcione de forma adecuada.

PARA ACTIVAR LA EDICION EL CONTENIDO (TIPO GET):

Tras el REDEFINE o el @ ...,... hay que poner:

HACER SGET oSgt COLOR CLR_BLUE,CLR_WHITE PICTURE "9999" VALID ....

Echale un vistazo al TGSAY.CH para ver las distintas posibilidades. Si lo prefieres, puedes cambiar (en TGSAY.CH) la palabra clave "HACER" (o cualquiera otra) por otra que más te convenga.
