Juan Carlos.. unas preguntas...

1.- Donde esta el servidor Mysql ?
Si está en una maquina que ademas es servidor web, solo se permite el acceso mediante el usuario configurado en Apache y/o Php. Verifica que tienes un usuario con derechos para conectarse desde la Lan o desde cualquier cliente. No es conveniente usar la misma maquina que cumple la funcion de servidor MYSQL/APACHE para acceder desde programas propios. Para eso usa acceso remoto o web, nunca usar localhost en el mismo servidor WAMP o LAMP con un sistema propietario y en explotacion. Malo para la seguridad y para la performance. Si es para desarrollo y pruebas,si.

2.- Tienes acceso a la consola de Mysql en forma local ?
Si tienes acceso, puedes crear el usuario que necesita tu programa, crealo solo con los derechos que debe tener y configura desde que clientes va a acceder.

3.- Mysql esta en un servidor con acceso desde la LAN ?
No es necesario saber en que lugar esta la DB fisicamente, solo saber la IP del servidor Mysql y el nombre/clave de un usuario con derechos de ingreso. Si el servidor tiene 2 tarjetas de red, una que mira hacia adentro (Lan) y otra que mira hacia afuera (Internet), necesitas solo la IP interna para pasarsela como parametro a la cadena de conexion de tu prg.

Cualquier duda.. solo pregunta, a ver si te tenemos respuesta.

Si tienes acceso fisico al servidor y la passsword del administrador de windows...

Facil...

Ingresa a una ventana DOS (comando del sistema), dirigete al directorio C:\Program Files (x86)\MySQL\MySQL Server 5.5\bin y ejecuta lo siguiente

Code (fw): Select all Collapse
mysql -u root -p

Te va a solicitar la clave del root
Una vez dentro del shell de mysql...

Crea un usuario nuevo para la base de datos con derechos de acceso dentro de la lan.

Code (fw): Select all Collapse
create user 'minombre'@'192.168.0.%' identified by 'miclave';


Luego asignales derechos sobre la DB...
Code (fw): Select all Collapse
grant all privileges on mibasededatos.* to minombre@192.168.0.% ;


Luego resetee los privilegios...
Code (fw): Select all Collapse
flush privileges;



Igual existe una forma mas comoda, que es una mezcla de las 2 primeras.
Code (fw): Select all Collapse
grant all privileges on mibasededatos.* to 'minombre'@'192.168.0.%' identified by 'miclave' with grant option;

Igual debes hacer un flush. Pongale exit y sale de la ventana DOS
Con algun administrador de Mysql trata de conectarte desde algun Pc, puedes usar NAVICAT,Mysql Tools,Mysql WorkBench, phpAdmin,Sql Maestro etc etc etc, usando la ip del servidor y el nombre de usuario y clave recien creadas
