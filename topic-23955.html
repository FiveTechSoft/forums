Hi,

Why you use an index + scope

Code (fw): Select all Collapse
cFSERIE1 := "A"
nCodigo1 := 24

cFSERIE2 := "D"
nCodigo2 := 1
INDEX ON UPPER(FSERIE)+STR(FNUMERO,2,0)
SET SCOPETOP TO UPPER(cFSERIE1)+STR(nCodigo1,2,0)
SET SCOPEBOTTOM TO UPPER(cFSERIE2)+STR(nCodigo2,2,0)
