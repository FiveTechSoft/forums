Lorenzo:

Con buttonbmp puedes usar bitmaps desde recursos asi

REDEFINE ButtonBmp oBtn ID 110 OF oDlg PROMPT "&Cancelar" ;
BITMAP "CANCELROJO" TEXTRIGHT ;
ACTION (oDlg:End())
oBtn:cTooltip:= "Algo" ; oBtn:oFont:= oFont ; oBtn:lCancel:=.T.

Si pones el nombre del bmp sin extension lo busca en el archivo de recursos
