La claúsula ACTIVATE DIALOG ... ON INIT debe devolver una expresión lógica .F. para poder cambiar el foco

Por ejemplo:
ACTIVATE DIALOG oDlg CENTERED ;
ON INIT ( oGet:SetFocus(), .F. )
