Descripción del MCP en uso: gui-inspector

Actualmente, el servidor MCP gui-inspector está conectado y operativo. Este servidor está diseñado para inspeccionar, controlar y testear aplicaciones de escritorio con interfaz gráfica (GUI) en entornos Windows.

---

🚀 Capacidades del Servidor

📦 Gestión de Aplicaciones

  • launch_exe: Lanzar ejecutables.
  • close_app: Cerrar aplicaciones.
  • connect_to_app: Conectarse a una app ya en ejecución.
  • list_desktop_windows: Listar ventanas abiertas en el escritorio.
  • list_managed_processes: Listar procesos gestionados.

🔍 Inspección de UI

  • inspect_controls: Inspeccionar controles/elementos de la interfaz.
  • find_controls: Buscar controles específicos en la UI.
  • capture_screen_state: Capturar el estado actual de la pantalla.
  • screenshot: Tomar capturas de pantalla.
  • recall_screen: Recuperar una captura previa.
  • get_window_rect: Obtener posición/tamaño de una ventana.

🖱️ Interacción con la GUI

  • click / right_click: Clicks en coordenadas.
  • click_control: Click en un control específico.
  • type_text / type_unicode: Escribir texto.
  • type_into_control: Escribir en un control específico.
  • press_key / hotkey: Presionar teclas o combinaciones.
  • scroll: Hacer scroll.
  • drag: Arrastrar elementos.
  • move_mouse: Mover el cursor.
  • focus_window / move_window: Enfocar o mover ventanas.
  • wait: Esperar (para sincronización).

🐞 Testing y Reporte de Bugs

  • report_bug / update_bug / get_bug_report: Crear, actualizar y obtener reportes de bugs.
  • export_bug_report_for_claude: Exportar reporte en formato para Claude.
  • add_analysis_note / get_analysis_report: Añadir notas de análisis y obtener reportes.

🤖 Automatización

  • define_workflow: Definir flujos de trabajo automatizados.
  • record_navigation: Grabar navegación por la app.
  • export_for_web_reconstruction: Exportar para reconstrucción web.
---

Resumen: Este MCP permite lanzar, inspeccionar, controlar y testear aplicaciones de escritorio Windows de forma programática, siendo una herramienta clave para pruebas de UI, automatización, detección de bugs y análisis de interfaces.

MCP Server in Use: gui-inspector

You have an MCP server named gui-inspector connected, designed for inspecting, controlling, and testing desktop applications with graphical user interfaces (GUI). Below are its capabilities:

---

📦 Application Management

  • launch_exe — Launch executables.
  • close_app — Close applications.
  • connect_to_app — Connect to an already running app.
  • list_desktop_windows — List open windows on the desktop.
  • list_managed_processes — List currently managed processes.

🔍 UI Inspection

  • inspect_controls — Inspect UI controls and elements.
  • find_controls — Search for specific controls within the UI.
  • capture_screen_state — Capture the current logical state of the screen.
  • screenshot — Take screen captures.
  • recall_screen — Retrieve a previously taken screenshot.
  • get_window_rect — Get the position and size of a window.

🖱️ GUI Interaction

  • click / right_click — Perform clicks at specific coordinates.
  • click_control — Click on a specific identified control.
  • type_text / type_unicode — Type text or Unicode characters.
  • type_into_control — Type text directly into a specific control.
  • press_key / hotkey — Press individual keys or key combinations.
  • scroll — Perform scrolling actions.
  • drag — Drag elements across the screen.
  • move_mouse — Move the cursor position.
  • focus_window / move_window — Focus or reposition windows.
  • wait — Pause execution (for synchronization).

🐞 Bug Testing & Reporting

  • report_bug / update_bug / get_bug_report — Create, update, and retrieve bug reports.
  • export_bug_report_for_claude — Export reports in a Claude-optimized format.
  • add_analysis_note / get_analysis_report — Add analysis notes and retrieve full reports.

🤖 Automation

  • define_workflow — Define automated workflows.
  • record_navigation — Record navigation steps within the app.
  • export_for_web_reconstruction — Export data for web-based UI reconstruction.
---

Summary: This MCP allows me to programmatically launch, inspect, control, and test Windows desktop applications—useful for UI testing, automation, bug detection, and graphical interface analysis.
