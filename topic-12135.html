EasyPreview is an EMF-based preview system that works on all platforms.
Here are some of the features we have built in:
- exact, wysiwyg preview
- Save to DOC (Word) and RTF files
- Save to clipboard
- Save-to-file possibility: you can conditionally store your printouts to
various graphic formats like JPG, BMP etc.
- 4/6/8 page preview
- multilingual: it supports English, Spanish, Italian, German, French,
Portuguese and Dutch
- many setup possibilities: background color, shadow proberties,
icon etc.
- multiple previews
- individual adjustable zoom factors
- softzoom
- "Send to" function using MSOutlook or the EP email client
