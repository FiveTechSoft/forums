Jimmy,

You can also use Harbour's hbmk2 to build FWH apps with multiple PRGs:

test.hbp
Code (fw): Select all Collapse
tutor01.prg

-I..\include
-L..\lib
-lfiveh
-lfivehc

xhb.hbc
hbwin.hbc
hbmzip.hbc
hbziparc.hbc


hbmk2.bat
Code (fw): Select all Collapse
set path=c:\bcc7\bin
c:\harbour\bin\hbmk2 test


From the cmd do:
hbmk2 test -comp=bcc
