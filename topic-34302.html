Bueno, la unica version gratuita es la del c++ compiler 10.1 la cual es un compilador de 32 bits basado en c-lang. esta es la version que se incluye en la version 10.1 del rad studio (Seattle).
(internamente el compilador c++ esta versionado como 7.2)

actualmente no es usada por los fivewiner ya que no es una version soportada por fwh.
(bueno de poderse se puede, pero con algo de esfuerzo, ya que como es un compilador muy distinto, los parametros son diferentes, pues es complicarse la vida)

Las versiones de prueba de embarcadero del rad studio llamese la version 10.1 (Seattle) o la version 10.2 (tokio)
incluyen la version 32 bits de c++ compiler en dos sabores ( la ya mensionada c-lang y la version legancy que es la que usamos todos derivada del borland c)
asi como la version de 64 bits del c++ compiler solamente basada en c-lang.
(internamente el compilador esta versionado como 7.3)

repasando:
compiladores c++ en embarcadero:

compiler c++ 32 bits basado en c-lang.
compiler c++ 32 bits basado en el antiguo bcc de borland. (usado por los fivewiners)
*compiler c++ 64 bist basado en c-lang.

todo ellos al solictar la version ( por ejemplo bcc32 -v) retornan 7.30, pero ellos son parte del famoso rad studio 10.2
