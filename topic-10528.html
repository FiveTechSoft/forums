Boa tarde Daniel,

Esta opção SETBALLOON( .T. ), é para o FWH8.02, veja window.prg da versão. Mas, se voce fizer as seguintes alterações:

Incluir esta linha
static lTTBalloon := .F. // Global tooltips shape: defaults to standard

e esta função

Code: Select all Collapse
//----------------------------------------------------------------------------//
function SetBalloon( lOnOff )
 
   local lPrevious := lTTBalloon
 
   lTTBalloon = lOnOff
   
return lPrevious


na window.prg e compilar junto com sua aplicação, vai funcionar, eu testei aqui e funcionou.
