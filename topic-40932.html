Code (fw): Select all Collapse
NOTA: GET funciones / reglas de formato:

      @A Permite solo caracteres alfabéticos.
      Los números @B se justificarán a la izquierda
      @C Todos los números positivos irán seguidos de CR.
      @D Todas las fechas estarán en formato FIJAR FECHA.
      Las fechas @E estarán en formato británico: números en formato europeo.
      @K Permite que se vea un valor sugerido dentro de GET
                  área, pero la borra si se presiona cualquier tecla que no sea el cursor cuando
                  el cursor está en la primera posición en el área GET.
      @R Se insertarán caracteres que no sean de plantilla.
      @S <nSize> Permite el desplazamiento horizontal de un campo o variable que
                  tiene <nSize> caracteres de ancho.
      @X Todos los números negativos serán seguidos por DB
      @Z Muestra los valores cero como espacios en blanco.
      @! forzar letras mayúsculas
      @ (Muestra números negativos entre paréntesis con espacios iniciales.
      @) Muestra números negativos entre paréntesis sin espacios iniciales.
