Hernan,

Nosotros estamos probando FiveMac sobre PowerPC y sobre Intel tambien. No estamos usando Rosetta, sino ejecutables nativos para cada plataforma.

FiveMac proporcionará librerías para ambas plataformas. Se pueden crear aplicaciones nativas para PowerPC, ó para Intel, y también "universal binaries", es decir ejecutables que llevan ambas plataformas "incrustadas" en el ejecutable final.

Usamos el gcc (GNU C compiler) que viene incluido en Mac y el crear para una u otra plataforma, ó para crear universal binaries, es cuestion tan simple como pasarle unos flags al gcc
