El problema està en que si la cadena contiene como parte de los literales las comillas simples ( ' ), al evaluarse peta.

Lo he solucionado modificando la cadena antes de llamar al METHOD así:

cStateXbrowse := StrTran( cStateXbrowse, "{'", '{"' )

cStateXbrowse := StrTran( cStateXbrowse, "'}", '"}' )

cStateXbrowse := StrTran( cStateXbrowse, ",'", ',"' )

cStateXbrowse := StrTran( cStateXbrowse, "',", '",' )

cStateXbrowse := StrTran( cStateXbrowse, "'", '-' )  &lt;&lt;-- esto se carga las comillas simples como parte de los literales.

Evidentemente esto funcionarà siempre que no se altere la grabación de la cadena en el METHOD SaveState()

Bueno, ahí lo dejo.
