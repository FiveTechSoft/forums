Pablo...

realmente no es un Bug, es una limitacion del "lamentable" comportamiento de los DIALOG en diferencia de las WINDOW...

para poder agregar las imagenes a los TITULOS (TTITLE) es necesario hacer una seria de calculos previos que involucran datos del contenedor, en tu caso es un DIALOG, por lo que el api aun no ha procesado su creacion sino hasta que se ejecuta el ON INIT

por lo tanto deberas agragar las imagenes a los TITULOS en el ON INIT

Code (fw): Select all Collapse
 Activate Dialog oDlg Center ON INIT oTitle:LoadBitmaps( "BMP_24_EXIT", 10, 10 ) // SE DEBE AGREGAR FILA Y COLUMNA


intentare buscar otra solucion que no rompa con versiones anteriores
