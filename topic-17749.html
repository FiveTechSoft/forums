Ya existe una funcion Week() en xHarbour

Esta es la explicación.

Week()
Calculates the numeric calendar week from a date.
Syntax
Week( [<dDate>] ) --> nWeekOfYear

Arguments
<dDate>
Any Date value, except for an empty date, can be passed. The default value is Date(). Return
The function returns a numeric value. It is the week number that includes <dDate>. If an invalid date is passed, the return value is zero.

Existe una norma internacional (ISO 8601) para la numeración de las semanas del año que pueden ir de 1 a 53. De acuerdo con esta norma la semana 1 puede iniciar el año anterior y también puede iniciar del 1 al 4 de enero del año en curso.

Para obtener el número de semana de acuerdo con dicha norma, te recomiendo analizar la solución propuesta por el Sr. G. N. Rao (nageswaragunupudi) en el siguiente link:

Hi Mauricio,

the Week () function is fully ISO compliant. You can use it every time you need the ISO number of week.
If you want to get te number of week that is not ISO compliant, you have to use a second undocumented parameter of the week () function and set it true.

Code (fw): Select all Collapse
nNonISOWeek := Week (<dDate>, .t.)
