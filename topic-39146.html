Server Apache24 para ModHarbour
Posted: Wed Jul 01, 2020 10:45 PM
Estimados

Aquí les dejo la forma mas fácil de instalar el server Apache24 x64 en Windows
Descargar:
https://drive.google.com/uc?id=1HEae-kEG_b40bjxO30E57AVJtMlGp6sA&export=download
Descomprimir y copiar la carpeta Apache24 en C:\

De la carpeta C:\Apache24 ejecutar como administrador "Instalar Servicio e Iniciar.bat"

En su explorador web escribir: localhost

Y eso es todo.

Está actualizado a hoy día.

Para mas información de ModHarbour con Apache o IIS deben seguir éste foro:
http://forums.fivetechsupport.com/viewforum.php?f=45&start=0

Para crear sus programas o proyectos lo único que tienen que hacer es crear una carpeta dentro de C:\Apache24\htdocs
Y para ejecutar desde su explorador web escribir: localhost/sucarpeta

Pueden analizar o revisar los ejemplos que existen: http://localhost/mod_harbour/samples/

Espero les ayude.
Saludos,
