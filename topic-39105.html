We have not documented "all" functions.
But you can easily see ALL functions of FWH and their source code very easily with \fwh\samples\fwfuncs.prg

Go to fwh\samples folder
if your bcc folder is c:\bcc7, it is ok. If not open the program and modify the static variable
Code (fw): Select all Collapse
//----------------------------------------------------------------------------//

static cBccFolder  := "c:\bcc7\"   // your bcc folder here

//----------------------------------------------------------------------------//

to your bcc folder.

Then build this sample with
Code (fw): Select all Collapse
buildh fwfuncs
// or
buildx fwfuncs


You will find the list of all functions and their source code.
