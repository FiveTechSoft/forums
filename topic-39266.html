Esto descarga el código de mod_harbour y lo construye para CentOS:
git clone https://github.com/fivetechsoft/mod_harbour
chmod +x go.sh
./go.sh

Para instalarlo:
cd /var/www/html
sudo ln -sf ~/mod_harbour/linux/libharbour.so.3.2.0 libharbour.so.3.2.0
sudo ln -sf ~/mod_harbour/samples modharbour_samples
cd /usr/lib/apache2/modules
sudo ln -sf ~/mod_harbour/linux/mod_harbour.so mod_harbour.so

Finalmente añade estas líneas al final de /etc/apache2/apache2.conf
LoadModule harbour_module /usr/lib/apache2/modules/mod_harbour.so
<FilesMatch "\.(prg|hrb)$">
SetHandler harbour
</FilesMatch>

Está explicado aqui:
https://github.com/FiveTechSoft/mod_harbour/blob/master/linux/readme.md
