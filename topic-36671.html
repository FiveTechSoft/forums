Setting oBrw:lGetBar to .t. or .f. only toggles display of the bar.

To make the Gets work, we should first prepare the Gets for the required columns.
This is simple
Code (fw): Select all Collapse
oCol:uBarGetVal := <blankvalue_of the datatype>

You need to do this for those columns for which you want the Gets to work.
Once you assign this, XBrowse constructs Gets for these columns when initializing the browse.

Optionally you can also provide a picture.
Code (fw): Select all Collapse
oCol:cBarGetPic := <picture>


Assuming that the field Marca is character type length 20
You can do
Code (fw): Select all Collapse
oBrw:Marca:uBarGetVal := Space(20)
