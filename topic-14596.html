Carlos,

El código parece correcto. Te faltan dos cosas:

1) Los valores lógicos (BOOL) tienes que recogerlos con hb_parl() y no con hb_parnl()

2) Tienes que hacer "casting" de los valores recogidos con hb_parnl() cuando se trate de un valor tipo HWND, HGLOBAL, es decir en tipos que no son "números" tal cual, sino otro "tipo" de valor, por ejemplo:

aParams.hParentWnd = ( HWND ) hb_parnl(1,27);

a ( HWND ) se le denomina hacer un "casting" (cambiar de un tipo a otro).

Otra cosa:

En el caso de un "array" de BYTEs, entonces tienes tu que copiar los BYTEs:

strcpy( aParams.file, hb_parc( 1, 31 ) );

ó tambien puedes usar memcpy() que copia "bytes" y no "strings", en el sentido de que un string es una secuencia de bytes terminados en cero.

Esta una de las bellezas del lenguaje C:
no hace nada que no se le indique, y cumple estrictamente nuestras instrucciones
