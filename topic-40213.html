To All

I use ResEdit as well and save all my forms as .rc .... I use the batch file rc2dll32 to compile all my .rc forms to .res and all I to is add the .res to my link list .. As you can see below I am using Borland BCC74 32 bit.

Here are my modification to Rc2dll32.bat

Code (fw): Select all Collapse
@Echo Off

DEL DCareW32.RC
DEL DCareW32.RES

COPY *.RC DCareW32.RC

C:\BORLAND\BCC74\BIN\brcc32 -iC:\BORLAND\BCC74\include -v -iC:\BORLAND\BCC74\include\dinkumware -iC:\BORLAND\BCC74\include\windows\sdk DCareW32.Rc >Rick.Txt

del *.iL?
del *.map
del *.obj
del *.~rc
del *.rws
:del *.res
del *.tds
echo done!
