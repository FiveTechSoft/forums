yo utilizo Ads y los ALIAS deben ser un maximo de 10 caracteres

ahora puedes utilizar esto, no se si será lo que necesitas


Code (fw): Select all Collapse
#define CUSTOMERSCENTRAL    aAlias[1]

aAlias[1] := NewAlias()
SELE 0 ; USE "CUST03"  NEW VIA "ADS" SHARED ALIAS (CUSTOMERSCENTRAL)
ALERT(ALIAS())
(CUSTOMERSCENTRAL)->( browse() )
