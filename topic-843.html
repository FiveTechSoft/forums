Leandro:

         Si ya tenes usuario y clave solo te falta el host, que puede ser la ip del servidor, tipo 200.58.112.91 o el dominio, tipo sol1.telpin.com.ar. De la misma manera que te conectas al servidor local con localhost, 127.0.0.1 o una ip local tipo 192.168.0.1 te conectas al remoto especificando el host.

          Un ejemplo con eagle1 y xharbour:

FUNCTION CONECTA(cEstado,oEstado,oTimer,oIco)

Local cHost:='sol1.telpin.com.ar,cUser:='daniel',cPassword:="xxx"

oMySql := TMSConnect():New()

oMySql:Connect( cHost, cUser, cPassword )

No se si te sirva pero la empresa polaca OCT ( www.otc.pl ) tiene un driver RDD que se llama MEDIATOR donde podes enlazar con distintas bases de datos ( mysql, sqlserver, postgresql, oracle y otras ) tus fuentes XBASE directos.

En este caso, en la conexion ODBC tenés que poner el IP remoto del servidor tal como te explicaron en las otras respuestas.

Este gente de otc tiene tambien el RDD para xHarbour desde windows o linux.

No se si te sirva pero si tenes un soft bajo xbase esta solucion anda hasta donde probé, no necesitas programar nuevamente todo tu soft bajo SQL.
