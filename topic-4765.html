Para usar FiveLinux en Ubuntu hay que instalar estos paquetes:

sudo apt-get install libgtk2.0-dev
sudo apt-get install gcc
sudo apt-get install libncurses5-dev
sudo apt-get install libgpmg1-dev
sudo apt-get install libgnomeprintui2.2-dev
sudo apt-get install libglade2-dev
sudo apt-get install glade

Para MySQL:
sudo apt-get install libmysqlclient-dev

Estimado Jairo,

> A) En desarrollos para windos yo uso el gestor de compilación ide Xmate que uso en linux???

Revisa FiveLinx/samples/build.sh

> B) Que versión de Fivelinux debo usar y en donde la puedo bajar????

https://bitbucket.org/fivetech/fivelinux

Para descargarlo y construirlo haz (desde el terminal de Linux):
git clone https://bitbucket.org/fivetech/fivelinux fivelinux
cd fivelinux
make
cd samples
./build.sh tutor01

> C) La Versión de Harbour es especial para desarrollos en linux???

Construye Harbour para Linux siguiendo estas instrucciones (desde el terminal de Linux):
git clone https://github.com/harbour/core.git harbour
make

> D) La versión de Borland es especial???

No se usa Borland. Linux proporciona su propio compilador de c: gcc

> E) Son de paga las lib de FiveLinux o son free???

Free
