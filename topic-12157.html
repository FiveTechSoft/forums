Overview of Bar-Tools Version 3a
Posted: Fri Aug 08, 2008 09:22 AM
Hello,

here is a short description of the new Bar-tool Version 3a.
( Because of the new font-selection,
the download is possible, after FWH 8.08 is released. )

Many new things are possible now :

1. Position of Color 2 with SLIDER-adjustment
the min-value has to be > 0 ( i started with 0.01 )
A value 0 works, but it takes calculation-time.
2. Font defines of buttons
3. Bar-Button-size change
4. Defined Text : 1 or 2 lines for 3 buttons
5. Copy complete source to memory ( saved to 1 field )
6. Help included
7. Save / restore a project to / from disk.



It is possible now, to leave the test without changing the main-bar.



I hope, it will be useful
