Del Escritorio (FWH + Eagle1) a la Web (Hix + Harbour + Node.js) en mes y medio..

Posted: Sat Feb 14, 2026 03:32 PM

Hola a todos,

Quiero compartir con la comunidad el progreso de mi proyecto SEM Web. Soy un desarrollador que, como muchos de ustedes, hemos caminado junto a Clipper (yo desde 1993) y FiveWin ( yo desde finales de los 90). Tras más de 25 años desarrollando mi ERP para Windows con la combinación de Harbour + FWH + Eagle1 + C++ , acepté el reto de llevarlo a la web, el cual ya habia pensado en PHP y adicionale para hacerlo.

En apenas mes y medio, he logrado migrar el módulo completo de PDV/Caja. Un punto clave en este proceso ha sido la transición de los datos. En mi versión de escritorio utilizo la librería Eagle1 de Manu Expósito para la gestión de MariaDB/MySQL. Tomando los conceptos de Eagle1 y con el apoyo de Antigravity, desarrollé una API en Node.js que sirve de puente para este nuevo entorno web.

Módulos ya operativos en la web:

  • Inventario: Consultas en tiempo real.

  • Facturación FEL: Integración completa con certificadores de facturación electrónica en Guatemala.

  • Caja: Aperturas, cierres y flujo de efectivo.

  • Restaurante: Control dinámico de mesas.

  • Ventas: Cotizaciones y Pedidos.

Stack Tecnológico:

  • Backend: Harbour + Hix Server.

  • Capa de Datos: API en Node.js (inspirada en Eagle1) + MariaDB/MySQL.

  • Frontend: HTML5, CSS3, JavaScript y Bootstrap (Vibe Coding con Antigravity).

La transición ha sido fluida. La lógica de negocio que desarrollé y mejoré durante décadas en Windows ahora vive en la web, demostrando que nuestra experiencia con Harbour es una base poderosa para la modernización.
