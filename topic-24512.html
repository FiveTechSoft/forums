Julio,

Te traslado los sabios consejos de nuestro querido Jose Elias:

‎1. Adopten la web (no me refiero a lo que hacen muchos ERP, que sencillamente incluyen Java dentro de un navegador web y lo llaman "una aplicación web". Me refiero a verdaderas aplicaciones web con HTML5 (HTML, AJAX, CSS, etc).

2. Crear infraestructuras de cobro (para cobrar por servicios por Internet), y hacerla fácil de implementar (los fabricantes de ERP no tienen la más mínima idea de lo que están dejando de ganar por no hacer esto).

3. Crear infraestructura para tiendas de aplicaciones, productos y/o servicios (relacionado al punto anterior). Incluso el mismo ERP debe ser construído con esa metodología.

4. Adoptar la nube, pero con capacidad de funcionar desconectado de la Web (esto se hace trivial hoy día en HTML5 con WebStorage).
