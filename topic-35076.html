Extrañamente se corrige este problema con lo siguiente:

en las configuraciones de windows, hay que activar que el _ se muestre en los controles cuando presionamos Alt.
por ejemplo hay botones que de caption ponemos "&Aceptar" para que la A cuando presionamos Alt se muestre con el _ abajo y se ejecute la accion que tiene se control.
cuando se activa esa opcion ahi si sale punteado el combobox la primera vez que obtiene focos, asi como el primer item de un radiobutton los cuales tiene el mismo problema.

Mira se ayuda:

Para marcar o desmarcar una casilla o un botón, presione la tecla TAB hasta que el elemento se seleccione, y luego presione la tecla BARRA ESPACIADORA.
Abra el Centro de accesibilidad presionando las teclas Windows + U.
En Explorar toda la configuración, haga clic en el enlace Usar el equipo sin un mouse o teclado.
