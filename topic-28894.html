Carlos,

Prueba así:

WITH OBJECT loMsg
:Configuration = loCfg
:From = cRemitente
:To = cDestino
:Subject = AnsiToOem( cAsunto )
:TextBody = AnsiToOem( cMensaje )
:AddAttachment( cFichero )
:Send()
END WITH
