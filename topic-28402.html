Hello,

the new release of my imagepainter :
( use it, to modify Your mainscreen adding a logo, signature and more ... )

Download ( 8.4 MB ) :
http://www.pflegeplus.com/DOWNLOADS/ImgPaint1.zip

As addon, You can include the SIGNATURE-editor
You can add 20 possible forms to the image with any size and color.
The original can be cropped to a new image.

Easy to use :
1. select the form and settings
2. click Top / Left ( and Bottom / right ) inside the image
---( You can select UNDO of the last painting )
3. select button < Export >
4. select button < Show > to view the result
5. usage of freehand-painting :
--- Move to the paint-area
--- left mouseclick and release mousebutton
--- start painting
--- press mousebutton again, to stop painting
--- keep pressed, moving to the next start-point
--- release the mousbutton start painting again.

There is a file-counter of the defined export-file like < Myimage1,2,3,4 ... >

The combobox shows the different image-click-solutions
Top / Left or a needed Top / Left and Bottom / Right
Be sure, loading images ( logos ) from the path, defined in the project.
Path < IMAGES > is used / defined, to load logos.



A overview of the different forms and styles
from logos, signature, free text, date and time, linestyles and filled forms
on any position with any size and color.



With the PROJECT-editor You can save all settings to a DBF
To change a project, select the record and next the button < Select >
A right mouseclick activates a project and will be shown at startup.

To change images, please select from subdirectory \IMAGES, because
this directory is defined as imagepath for the different projects inside PROJECT.DBF



The option < Extras > defines the dialog-background and some other settings
like changing the fonts, date-format and freetext.



Shows the result of a cropped image.
Define the area with mouseclick on Top / Left and Bottom / Right
more image-options can be added like sharpen and gray-scale

To make it different to other crop-solutions,
I added section frames to the image.
That makes it possible, to create multiple crop-sections.
Once a section is defined, it stays visible on screen with a box around.
Next, You can define the exact position of the following crop-section.
On the dialog-bottom, there is a file-counter to auto-define the different filenames.

The image shows all selected crop-sections and the crop-results.
