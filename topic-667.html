No te recomiendo usar ODBC, utiliza mejor ADO, funciona perfecto con MySQL, solo necesitas descargar el servidor, y el MyODBC que es el driver que te va a servir para conectarte tanto por ODBC como por ADO.

La cadena de conexion con MySQL es esta:

oAdo := TOleAuto():New("adodb.connection")
oAdo:Open ("DRIVER={MySQL ODBC 3.51 Driver}; SERVER=127.0.0.1; DATABASE=garis;UID=root;PWD=18167231")

y listo, es todo lo que necesitas, con recordset puedes solucionar todos tus probelmas de acceso a datos.

Code: Select all Collapse
#include "fivewin.ch"


FUNCTION MAIN()

   oAdo := TOLEAUTO():New("adodb.connection")
   oAdo:Open("Provider=MSDASQL.1;Persist Security Info=False;Data Source=TestAdo")  // Conexion ODBC


   oRsMySql     := TOLEAUTO():New("adodb.recordset")

    oRsMySql:Open("select * from Clientes", oAdo, 1, 3)  //el 2 := cursor dinámico



   DEFINE FONT oFont NAME "Georgia" SIZE 0,-15
   DEFINE DIALOG oDlg FROM 0,0 TO 550,800 PIXEL TITLE "Ado Connections My SQL" FONT oFont
350,250 PIXEL OF oDlg
   @ 1,1 LISTBOX oBrow FIELDS Str( oRsMySql:Fields(0):Value ) ,;
                              oRsMySql:Fields(1):Value ,;
                              oRsMySql:Fields(2):Value ;
                       HEADERS "Field 1","Field 2","Field 3" SIZE 350,250 PIXEL OF oDlg

   oBrow:bLogicLen := {|| oRsMySql:RecordCount() }
   oBrow:bGoTop    := {|| oRsMySql:MoveFirst() }
   oBrow:bGoBottom := {|| oRsMySql:MoveLast() }
   oBrow:bSkip     := {| nSkip | Skipper( oRsMySql, nSkip ) }
   oBrow:cAlias    := "ARRAY"

   ACTIVATE DIALOG oDlg


   oRsMySql:Close()
   oAdo:Close()
RETURN NIL
