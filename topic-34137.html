Re: Set filter for deleted in MariaSql

Posted: Wed May 31, 2017 03:35 PM
Add a new field
Code (fw): Select all Collapse
oCn:AddColumn( tablename, { "DELETED", "L', 1, 0 } )


Then use this
Code (fw): Select all Collapse
ACTION (oRs:SetFilter( "ploeg2017 = ? .AND. deleted = ?", { ::cCaption, .f. } )
