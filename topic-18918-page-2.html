Martin,

El hbide solo lo he usado con harbour y aqui va un .hbp de ejemplo, solo hay que cambiar las rutas para las librerias.

Yo uso harbour ( del svn ) y MSVC. del sdk para windows 7

Code: Select all Collapse
-ic:\trabajo_abinsa\fwh9.08\include
-lc:\trabajo_abinsa\trees\treereal\lib\tree32m
-lc:\trabajo_abinsa\fwh9.08_corregida\fivehm
-lc:\trabajo_abinsa\fwh9.08_corregida\fivehcm
-lhbwin
-gui
-debug
-ldflag={msvc}-nodefaultlib:msvcrt.lib
-ldflag={msvc}-defaultlib:libcmt.lib
-ldflag={msvc}-nodefaultlib:libc.lib
-lminizip
-lhbzlib
-lhbziparc
-lhbmzip
-loledlg
-lversion
-ldflag={msvc}-force:multiple
-lhbfimage
-lE:\proyectos\imagenes\FreeImage\Dist\freeimage
 
Source/DOCTOS.PRG
Resource/doctos.Res
Source/utiles.prg
Source/mitarray.prg
Source/xbrowse.prg
Source/bitmap.prg

En general si tienes la instalacion de harbour, no deberias tener problemas para usarlo, ya que es solo una mascara para el hbmk2 de harbour.
