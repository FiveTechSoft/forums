Hello Rao,

you are right!

There was a bug in my Function 'AdsRecCount()', probably caused by 'cut & paste' and forgotten to change .F. to .T.

See source:
Code (fw): Select all Collapse
 Set( _SET_DELETED, .F. )
      AdsSetDeleted( .F. )
      nRecs := AdsKeyCount()
      AdsSetDeleted( .T. )
      Set( _SET_DELETED, .F. )  // Here is the bug!
