Hola Aida:

Prueba así:
Code: Select all Collapse
LOCAL oDlg, oMtr 

LOCAL nPos 

USE sos 

DEFINE DIALOG oDlg 

@ 1, 1 METER oMtr VAR nPos TOTAL LASTREC(); 
SIZE 100, 20 BARCOLOR nRGB( 255,0,255), CLR_WHITE

ACTIVATE DIALOG oDlg CENTER ON INIT ( STARTINDEX( oMtr ), oDlg:End() ) 

RETURN NIL
