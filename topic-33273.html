Si aProductos es una tabla hash

Para los valores:

prueba esto en harbour

aVal := HB_HVALUES( aProductos )

o en xHarbour

aVal := HGETVALUES( aProductos )

Para las claves

prueba esto en harbour

aKeys := HB_HKEYS( aProductos )

o en xHarbour

aKeys := HGETKEYS( aProductos )
