Lucas,

No tienes que añadir nada. La función GetComputerName() es propia del API de Windows:

http://msdn.microsoft.com/en-us/library/windows/desktop/ms724295(v=vs.85).aspx

Solo tienes que crear la función en C para acceder a ella:

Code (fw): Select all Collapse
#pragma BEGINDUMP

#include <windows.h>
#include <hbapi.h>

HB_FUNC( GETCOMPUTERNAME )
{
   char buffer[ MAX_COMPUTERNAME_LENGTH + 1 ];
   DWORD dwLen = MAX_COMPUTERNAME_LENGTH; 

   GetComputerName( buffer, &dwLen );
   hb_retc( buffer );
}

#pragma ENDDUMP
