Gracias Carlos por la ayuda. Me había olvidado un paso pero ya tengo listo un bat y con el genero facilmente la RDDADS.LIB para HARBOUR lo cual me servirá para futuras versiones de HARBOUR.

Copio el bat que hice para crear la RDDADS.LIB para Harbour por si a alguien le sirve, a este bat lo puse en la carpeta "HARBOUR" que es donde lo tengo construído y se actualiza con el git:

Code (fw): Select all Collapse
@echo off

:: La línea que sigue setea el path de borland
set path=D:\bcc582\bin

:: La línea que sigue setea a borland
set HB_COMPILER=bcc

:: La línea que sigue setea el path al directorio "acesdk" de nuestro ADS
set HB_WITH_ADS=D:\Advantage 9.10\acesdk

set HB_BUILD_DYN=no

@echo on

win-make.exe


Luego de ejecutarlo (desde línea de comandos DOS), quedan creados en el dir: "D:\HARBOUR\lib\win\bcc"

Los:
ace32.lib
rddads.lib
rddads-bcc.lib
