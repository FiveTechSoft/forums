Pete,

Please try this change in Class TControl:
Code (fw): Select all Collapse
METHOD GotFocus( hCtlLost ) CLASS TControl

   ::lFocused       = .t.
   if ::oWnd != nil
      ::oWnd:nResult   = Self  // old code pending to be oCtlFocus
      ::oWnd:oCtlFocus = Self
      ::oWnd:hCtlFocus = ::hWnd
   endif
   ::SetMsg( ::cMsg )
   ...
