Hola Angel,
Dentro de la carpeta docs del paquete hay documentación con muchos ejemplos.
Yo utilizo un archivo por lotes que llamo desde mi aplicación y que contiene algo así como:

jsignpdfC.exe -kst WINDOWS-MY archivo_original -d ruta_archivo_firmado -V -llx 695.13 -lly 32.0 -urx 868.0 -ury 108.7 -q -ki 0

jsignpdfC.exe -> es el archivo para comandos de consola
WINDOWS-MY -> indica a windows que 'ceda' la informacion de los certificados instalados
-V (mayúscula) -> hace visible la firma
llx, lly, urx, ury -> son respectivamente las coordenadas de la firma en el documento
-q -> quiet
-k 0, 1, 2 etc. -> selecciona el certificado con el que se firmará (siempre que haya mas de uno, claro)

El resto son parámetros obligatorios.
