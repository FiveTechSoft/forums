Adjunto código e imagen utilizando ambos controles

Es la primera vez que utilizo combometro, es correcto el código?
Code (fw): Select all Collapse
   @ nFIG+nInc*4+4, nCIG+80 COMBOMETRO oTipos VAR cTipoEx ;
     ITEMS aTipo ;
     SIZE Int(nW*0.68),  190  ;  //Int(nH*0.9) ;
     OF oPnel1 ;
     COLOR METRO_GRIS1, METRO_WINDOW ; //METRO_APPWORKSPACE, METRO_WINDOW
     PIXEL FONT oFontX




Y con Combobox:
Code (fw): Select all Collapse
   @ nFI+nInc*4+4, nCIG+80 COMBOBOX oTipos VAR cTipoEx ;
     ITEMS aTipo ;
     OF oPnel1 ;
     SIZE Int(nW*0.68), 190 ;   //275
     FONT oFontX ;              //     STYLE CBS_DROPDOWN  LIST ;
     PIXEL COLOR METRO_GRIS1, METRO_WINDOW




Creo que el código en ambos casos es equivalente

Y una pregunta más, es posible que el combobox no exceda su tamaño del control que lo contiene?
