José:
Podría forzar a que el reporte se vaya abajo y luego poner los finales de línea
Code (fw): Select all Collapse
ACTIVATE REPORT oRep WHILE !oQry:EOF() ON INIT CursorArrow() ;
         ON STARTPAGE oRep:SayBitmap(.1,.1,"LOGO.BMP",.5,.5);
         ON END (oRep:nRow := oRep:nBottomRow - oRep:nStdLineHeight * 5) ; // Acá forzas a que se vaya a 5 lineas del final
         ON POSTEND Fin1(oRep) // Acá despues del final, pones las lineas 
.....
STATIC FUNCTION Fin1(oRep)
oRep:NewLine()
oRep:Say(1,"Este presupuesto tiene validez por 8 dias")
oRep:NewLine()
oRep:Say(1,"Los equipos electricos tienen garantia por 15 dias")
RETURN nil

Para hacerlo bien, tendrias que controlar que cuando termina el reporte, oRep:nRow ya no sea mayor que el final de pagina menos 5
