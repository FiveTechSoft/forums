Here you have an updated version of "BuildIt" a ChatGPT AI + FWH + Harbour app generator that will greatly speed up your apps prototyping and implementation.



Full source code and demo app included:

https://github.com/FiveTechSoft/FWH_tools/blob/master/buildit.zip" rel="noopener">https://github.com/FiveTechSoft/FWH_tools/blob/master/buildit.zip



You need an OpenAI key from https://platform.openai.com/api-keys. It is very cheap as we use the mini-4o model:
The OpenAI GPT-4o mini model is priced at 15 cents per million input tokens and 60 cents per million output tokens1. This makes it a cost-effective option compared to previous models like GPT-3.5 Turbo.

This is a work in progress idea and concept. We can go and we will go much further, making it generate the database to use, etc. You can directly ask ChatGPT from the app to improve the screens GUI, add elements, modify them, etc. You can set an overall context as in example "use dark mode", etc. Slash your imagination and start adapting to AI!



Enjoy it! :-)
