Julio César,

La tecla Enter genera un mensaje WM_COMMAND con el valor IDOK (1) en un diálogo, es decir Windows no
lo ruta como la pulsación de una tecla.

Puedes modificar la Clase TDialog de esta forma para comprobarlo:

METHOD Command( nWParam, nLParam ) CLASS TDialog

local oWnd, nNotifyCode, nID, hWndCtl, oCtrl

nNotifyCode = nHiWord( nWParam )
nID = nLoWord( nWParam )
hWndCtl = nLParam

if nWParam == 1
MsgInfo( "IDOK" )
endif

...

y verás como se muestra ese mensaje.

Cuando no hay controles en el diálogo, Windows enruta el enter al mensaje WM_COMMAND

Cuando hay controles, los controles reciben la pulsación y se la dan al contenedor, si ellos no hacen nada
con la pulsación.
