Lo que ha enviado es el código fuente de la aplicación para adaptar Visual studio.
1º.- Se abre el código fuente en visual studio
2º.- Se ejecuta dicho código en modo debug (botón play)
Una vez ejecutado, vuelve a abrir otra instancia de visual studio, pero esta vez soportado el código de colores de Harbour.

Si se produce algún error (como cualquier aplicación VS en modo debug), avisa del error y se posiciona en la línea de código que ha generado el error
