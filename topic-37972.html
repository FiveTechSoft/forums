Yes
Please try this way:
Code (fw): Select all Collapse
oBrw:aCols[ n ]:bClrStd := { || { CLR_BLACK, {{ 0.5, nClr1,nClr2 },{0.5, nClr2,nClr1}, .F. } } }

If the last element of the gradient array is not an array but logical .T. is vertical and .F. is horizontal,
