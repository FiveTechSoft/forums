Today I saw that Word 2010 let you start the keyboard navigation with ALT or also with the
F10 function key.

I inserted in the ribbon – fivewin – sample:
Code (fw): Select all Collapse
SET KEY VK_F10 TO RibbonMenu()
function RibbonMenu ()
   oRBar:KeybMode()
 return

The keyboard navigation opens perfectly with the F10 key.
