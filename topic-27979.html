Here you have a working example:

Code: Select all Collapse
#include "FiveWin.ch"

function Main()

   local oDlg 

   DEFINE DIALOG oDlg

   oDlg:bStart = { || SetCapture( oDlg:hWnd ) }

   ACTIVATE DIALOG oDlg CENTERED ;
      ON CLICK oDlg:End()

return nil

Here you have it:

Code (fw): Select all Collapse
#include "FiveWin.ch"

function Main()

   local oDlg 

   DEFINE DIALOG oDlg

   oDlg:bStart = { || SetCapture( oDlg:hWnd ) }

   ACTIVATE DIALOG oDlg CENTERED ;
      ON CLICK If( nRow < 0 .or. nCol < 0 .or. ;
                   nRow > oDlg:nHeight - 30 .or. nCol > oDlg:nWidth,;
                   oDlg:End(),)
return nil
