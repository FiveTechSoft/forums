Luis,

Puedes cambiar los colores inicial, al pasar el ratón por encima y una vez ha sido visitado:

Code (fw): Select all Collapse
#xcommand REDEFINE URLLINK [<oURL>] ;
             [ ID <nId> ] ;
             [ <dlg: OF, WINDOW, DIALOG> <oDlg> ] ;
             [ FONT <oFont> ] ;
             [ MESSAGE <cMsg> ] ;
             [ URL <cURL> ] ;
             [ TOOLTIP <cToolTip> ] ;
             [ CLRINIT <nClrInit> ] ;
             [ CLROVER <nClrOver> ] ;
             [ CLRVISIT <nClrVisit> ] ;
