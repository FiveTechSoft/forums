Hola Antonio,

No tengo ningun inconveniente con el PDFCREATOR, lo empezé a utilizar hace unos 5-6 años, y me ha funcionado muy bien desde entonces.

Viendo la web de pdfCreator también tiene un ActiveX para poder manejar la impresión, a lo mejor por este metodo lo puedes solucionar.

Lo siento, he visto el tema del registro y me he puesto las manos a la cabeza.


Aqui està el codigo,

Se tiene que hacer una vez selecionada la impresora y antes de enviar el documento a la impresora.

Code (fw): Select all Collapse
 
         Iif(File(cNomPdf),FErase(cNomPdf),NIL)
         oApp := CreateObject( "Bullzip.PDFPrinterSettings" )

         oApp:LoadSettings(.F.)
         oApp:SetValue("output", cNomPdf )
         oApp:SetValue("showpdf", "no")
         oApp:SetValue("showsaveas", "never")
         oApp:SetValue("showsettings", "never")
         oApp:SetValue("showprogress","no")
         oApp:SetValue("showprogressfinished","no")
         oApp:SetValue("confirmoverwrite","no")
         oApp:SetValue("showsaveas","nofile")
         oApp:WriteSettings(.T.)
