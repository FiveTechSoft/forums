We get this error with some versions of Excel.



We need to modify xbrowse.prg

Please locate this line of code in the method ToExcel()
Code (fw): Select all Collapse
</s>oWin:FreezePanes := .t.<e>


Replace with:
Code (fw): Select all Collapse
</s>   TRY
      oWin:FreezePanes := .t.
   CATCH
   END<e>
