Yunus,

Try to use a static var to avoid recursion:
Code (fw): Select all Collapse
function UpdateDbf()

   static lWorking := .F.

   if ! lWorking
      lWorking = .T.
      oTimer:Deactivate()
      ReadMessages()  // Read mesages using http post from telegram and appned to dbf
      sysrefresh()
      AutoReplyMsg()  // reply msgs using http post
      sysrefresh()

      oTimer:Activate()
      oBrw:setfocus()
      sysrefresh()   
      lWorking = .F.
   endif

return nil
