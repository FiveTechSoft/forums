Hola, posiblemente al cambiar el host, no tengas el usuario ampa32% con los permisos para acceso remoto
Si tienes acceso a la base de datos, executa esto y te dira los usuarios que tienes en tu base de datos
Code (fw): Select all Collapse
SELECT HOST,USER FROM mysql.user

Otra posibilidad es que en una versión de mysql nueva, puedas que no accedas si no le pones que sea una password nativa
Code (fw): Select all Collapse
ALTER USER 'ampa32'@'%' IDENTIFIED WITH mysql_native_password BY 'mi_password';
