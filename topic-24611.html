This fixed it :
// returns NOT NULL if extended structure has DBS_NOTNULL field to true
local cNN := {|aArr| iif(Len(aArr) > DBS_DEC, iif(aArr[DBS_NOTNULL] # 0, " NOT NULL ", ""), "")}

Somehow it was assumed to be logical!
