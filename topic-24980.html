Yo uso el propio GEdit de Linux (gnome) como editor de código fuente. Para diseñar las ventanas mi consejo es que uses el propio FIveForm.prg (ejemplo desarrollado con FiveLinux): viewtopic.php?f=12&t=24766&start=0&hilit=fiveform
