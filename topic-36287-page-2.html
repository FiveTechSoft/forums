I just realized that this syntax is uncorrect:

Code (fw): Select all Collapse
Index on "id" tag "custno" to customer


The correct one is:

Code (fw): Select all Collapse
Index on id tag "custno" to customer


Or, even better:

Code (fw): Select all Collapse
Index on FIELD -> id tag "custno" to customer
