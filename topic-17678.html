Javier:

Por su pollo ! que se puede, aqui tienes la función.

Code (fw): Select all Collapse
    IF oReporte:lCreated
        oReporte:oTitle:aPad[1] := RPT_LEFT
        oReporte:oTitle:aPad[2] := RPT_RIGHT
    ENDIF


    ACTIVATE REPORT oReporte;  
        ON STARTPAGE (Alinea(oReporte,1,2));  // 1 y 2 son el número del título
        ON CHANGE PrtDet(oReporte);
        ON END (    oReporte:ENDLINE(),Totales(oReporte),MessageBeep())



/*
* --------------------------------------------------------------------------*
* --------------------------------------------------------------------------*
*/
FUNCTION Alinea(oReporte,Izq,Der)
    oReporte:oTitle:aRow[Izq]:=oReporte:oTitle:aRow[Der]
RETURN (NIL)


En este ejemplo estoy alineando los títulos 1 y 2 pero puedes facilmente ajustar el código
para alinear los Headers. Me había olvidado de ajustar los títulos, uno a la izquierda y otro
a la derecha.
