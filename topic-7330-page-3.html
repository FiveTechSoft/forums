Con permiso de Verhoven. autor de la clase, la he modificado para incluir opcionalmente los botones de la clase sButton de Manuel Mercado.




Si se incluye al principio la siguiente línea se compila con los botones estandar, si se comenta la linea, con los de sButton

Code (fw): Select all Collapse
#define SIN_SBUTTON


Además le he añadido que pegue el resultado en el campo GET en uso. Se puede llamar a la calculadora pulsando F12 o pulsando un botón próximo al get. Si hay varios gets en un diálogo pega el resultado en el que tienen el foco, aunque se llame la función con F12.

La tecla suprimir borra todas las operaciones. La tecla retroceso borra el último caracter. No se permite pulsar más de una vez el punto decimal.

Con la clase original adjunté un pequeño programa de ejemplo, que deberías tener, en el que mostraba como incorporar la clase en cualquier otro programa.
No obstante, dejo aquí ese código de ejemplo en las líneas que siguen esperando que el mismo pueda servirte para disipar cualquier duda:

CALCULADORA.PRG
#include 'fivewin.ch'
#include "common.ch"

static oWnd

function main()
local nResultCalculadora:=0
HB_Langselect("ES")
HB_SetCodePage("ESWIN")
setHandleCount(50)
set 3DLOOK ON

// Fuentes de Texto
public oFont, oFontDoble, oFontDobleN
DEFINE FONT oFont NAME "MS Sans Serif" SIZE 0,-14
DEFINE FONT oFontDoble NAME "MS Sans Serif" SIZE 0,-16
DEFINE FONT oFontDobleN NAME "MS Sans Serif" SIZE 0,-16 BOLD

// Establece los sets de la forma de trabajo.
set DATE ITALIAN
set EPOCH TO 2000
set SCOREBOARD OFF
set WRAP ON

DEFINE BRUSH oBru COLOR rgb(0,0,135) //Fondo de color

DEFINE WINDOW oWnd TITLE 'Calculadora' BRUSH oBru MENU MenuPpal() //ICON oIco
set MESSAGE OF oWnd TO 'Calculadora: VERSION: 23 Septiembre 2007 '+CurDir() CLOCK DATE KEYBOARD
public oCalc:=tCalc():new()
setkey(VK_F12, {|| oCalc:calculadora(oWnd)})
ACTIVATE WINDOW oWnd MAXIMIZED ON INIT;
( oWnd:setfont(oFontDoble), nResultCalculadora:=oCalc:calculadora(oWnd) )
return NIL

#include 'tCalc.prg'

static function MenuPpal()
local oMenu

MENU oMenu
MENUITEM '&Calculadora' ACTION (oCalc:calculadora(oWnd)) MESSAGE 'Encender Calculadora'
MENUITEM '&Salida' ACTION (oWnd:end()) MESSAGE 'Salida del Programa'
ENDMENU
return oMenu
