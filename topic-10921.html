Javier,

Solucionado. Tienes que modificar estas secciones en tu RC:
Code: Select all Collapse
CONTACTOS_DLG DIALOGEX DISCARDABLE 0, 0, 342, 71
STYLE WS_CHILD|4
FONT 8, "Ms Shell Dlg 2", 400, 0
{
  CONTROL "", 110, "TXBrowse", WS_BORDER|WS_TABSTOP, 3, 3, 336, 65
}

Fíjate que usamos otro estilo de diálogo y lo más importante, que el control se define como "TXBrowse" y no como "SysListView32" que es como estaba y por eso desaparecían las barras de scroll
