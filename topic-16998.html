The new Update VTitle Tools Release 2.0a ( Size : 7.1 MB )
Some last Minutes Changes.

The Start-Screen includes 3 Deco-Images : Deco1.bmp, Deco2.bmp and Deco3.bmp
You can delete or replace them with Your own Images.
A subdir < ARCHIVE > with some Alpha-Bmp's is added.
The used BMP's are loaded from Sub-Dir \IMAGES. That means Your BMP's You want to use
must be copied to this Sub-Directory.
Please check the included Make-file ( Path ).
< Freeimage.dll > must be added.
Now You can create a working EXE-file from inside the Application.
The Project-Dialog includes a new Preview-Button. You don't need to close the Project-Dialog,
to show a VTitle-Design.
There is no extra Activation for the Body-Shadow. A Shadow-Slider-Pos. of < 0 > means < no Shadow >
The Application starts with the Values of the 1. Project-Record.
There is a Index on Field < Project >.
Just change a Project-Number like 8 to -8 to bring the Record on Top for the next Start.

The easy Start :
1. Select / open the Dialog < Project > from the right Buttonbar.
2. Have a look at the different Projects with selecting the Button < Preview >
3. For changing a Project, select a Record from the project-Browser and close the Dialog.
-- The Painter opens and uses the new Values from the Project-Record.
4. Change any Settings ( Preview at any Time ) and select < Project > again.
5. Save the new values to the old Project-Record, or choose < New > and define a new Project-Title.
6. To create the PRG and EXE, select < Source > from the Buttonbar.

Just let me know, if there is still something wrong or missing.

Change or delete the 3 Background-Logos
