Silvio,

Here you have a working example:

Code (fw): Select all Collapse
#include "Fivewin.ch"

FUNCTION MAIN()

    LOCAL oDlg, oGet

    LOCAL nVal := -1

    DEFINE DIALOG oDlg

    @ 1,1 GET oGet VAR nVal;
          PICTURE "999.9";
          SPINNER ON UP ( nVal += 0.1, oGet:Refresh() ) ;
                  ON DOWN ( nVal -= 0.1, oGet:Refresh() )

    ACTIVATE DIALOG oDlg;
             CENTER

    RETURN NIL
