Hola amigos:

El amigo Cristobal Navarro, me ha enseñado a instalar - configurar mod_harbour en Windows
por eso le he prometido que publicaria este pequeño manual paso a paso para que lo podais hacer tambien vosotros.

Por supuesto si hay algun error es mio. Ya que a mi me funciona bien.

Bueno ahi va.

Code (fw): Select all Collapse
*** MANUAL PARA INSTALACION Y EJECUCION DE MOD_HARBOUR EN WINDOWS ***
======================================================================

// Consejos, para evitar errores de escritura te aconsejo que copies pegues todo lo que puedas :-))
// No lo he hecho aposta pero me ha salido en 10 pasos.
// El tutorial aunque yo lo he transcrito se debe a la sabiduria y a las ganas de ayudar
// del amigo Cristobal Navarro. Sin el nada seria igual, Ah podemos programar en el FIVEDIT
// y ejecutar el codigo desde el mismo IDE. 
// Gracias a Antonio Linares, a Cristobal Navarro y a todos los que contribuyen, en especial
// a Carles Aubia  para que mod_harbour crezca cada dia. MUCHISIMAS GRACIAS.

1.- Descargar XAMPP para windows web. https://www.apachefriends.org/es/index.html
2.- Descargar mod_harbour desde web.  https://github.com/FiveTechSoft/mod_harbour 

3.- Instalamos XAMPP carpeta por defecto C:\xampp

4.- creamos una nueva carpeta llamada modharbour_samples en C:\xampp\htdocs

5.- En esa carpeta creada "modharbour_samples" copiamos el interior de 
la carpeta samples del fichero "mod_harbour-master.zip" que hemos descargado.
Pero ojo el interior de la carpeta "samples" no se incluye la misma carpeta
hay una serie de carpetas apps, blog , todo eso.

6.- Busca dentro del zip descargado "mod_harbour-master.zip" el fichero mod_harbour.so de la 
version de windows que utilices ( en mi caso 64 bits, esta en  
mod_harbour-master.zip\mod_harbour-master\windows\win64 ) y copiálo a C:\xampp\apache\modules

7.- En la misma carpeta que tienes mod_harbour.so en este caso mod_harbour-master\windows\win64
hay una dll libharbour.dll copiála a C:\xampp\htdocs

8.- En la carpeta C:\xampp\apache\conf hay un fichero que se llama httpd.conf, realiza una copia
de seguridad del mismo por precaucion, ahora edita el fichero y al principio del mismo y antes de 

Define SRVROOT "C:/xampp/apache"

ServerRoot "C:/xampp/apache"

pega estas dos lineas

Define SRVROOT "C:/xampp/apache"
SetEnv LIBHARBOUR "c:\xampp\htdocs\libharbour.dll"

A continuacion busca Loadmodule en el fichero, te sale una lista(hay bastantes ) 
vete al final de todos ellos y coloca.

LoadModule harbour_module modules/mod_harbour.so

Ahora vete al final del fichero y coloca esto.

<FilesMatch "\.(prg|hrb)$">
   SetHandler harbour
</FilesMatch>

Guarda el fichero.

9.- Ahora arranca el Apache y en el navegador teclea:

localhost/modharbour_samples/info.prg

Ya te deberia de salir el pantallazo correcto con el codigo .prg ejecutado en Apache.

10.- Ahora ya puedes ir probando todos los ficheros que tienes en Samples ó añadir los tuyos
propios, ya sabes.

localhost/modharbour_samples/mifichero.prg

Y esto es todo amigos.

Jose Vte. Cheto Agosto - 2019.


Espero que os funcione como a mi.

Un saludo.
